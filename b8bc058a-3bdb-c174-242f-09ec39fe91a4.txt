<br/>　　顾凉晨一惊，随后正色道：“这么严重？”
<br/>　　“神秘家族他们这些人最难以对抗的，就是法则。”叶永阳说出了一句很难懂的话，随后便有些疲惫的闭上眼睛：“回炉一门给了制裁者最大的权利，那就是管理法则。”

<br/>　　除了陆玟丞，还有几名在世上并不出名，但在狼牙地位不低的警戒队成员，姜宏宇便是其中之一。
<br/>　　作为楚涵最信任的一批部下，这些人没有跟随大部分倾巢出动，而是留在基地继续默默无闻的守候。
<br/>　　当他们看到机舱内那个面色苍白，似乎是睡过去的老人时，都一时难掩激动之色，尤其是对一切信息都了解透彻的情报部部·长陆玟丞，当下看着叶永阳的眼睛都快直了。
<br/>　　这就是他们辛苦这么久，差点把整个华夏都翻个遍，甚至不惜让顾凉晨涉险去寻找的人？
<br/>　　叶家最后一代嫡系！
<br/>　　看到这些人的目光都不约而同的看向叶永阳，顾凉晨一下直升机就开口道：“中将，外面风大，我们进屋再说。”
<br/>　　几人当下收回目光，几个人小心翼翼的将叶永阳扶着安排休息，陆玟丞则是与顾凉晨并肩走向情报部。

<br/>　　顾凉晨走下直升机所说的第一句话，实际上也是一句暗号，示意陆玟丞需要慎重对待。

<br/>　　陆玟丞大惊，立即转身打算寻找医护人员。
<br/>　　“等等！”顾凉晨却叫住他，捂着胸口面色苍白道：“越阶杀人后高度集中驾驶直升机，又受了一次刺激而已，不碍事。”

<br/>　　顾凉晨活的如何水深火热，陆玟丞再清楚不过，这两年近乎就是在刀尖上跳舞。

<br/>　　听到这样的话，陆玟丞惊讶无比：“什么都说了？这个人可是四大神秘家族都致力寻找的对象，尤其是白家，早在两年前就用尽了各种手段。叶永***本软硬不吃啊，怎么会在这么短的时间里，对你这个陌生人什么都说？”

<br/>　　陆玟丞的表情瞬间想哭：“你这话说的，我一下子觉得那几夜的工作全白忙活了！”
<br/>　　顾凉晨咧嘴一笑：“对了老大呢？听说狼牙战团去了西南地带有大动作，老大是不是在那里？那里其实距离我找到叶永阳的高原基地不远，但害怕太冒险所以我还是先回了狼牙。”

<br/>　　“老大出事了。”陆玟丞的语气低沉：“已经昏迷了近半个月。”
<br/>　　顾凉晨难得多说的话一下子噎住，惊喜的表情也顿时消失，取而代之的是愕然和无措。
