　　第435章
<br/>　　烈阳熊一直在关注这个独眼少年，因为对方给他的感觉太奇怪了。从开始到现在，对方脸上的表情都很平静，近乎于木纳。但烈阳熊习惯和草原上的野兽战斗，狼、鬣狗、老虎、狮子……越是强壮的动物，在准备发动进攻前便越是平静。
<br/>　　向这少年发动进攻的，也是烈阳部中一位有数的强者。名叫烈阳铁，今年才二十一岁，正是精力和锐气最旺盛的时候。马下摔跤的话，连烈阳熊都不是他对手。
<br/>　　烈阳熊双目微眯，紧紧盯着独眼少年。
<br/>　　只见烈阳铁纵马狂奔，以极快的速度冲向对方。手中的马刀高高扬起，向对方脖子斩去。
<br/>　　独眼少年却端坐不动，像一尊木头塑像。但就在两马相交的刹那，烈阳熊心中一动，感觉到一种凛冽的杀意从对方身上散开。刚才的木头塑像，变成了草原上最凶恶的野兽。
<br/>　　寒光一闪，诸人几乎没看清楚发生了什么。只见烈阳铁的马狂奔，烈阳铁依旧端坐在马背上，但随着马身的欺负，他一颗脑袋突然骨碌碌滚落在地。
<br/>　　少年手中的马刀，湿漉漉滴血。
<br/>　　“你们可愿意奉我为主？”
<br/>　　“铁儿！”
<br/>　　一个中年男人气得大叫，他是烈阳铁的父亲，也是烈阳熊的结拜兄弟。在看到儿子毙命后，他迫不及待向少年发起冲锋。一同冲锋的，还有他另外两儿子。
<br/>　　烈阳熊终于看清楚了少年的动作。这次他的动作幅度很大，从马背上跳起来，跃到另外一人马背后，马刀很干脆的割断了一人的喉咙。
<br/>　　一柄刀向他挥来，他用尸体接这一刀，跳起来，将敌人扑倒在地。随之，马刀从对方嘴巴里扎了进去。
<br/>　　看到自己三个儿子被同一人杀死，剩下那中年男人气得哇哇大叫，不顾一切的扑向对方。
<br/>　　独眼少年回头，盯着对方的马势。退步，躬身，双手握刀，像一张蓄满力气的弓。
<br/>　　在马以极快速度接近他时，他挥出一刀，砍向马蹄。烈马塌架，马背上的人骨碌碌滚落在地上。
<br/>　　独眼少年没有忘记，回头给中年男人喉咙补上一刀。
<br/>　　烈阳熊倒抽一口凉气，稍微移开注视少年的目光，在这一刻，他竟不敢和对方眼神有任何碰撞。
<br/>　　短时间内，烈阳铁一家的男人便都被少年杀死了。对方动作干净利落，是戎族杀人的手段，可招式中又有一些别的东西，他确实看不明白。
<br/>　　烈阳熊并不知道这少年的来历，也不知道和他为伴的都是怎样的存在。的确，福德勒在蛤蟆城沉默寡言，没有什么朋友。可这些年，他并没有闲着。秦蛮、张肥等人习武的时候，他也在习武。关鱼、赵子龙上阵打仗时，他也上阵打仗。高飞豹、张肥坐着小板凳听课时，他也同样在听课，心里琢磨着该如何领兵打仗。甚至，李行哉带着商队经商时，他也会随行。
<br/>　　他和李行哉一起流浪很久，经历过各种各样的事，见过形形色色的人。从一开始，李行哉就有自己的目的，而福德勒也有自己的目的。
<br/>　　李行哉是以为自己准备好了再离开的，而福德勒其实也一样。
<br/>　　他将刀从中年男人喉咙划过，任汩汩的血冒出来。之后才站起身，摆出一个随时可以进攻的招式。
<br/>　　红石部首领黑雁一直盯着这少年，突然喊道：“如果我们让你当我们的首领，你是不是能保护我们？”
<br/>　　“站在我身后。”
<br/>　　独眼少年惜字如金，而从口音听上去，还有些北蛮部的口音。
<br/>　　黑雁用大棍砸开一道缺口，带着红石部残存的人冲了过来，各持兵器立在了少年身后。
<br/>　　烈阳熊并没有阻拦这一幕的发生，他已经察觉到这个男人不好对付了，真的将对方当做一个敌人，而不是某个突然跳出来的小丑。
<br/>　　“好，如果你真能打败我，我也让你做烈阳部的首领。”烈阳熊大声道。
<br/>　　“不。”少年摇摇头：“我不接受挑战，我只接受臣服，要么臣服在我的马前，要么，死亡。”
<br/>　　“野小子，你太狂妄些了吧！”一腔怒火从烈阳熊的体内冒起来，他恶狠狠道：“你叫什么名字？”
<br/>　　“我的名字是……”独眼少年顿了顿，忽然觉得自己应该有个新的名字。
<br/>　　他看到晨雾未散，天地茫茫，口中吐出一个字。
<br/>　　“野。”
<br/>　　他看到天高地阔，原野苍苍，口中吐出第二个字。
<br/>　　“原。”
<br/>　　此刻一轮红日东方跃出，浩浩荡荡的荒原若卷起一场大火，他吐出了最后一个字。
<br/>　　“火。”
<br/>　　然后，他的眼睛亮了起来，冲烈阳熊道：
<br/>　　“嗯，是了，我叫野原火，现在我来了，你们可愿意让我当你们的首领？”
<br/>　　“野原火？”诸人都皱起眉头：“从没听过这个名字呐，究竟是哪里来的？”
<br/>　　“小子，你去死吧。”烈阳熊发起了进攻。
<br/>　　“既然你们不愿臣服……”野原火摇摇头：“那么你们都去死吧。”
<br/>　　他翻身跃上马背，逆着敌人冲锋，只是一合，烈阳熊被斩落马下。独眼少年骑在马背上，闯进了队伍中央，初升的阳光落在他身上，给他浑身上下染上一层红色的光，像极了一团跃动的火焰。
<br/>　　黑雁缀在野原火身后，随着他战斗，微微有些失神。忽然感觉独眼少年的样子，像极了一个人。
<br/>　　当初，黑雁也参与了入侵幽州的战斗，记得当时有个少年，非要占着七八个女人，受了很多欺负。
<br/>　　当然，黑雁只是个小喽啰，也没和对方说过话。但是看对方的相貌动作，的确和曾经的少年有些相似。
<br/>　　不过，怕也仅仅是相似而已。那一战死了很多人，自己是侥幸逃了回来，而当初那个软弱的少年，怕已经死在山谷中，时至今日，恐怕尸体腐烂得也只剩下白骨了。
<br/>　　他没命的大喊一声，挥舞着木棒砸烂一个敌人的脑袋。烈阳熊一死，已经无人可拦得住这独眼少年。
<br/>　　太多人选择放下武器，跪在草地上，向这少年表示臣服。
<br/>　　战斗结束后，少年横刀立马，仍旧是一身鲜红，却已分不清是阳光，或是染上的血。
