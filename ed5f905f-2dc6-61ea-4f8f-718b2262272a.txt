　　第212章
<br/>　　“你都这样穷了，还带仆人！”
<br/>　　夸张张大的嘴巴，负责登记的官员手指着程大雷生锈的盔甲，故意做作的表情是想让所有人都听见。
<br/>　　程大雷此刻已到考官报到的地方，其实关于武举究竟由哪个衙门负责，里面还颇有文章。兵部说『选拔带兵打仗的奖励自然要我们负责』，吏部说『从来都是我们负责选拔官员的』，两家闹得不可开交，其中涉及诸多利益纠结，又涉及到一直的文官和武将之争，实在是分赃不均，最后明帝开口：那就把办公地点选在皇城，两家共同负责吧。
<br/>　　所以说庙堂之高，实在不是程大雷这个山贼能够看懂的。
<br/>　　现在的他，就身在皇城，周围都是来报到的考官，粗略看上去有上百人。负责登记的官员指着程大雷哈哈大笑，把其他人的目光都吸引到程大雷身上。
<br/>　　看着他的装扮，大家脸上也难免露出笑容，长安城的官员还是很有涵养的，没有笑出声来。
<br/>　　徐灵儿和徐神机立在程大雷身后，此刻都感觉无地自容。
<br/>　　“好丢人喔。”
<br/>　　“早知道就不和他一起来了。”
<br/>　　程大雷把自己的官凭印信递了过去，脸上面不改色。
<br/>　　“看看有问题没有？”
<br/>　　“能有什么问题，你这样的穷的独一份，你这盔甲可都算得上古董了，比你爷爷年纪大吧？哈哈。”
<br/>　　那官员根本没有比对程大雷的相貌，直接就将牛三斤的名字登记在册子上，嘴里呵呵笑着。
<br/>　　“你没有问题，我还有问题。”程大雷将自己，或者可以说是牛三斤的官凭收起来。
<br/>　　“你有什么问题？”官员睁大眼睛。
<br/>　　程大雷猛地伸出手，一把抓住他的领口，直接将他从桌子后面拎出来。一拳就砸得他脸上开花，血从鼻子上窜了出来。
<br/>　　所有人都愣住了。
<br/>　　官场是一个讲究体面的地方，大家背地里勾心斗角，表面上却需要保持风度。像程大雷这种边关来的守将，又穷又没后台，来到长安这宦海漩涡，不知夹起尾巴做人，竟然敢殴打天子脚下的官员。
<br/>　　“反了，反了！”那头破血流的官员大吼：“禁卫，禁卫将他抓起来，我要斩他的脑袋。”
<br/>　　徐神机和徐灵儿也吓了一跳，他们从偏僻的幽州来到长安，如今又身在皇城，距离天子不过数里之地，难免心里就充斥着一种乡巴佬进城的自卑感。可却是没想到，这一年不见，大当家怎么变得这么暴力。
<br/>　　周围人刚才以嘲讽的眼神看着程大雷，如今眼神变成了可怜。
<br/>　　“毕竟是偏僻之地来的，真是半点不懂规矩。”
<br/>　　“殴打朝廷命官，仅此一条，就够他斩立决。”
<br/>　　负责皇城安全的禁卫进来七八个人，立时将程大雷围住，为首的小校将那官员扶起。
<br/>　　那官员恨不得扑到程大雷身上，怒不可遏道：“这个狗贼竟然敢打我，把他押下去，活活打死，让他知道天高地厚。”
<br/>　　眼看程大雷已要被带走，此时忽有一人从门外走进来。
<br/>　　“孟大人，谁反了，你又有把谁活活打死？”
<br/>　　见到他进来，周围顿时安静几分，此人名叫百里胜，从四品宣武将军，封常忠伯，武将。
<br/>　　“陛下垂拱而治，国泰民安，海晏升平，孟大人你告诉我，在这皇城脚下，谁反了，谁又敢反？”
<br/>　　“这……”孟大人一惊。
<br/>　　“你一个区区六品中丞，又要将谁活活打死？”百里胜近前一步，威压顿出。
<br/>　　程大雷也有些不明所以，不知道这突然冒出来的百里胜为何要帮助自己。
<br/>　　“常忠伯息怒。”
<br/>　　又有一人从门外走了出来，着文官官服，此人名叫宋天函，四品御史，无勋，文官。
<br/>　　“天子脚下，庙堂重地，此人以九品校尉之职，殴打六品中丞便是以下犯上。孟大人身居此地，便是为为陛下分忧，其人对孟大人不敬，便是对陛下不敬，不将孟大人看在眼里，便是不将……”宋天函抿嘴一笑：“还有，奉劝常忠伯一句，都是帝国的官员，无论品阶大小，从无区区二字。”
<br/>　　一个表情肃穆，一个笑若春风，却是针尖对麦芒，剑拔弩张。
<br/>　　程大雷有些懵，二人究竟在搞什么。
<br/>　　程大雷并不知道，他只迈出一步，就已经踏入帝国两大利益集团斗争的前线。
<br/>　　文官与武官之争。
<br/>　　文官往往是世家出身，武将靠军功攀登，两家向来是针尖对麦芒。而这次武举，又同时撬动两家的蛋糕，私下里早已争破了头。
<br/>　　那孟中丞代表的就是文官集团，每个武将来他都有奚落两句，只是大家都恪守官场的体面，没和他发生争执。而遇到程大雷这明显有槽点的，自然就吐槽能力MAX，只是没料到，程大雷的拳头是真硬呐。
<br/>　　百里胜不是要帮程大雷，宋天函也和程大雷无仇，程大雷只是二人交锋的战场而已。
<br/>　　两大派系的代表人物一出现，在场的考官就分成两批，几乎是对着骂街的架势。不得不说，文官实力占优，骂起人来引经据典，从开天辟地，到早餐吃了碗豆腐脑，而武将都是拙口笨腮的汉子。气得满脸涨红，眼神不停望着左右，潜台词就是：我刀呢，我的刀呢……
<br/>　　“那就暂且让大理寺将这人收押，问明情由后依法处决。”宋天函微微一笑，颇有一种打完收工，我还没出力你就倒下的感觉。
<br/>　　“一定要严办，狠狠的严办！”孟中丞咬着牙道。
<br/>　　文官得意洋洋，武将垂头丧气，百里胜也很郁闷，他是有辩才的，奈何队友实在不给力。
<br/>　　“牛校尉，你可有需要自辩的？”百里胜看了程大雷一眼，其实心里恨得牙痒痒。
<br/>　　只见程大雷面不改色心不跳，冷冷的眼神扫视全场，顿有种尔等渣渣皆不可匹敌的感觉。
<br/>　　在众人的注视中，他缓缓摘下头上锈迹斑斑的铁盔，声音变得深沉，沧桑，以及时不时的长吁短叹。
<br/>　　“我想我可以回答孟大人之前的问题，这盔甲的确有很长时间历史，乃是我牛家祖祖辈辈传下来，最后再由家父传给我。它的确已经很破，很久，但我并不觉得以此为耻，反而十分爱惜。”
<br/>　　程大雷抚摸着头盔，像抚摸着情人的脸颊。
<br/>　　“家父曾穿着它杀过戎族，祖父曾经用它挡过弓箭，太祖曾以其为盾庇护过身后百姓。这上面沾染过戎族的血，也有我牛家历代祖先的血。而如今它很破很旧，说实话我也没钱打一副新的盔甲。但是，当戎族打过来时，我还会穿着他上战场，重复牛家历代祖先做过的事情。”
<br/>　　“我想请问孟大人，这可笑么？”程大雷慷慨激昂，眼含热泪：“不，是荣耀。”
<br/>　　我擦！
<br/>　　百里胜瞬时睁大眼睛：是个人才呐！
<br/>　　程大雷以极其鄙夷的目光扫过鄙夷他的人，心中只有一句话翻腾。
<br/>　　『真当我的雄辩技能白给的么』
