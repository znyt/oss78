　　西南方的山脉与埋骨之地之间，已经在极短的时间里被狼牙战团摸索出一条通道，大量扩军后的狼牙战团实力非比寻常，细化又有了新生力量的各小队也能力更上一层楼，更别说在这世界上最为危险的地带开荒，让狼牙战团的全体人员都到达了一个恐怖的境界。
<br/>　　依照原定计划，由侦查队探路，杀羽战队打头阵，带着一批又一批的队伍深入埋骨之地腹地，从山脉一路到埋骨之地的核心地段附近，在这到处都是丧尸的地带，狼牙战团的近十万人就这么渡了过来。
<br/>　　他们分成好几批，驻扎在一路上的各个定点，相互间连成一条线，与最后方的根基地时刻保持联系。
<br/>　　武器、弹药、食物和水，从未有一天间断。
<br/>　　这里是全世界丧尸最密集的地带，越往中心去丧尸密集度越高，其埋骨之地的核心处更是密密麻麻全是连个落脚地都挤不进。
<br/>　　索性丧尸的分布并不平均，核心地之外的埋骨之地有着丧尸集中地和分散地，从中选取一条避开丧尸集中地的道路并不容易，需要极强的侦查技术和隐匿能力。
<br/>　　近十万数量的人类军团深入丧尸腹地，无论从热感应还是气味上来说，不被对方察觉简直不可思议，但狼牙战团依然做到了，并且时间极短！
<br/>　　此时的杀羽战队就待在距离埋骨之地核心地不到一公里的地方，他们选取了一处半山腰地方隐藏，部分精英队成员和参谋部成员参杂其中，形成了一个小型的全能型团队。
<br/>　　神隐战队和黑芒战队同样如此，这也是三战队史上第一次加入了其他狼牙成员共同行动，只不过他们待在距离杀羽战队所在地的侧翼一公里外，时刻待命。
<br/>　　楚涵不在，这次战略的指挥官为何峰。
<br/>　　此时何峰就与杀羽战队的成员一起，正在不停的查看各队伍方位情况，紧张备战。
<br/>　　打仗，是狼牙战团从未停下的使命。
<br/>　　从第一次大型尸潮死城之战开始，狼牙风格便已形成，就与丧尸这一威胁全世界的种族结下了梁子，楚涵更是带着这一信念不断壮大。
<br/>　　而战团的最终目的地，就是在这里——
<br/>　　埋骨之地！
<br/>　　战略方案被制定修改过无数次，全体参谋部成员的参与，更倾注了楚涵毕生的军事才华，何峰更是早已将所有战术和备用战术熟记在脑子里，所以楚涵在南方尸潮战中的忽然昏迷，丝毫没有影响到这次埋骨大战的进度。
<br/>　　事情要搞就搞大，要搞就搞的对方措手不及，再节节败退！
<br/>　　所以狼牙战团结束战斗后立即出发，用极其恐怖的前进速度来到此处，又用了不可思议的隐匿手段深入核心地。
<br/>　　一切准备就绪，只等开打！
<br/>　　若是被其他人知道狼牙此时此刻的行动，恐怕所有人都会指着楚涵的鼻子大骂他是不是疯了，这显然就是要这十万人类精英中的精英去送死。
<br/>　　但狼牙就是敢来，就是要赢，就是要一次性，将其解决掉！
<br/>　　“是个天坑，很像火山口但此处没有火山。”蒋天庆小声的向何峰汇报前方地理情况。
<br/>　　何峰点头将其记录下来，说道：“气温不寻常，这里明明是亚热带，但天气严寒的过分。”
<br/>　　“丧尸血有温度？那不是一帮活死人嘛！”蒋天庆说着，嘴角勾起笑容鬼魅。
<br/>　　何峰瞥了他一眼，眼神闪着某种不知名的光：“你很兴奋？”
<br/>　　蒋天庆笑的更灿烂，重重点头：“非常兴奋。”
<br/>　　何峰又看了眼不远处的其他狼牙成员，每一人的神色都与蒋天庆相当，那一双双发亮的眼神让人难以移开视线。
<br/>　　“都是一群好战分子。”何峰无奈摇摇头，继续低头整理资料。
<br/>　　此时徐锋无声靠近，面无表情的看着前方，道：“长官，情报部紧急消息，需要一批战士前往阴阳谷。”
<br/>　　何峰手中的笔一顿，眉头紧皱：“这个时候？”
<br/>　　徐锋点头：“楚爹被抓了，不需要太多人，一百精英即可，狼牙受到了一次重创，没人了。”
<br/>　　何峰瞳孔一缩，毫不犹豫的开口：“调三十九队过去。”
<br/>　　“是。”徐锋无声的朝着后方比了个手势，而后继续道：“老大醒了，顾凉晨阵亡，全体狼牙基地防守队成员牺牲。”
<br/>　　何峰瞳孔剧烈收缩了好几下，嘴唇紧抿：“怎么回事？”
<br/>　　“丧尸王和异种王的关系与我们猜测的一样，他们合作了。”徐锋一板一眼的汇报，不加以任何主观情绪：“白允儿死挡了一战现在生命垂危，步纱将老大就醒刚脱离生命危险，陆玟丞在稳住狼牙，白家族长都去了趟，恐怕阴阳谷里……”
<br/>　　“我不能过去。”何峰立即开口，语气很重：“老大也不能去！”
<br/>　　“高少辉前来把老大带走，但陆玟丞的情报里没有显示老大或天照3000出现在阴阳谷附近，所以……高少辉可能是故意将老大带到其他地方。”徐锋说着，有些不确定：“只是楚爹和白允儿同在阴阳谷，我们又正在紧急关头。”
<br/>　　听到这里，何峰眼神发冷：“神秘家族想要知道狼牙动向并不难。”
<br/>　　“你是说他们故意挑了这个时间点？”徐锋震惊问道。
<br/>　　何峰叹气：“我不能确定，但我们行动要提前了。”
<br/>　　“为什么？”徐锋不解：“很多队伍还没准备好，现在并不是最佳进攻时机。”
<br/>　　何峰低下头，快速在纸上写着什么：“我有可能会被强制召唤，如果我和楚涵都不在，这里由你负责。”
<br/>　　旁边的蒋天庆听着两人对话，只觉得心脏都快要跳出来，兴奋感不再，取代而之的是真正的紧张，这是狼牙史上从未经历过的事。
<br/>　　以前哪怕楚涵不在，也有参谋部的第一人何峰镇守，可如果何峰也不在，这仗还怎么打？
<br/>　　谁来指挥？
<br/>　　谁来变动战术？
<br/>　　要知道这种规模的战役，不可能每一步都能依照原定计划进行，往往都会在各方面细节处有调整，甚至一整个战术前后的变动都可能很大。
<br/>　　灵活调整以应对实际情况，才是狼牙最得意的打法，而整个狼牙能做到这一点的人，也只有楚涵与何峰！
