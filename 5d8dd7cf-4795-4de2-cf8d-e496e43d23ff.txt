　　第642章
<br/>　　船队押着俘虏，载着战利品只用一日光景便回了朔方城。探子已经提前把消息传回去，和珅带着人在港口等候。
<br/>　　等船队到了，他立刻带人将粮食卸下来送进粮仓，另外关于俘虏也需妥善安置。做这些事情，蛤蟆城已经有相当经验，也不需要程大雷插手。
<br/>　　程大雷回了城主府，有人给准备好热水。在程大雷苦心经营下，朔方城一座死城渐渐有了生气。在沐浴更衣后，程大雷才洗去此行的杀伐之气。
<br/>　　他换了一套白袍，来到大厅，看到杨国忠、杨小妹和单雄信在这里等候。见程大雷进来，二人立刻站了起来。
<br/>　　“坐，坐。”程大雷摆摆手，道：“杨先生，现在黑风寨的山贼已除，你们的血仇已报，可愿留在我身边做事？”
<br/>　　杨国忠本无所谓杀父之仇，不过是一番托辞而已。但这次见识到程大雷的实力，他已隐隐有些心动。更何况，现在他也不是想走就能走的。
<br/>　　于是，他心里早已做出决定，此刻程大雷问起，他双手一抱拳道：“愿追随大当家，以效犬马。”
<br/>　　“蛮好，蛮好。”程大雷满意的点点头：“既然你愿留下来做事，明日我便差人将你送到蛤蟆城，你尽快熟悉蛤蟆城的事物，日后蛤蟆城的内政便由你管理。”
<br/>　　杨国忠不听这话则已，听到这话后大吃一惊。哪怕是他刚刚加入程大雷的队伍，却也明白蛤蟆城的重要性。相比之下，朔方城根基未稳，蛤蟆城才是程大雷的大本营。如今自己寸功未立，程大雷便交付自己给如此大任，这是怎样的信任。
<br/>　　所谓士为知己者死，身在这乱世，但凡有些本事也想要搅起一些风浪。
<br/>　　“大当家如此待我，属下定不辜负。”
<br/>　　杨国忠以『属下』自称，算是真心诚意打算辅佐程大雷。程大雷倒是不知杨国忠的心路历程。他心中想的明白，蛤蟆城已经步入正轨，只要不乱来，就不会有什么乱子。
<br/>　　而朔方城要做的事情就太多了，需要的是和珅而不是杨国忠。
<br/>　　杨国忠与杨小妹退了下去，大厅内还立着阿喜与单雄信几人，程大雷的目光从他们身上扫过。
<br/>　　这次行动可以说是大获全胜，如果论功行赏当是人人有份。不过这次行动也谈不上十全十美，否则也不会让他混江龙逃了。不能斩草除根，终究是个隐患。
<br/>　　程大雷的目光落在单雄信身上，这次本有历练他的意思，日后好委以重任，然而，单雄信的表现实在有些差强人意。他有悍将之勇，却无坐镇中军的帅才。
<br/>　　有赏便有罚，无罚则无威。程大雷沉思片刻，道：“单雄信，你可知错？”
<br/>　　单雄信本已十分惭愧，所以垂头丧气一直没有说话。此刻程大雷已经问起，他尴尬的点点头，却是没有开口。
<br/>　　“你既然已经知道，我便不说什么。你先从城门队长做起，守好朔方城的城门，日后我再给你别的任务。”
<br/>　　单雄信抱拳答应一声，羞愧得无地自容。
<br/>　　夜已经深了，程大雷回来后安排好这些事情，也是有些疲倦。他打了声哈欠，回屋里歇息去了。
<br/>　　阿喜单雄信一般人各个散去，其他人大都有奖赏，每个人脸上都洋溢着笑容，唯有单雄信苦着一张脸。
<br/>　　他出了城主府，冷风一吹更觉自己寒酸。本也想借着这次机会表现，但最后却让混江龙从自己手上逃了，如此也怪不得程大雷让自己去守城门。
<br/>　　但这口气，他实在有些咽不下去。也是堂堂正正的汉子，结果丢了这么大人，日后怎么在蛤蟆城扬眉吐气。
<br/>　　他心里一发狠，从哪里跌倒便该从哪里爬起来，大丈夫不能一辈子夹着尾巴做人。
<br/>　　他回到营房，拣一把腰刀佩在身上，牵出自己的坐骑，挂上金顶枣阳槊，一个人悄悄离了朔方城。
<br/>　　心里想得明白，自己定提混江龙的人头回来，以洗清自己身上的耻辱，好抬起头做人。
<br/>　　程大雷是第二天早上才得到的消息，只是说早晨点卯时不见了单雄信，连带他的兵器座骑也一块儿不见了。
<br/>　　阿喜道：“大当家，单雄信该不会反了吧？”
<br/>　　程大雷皱起眉头，单雄信临走前连个书信也没留，他自然也猜不到单雄信的用意。不过，自己人前呵斥他两句，他便离营而去，这等火爆脾气当真不好控制呐。
<br/>　　“看城中谁与他交好，问他临走前可留下什么话，也派弟兄们出去访一访，莫让他被别人骗了。”
<br/>　　“是，属下这就去办。”
<br/>　　“对了，差人将杨家兄妹送到蛤蟆城，路上一定要保护他们的安全，莫让二人出了意外。”
<br/>　　“是。”
<br/>　　阿喜答应一声去了，出门的时候和和珅碰个正着。程大雷看到和珅两眼乌青时，顿时吓了一跳：“和大人，昨夜该不会一夜没睡吧？”
<br/>　　“我忙着清点物资，哪有时间睡觉，刚清点好便来向大当家汇报。”和珅虽然一夜未眠，但整个人很兴奋，丝毫没有疲惫神态。
<br/>　　程大雷看在眼里，这完全是个工作狂呐。他点点头，将首领光环打开，面对和珅，他还是不敢有丝毫大意。
<br/>　　“和大人辛苦了，我正有件事，要与和大人商量。”
<br/>　　“大当家吩咐便是，谈不上商量。”
<br/>　　程大雷点点头：“我准备将朔方城交给和大人全权管理。”
<br/>　　“这……”和珅睁大眼睛：“那蛤蟆城那边？”
<br/>　　“蛤蟆城我另外选了人做，当然不会有和大人做得好，不过朔方城正是用人之际，我还是更信任和大人。”
<br/>　　如果让和珅选择，他自然愿意选择朔方城，毕竟，蛤蟆城太多人的位置摆在他前面。但在朔方城自己却可以大权独揽，将自己的才华尽情施展。他只是没想到，程大雷竟然如此信任自己。
<br/>　　和珅抱拳行礼，坚定道：“属下肝脑涂地，定然不辜负大当家的托付。”
<br/>　　程大雷点点头：“我在此地待不了多久，日后这座朔方城便以你为主。”
<br/>(本章完)
