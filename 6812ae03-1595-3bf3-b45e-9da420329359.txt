        &lt;h3&gt;第三卷第三百九十章  打开心结&lt;/h3&gt;
<br/>
<br/>        陈子昂回到冯睿家里的时候，江俞轩因为江家有事情回江家去了。
<br/>
<br/>        陈子昂接到守中电话的时候心里有些乱，她必须将孩子送走，无论是陆家还是张家都有可能盯着两个孩子，人心险恶，她说不清楚到底是谁。
<br/>
<br/>        孩子不可能不出门一直在家里待着，她必须回北京，必须动身了，冯睿这里的事情已经这样，她没有必要在这里呆下去了。
<br/>
<br/>        “大姐，二哥、四姐，我得回北京了，北京有事情我必须得回去处理！我要先走了。”陈子昂对哥哥姐姐说。
<br/>
<br/>        “你走吧，路上注意安全，到了给我们打电话，家里有我们呢，你不用操心了。”大姐陈子慧叮咛陈子昂。
<br/>
<br/>        大家都知道陈子昂忙，这么急匆匆的要走，肯定是有急事情。
<br/>
<br/>        “子昂，事情很严重吗？”陈子霖有些担忧。
<br/>
<br/>        “不严重，但是这两天必须得处理，哥，我可能后天就走了，家里和公司的事情就靠你自己了。”
<br/>
<br/>        “我知道了，你小心点，照顾好自己。”
<br/>
<br/>        “子昂。”四姐陈子乔走过来拥抱了一下陈子昂。
<br/>
<br/>        “四姐，一切都会好起来的！”陈子昂轻轻的对陈子乔说。
<br/>
<br/>        只要洛城一查起来，必然会动摇夏颜，按照夏颜和马总、陆家的关系肯定会给夏颜找一个重要的部门，那么陈子乔就可以抽身而退，至于以后的事情以后再说，最起码陈子乔不会愈陷愈深。
<br/>
<br/>        “保重！”
<br/>
<br/>        “子昂，战强说很久没有看见你了，下午他会过来，你这又走了。”陈子慧无奈的说，陈子昂就是空中飞人，前一段时间在西城的时候，战强去了深圳学习，战强现在是急诊科主任了，这几天事情多，都没有回家一直在单位，今天说下班过来看看，陈子昂又要走。
<br/>
<br/>        “下次回来我去看他，你给他说我忙，有什么事情电话联系。”陈子昂要走了，要不然赶不上飞机。
<br/>
<br/>        “伟冯，我们都是成年人了，做事情要考虑后果，别让别人当枪使。”陈子昂看了一眼陈伟冯，实在是忍不住提醒一下陈伟冯。
<br/>
<br/>        “是，我知道了！”陈伟冯对陈子昂的话不置可否，却也没有反驳。
<br/>
<br/>        “我走了。”陈子昂去房间拿自己的行李箱。
<br/>
<br/>        “小姑，你要走了？爸，我去送小姑吧。”陈天天见陈子昂拿行李箱走出房间随着陈子昂一起朝车库走。
<br/>
<br/>        “是的，北京有事情，我必须回去处理一下。”
<br/>
<br/>        “什么事情比死了人还着急？”陈天天说出这句话就后悔了。
<br/>
<br/>        “人死不能复生，但是活着的人还需要更好的活着，我认为活着的人比死了的人重要，天天，我不知道最近这一段时间你遭遇了什么，我希望你永远都是我们陈家的天天！”
<br/>
<br/>        “你喜欢江俞轩吗？”陈天天看着陈子昂将行李放进后备箱。
<br/>
<br/>        “喜不喜欢已经不重要了，重要的是我现在应该怎么样生活，人不能总是活在过去吧？”陈子昂拉开车门，坐进驾驶室。
<br/>
<br/>        “你们两个人都喜欢对方为什么就是不能各自朝前走一步呢？”陈天天也坐进副驾驶座位上。
<br/>
<br/>        “这个世界上有很多事情不是因为喜欢就可以随心所欲，爱情也不是说爱就爱，婚姻是包括两个人的三观，两个人的生活习惯还有家庭环境。”
<br/>
<br/>        “我今天是故意的，我想看你生不生气。因为我觉得你是喜欢江俞轩的，你们两个人表现出来的默契外人是插不进去的，小姑，我不觉得爱情有那么多大道理可讲，想爱就爱，别等到失去了才知道后悔。”陈天天呐呐的说。
<br/>
<br/>        “你这是给我上课呢？”陈子昂笑了一下。
<br/>
<br/>        “江俞轩说他喜欢我，是我骗他说的，我看见你在他身后，我故意的。我就是想让你不舒服，想让你知道你自己的心，外人都能看出来你们的感情，为什么你自己就不是面对呢，非要等到失去才知道后悔吗？”陈天天声音哽咽，她想到冯睿心里就难过。
<br/>
<br/>        “天天，江俞轩对我来说就是一种特别的存在，就好像是左膀右臂，我们一起工作很多年，我依赖他，信任他，欣赏他，但是我和他没有将来，因为张家，因为江家，越是大的家族越是会看重彼此的出身，有些家族寻找的是旗鼓相当的势力而不是妻子，我和他能做一生一世的好朋友，却不能做相相携到老的爱人。”陈子昂对自己说也是对陈天天说。
<br/>
<br/>        “为什么你就不争取一下呢？我不理解你现在的行为。你现在的事业也做到这种份上。难道还匹配不上江家吗？”
<br/>
<br/>        “江俞轩已经和张倩楠结婚了，你是鼓励我去当第三者呢？天天，人生有很多不得已，不是每一件事情都能按照自己的心意来做的。比如说你妈妈，她穷尽一生想要按照她的思维来生活，但是最后，落到现在这种下场，是我们大家都预想不到的。”
<br/>
<br/>        “小姑，你是不是怀疑我妈妈的死因有别的原因？”陈天天非常聪明，在悲伤之后，当平静下来，她也会去思考整个事件的经过。
<br/>
<br/>        “你妈妈在临走的时候，共编辑了两个短信，一个发给你爸爸，一个发给我，但是发给我的那个短信，永远都发不出来了，因为她只编辑了一半儿。我在想，按照你妈妈平时的思维习惯，她怎么会突然给我们发短信？她为什么会走在路上给我们发短信？说实话这么多年，你妈妈对我们陈家真的是一般般，包括对你，她为什么会突然良心发现？她是受到了什么刺激吗？”
<br/>
<br/>        “我看到了妈妈发的短信，我也知道了她做了一些不太好的事情，我很抱歉，小姑，因为她，你遭受了不一样的人生。但是，她是给了我生命的人，我无法去厌恶她、憎恨她。”陈天天泪水长流。
<br/>
<br/>        “天天，我相信你妈妈有些事情是被蒙蔽的，她可能根本不知道真相，当她知道真相之后一切都迟了，而她对陈家的态度也无可厚非，因为她爱她的小家，因为她爱她的娘家，我觉得作为一个女人，一个女儿，一个母亲，我是理解她的，因为出生环境的问题，因为社会环境的问题，每个人都在改变，每个人都在谋划，为自己，为自己身边的人或事。”
<br/>
<br/>        陈子昂现在就在谋划着一切，为了六哥，为了孩子，更是为了自己的公司众多同事！
<br/>
<br/>        人与人所站立的角度不同，看待问题观念形态也不一样，当陈子昂从一个小女人的角度出发时，她理解了冯睿的自私，冯睿无非就是尽心尽力的帮助娘家，想给她自己和她的儿子谋一个好的前程。
<br/>
<br/>        “天天，你以前是被养在温室里的花朵，现在，我希望是你是一只铿锵的玫瑰，你要用你的脑筋思考问题，你见到的东西，有可能隐藏着你很多你见不到的东西，明白吗？”
<br/>
<br/>        “明白，小姑，保重！”姑侄俩个人挥手告别，陈子昂和陈天天的心结都打开了。
