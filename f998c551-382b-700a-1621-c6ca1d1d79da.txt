        他在一片白色的世界里向她这边快速奔跑了过来。
<br/>
<br/>        是……师父吗？
<br/>
<br/>        又或许，这只是她的梦。
<br/>
<br/>        时间一分一秒地过去……
<br/>
<br/>        赛场上的欢呼声不绝于耳。
<br/>
<br/>        解说员更是慷慨激昂地说道：“下午场，MS战队的‘C’未出场，让很多粉丝担忧不已。今晚，我们很高兴又能欣赏到‘C’的IT高超技术。EA对战MS，今晚这场世界黑客战队大赛，将会是哪支战队进入半决赛呢？荧幕前的观众朋友们，请扫码进行投票。让我们拭目以待。”
<br/>
<br/>        MS战队（黑客部）面对“C”地回归，一个个跟打了鸡血似的，对EA战队展开了猛烈攻击，打得EA战队的个五成员措手不及。
<br/>
<br/>        比赛到最后还剩三十分钟的时候，“C”将EA的主攻服务器植入了MS的“病毒”程序，并快速“感染”到其他四台上，迅速造成EA战队的五台电脑死机，甚至重启或是重装系统都不奏效。
<br/>
<br/>        全场一阵唏嘘，沉浸在惊叹之中。
<br/>
<br/>        当主持人宣布“MS战队（黑客部）获胜最终进入半决赛”时，明司寒按下回车键，收回了植入在EA战队的“病毒”程序。
<br/>
<br/>        他恢复了对方的五台电脑的正常使用，完全不给对方一丁点去研究他的程序代码的机会。
<br/>
<br/>        队友和教练在观众的掌声中激动的相拥，明司寒却一个人独自先行退场，进入电梯，按下第十七层的按钮。
<br/>
<br/>        幽静的酒店长廊，因为地面铺着金色地毯而吞噬了明司寒的脚步声。
<br/>
<br/>        1708号房门口。
<br/>
<br/>        明司寒抬脚用力一踢，直接将房门踹开后，疾步走了进去。
<br/>
<br/>        在这总统套房的玄关处，他撂倒两名保镖后，直接冲到了一个坐在电视机前沙发上的男人面前。
<br/>
<br/>        明司寒二话不说地伸手揪住这个男人的衣领，将他从沙发上提起来后，便是一拳重重地打在对方的脸颊上。
<br/>
<br/>        男人打了个趔趄，再次摔回沙发上，嘴角渗出一丝鲜血，由此可知，刚刚明司寒的下手是有多重。
<br/>
<br/>        “你是怎么知道是我？”男人吃力地坐起身，抬起手来用手背擦了一下嘴角的血丝，毫不畏惧地问道。
<br/>
<br/>        明司寒冷哼：“你没资格知道。”
<br/>
<br/>        男人淡定地笑了笑，正襟危坐：“看样子，你是真的很在意你的未婚妻喽？”
<br/>
<br/>        “贺泽川，你若再敢动星儿，别说让你的战队输一场比赛，我还有能力让你的战队从这个圈子消失！”明司寒倾身上前，再次揪住贺泽川的衣领。
<br/>
<br/>        他的这张冷酷到极致英俊的脸，瞬间在贺泽川的瞳孔中放大。
<br/>
<br/>        贺泽川却坏坏地咧了咧嘴：“今晚，春宵一刻值千金啊！你需不需要谢我呢？”
<br/>
<br/>        “你说什么？”明司寒的瞳仁缩了缩。
<br/>
<br/>        贺泽川挪开明司寒的手，抬起手来，看了看腕表上的时间：“药效已经发作了。我原本计划的是，你找到她的时候，她刚刚药效发作还能在缠你个一时半会儿，让你彻底没法上场比赛。看样子，是我低估了你的能力。你赶在了比赛前找到了她。既然你赢了，那么我给她下的药，就当是送给你的贺礼好了。”
<br/>
<br/>        “这笔账，我以后再跟你算！”明司寒立即放开了贺泽川，拧着眉头，行色匆匆地转身离去。
<br/>
<br/>        在回自己的房间途中，明司寒掏出手机，给私人医生应光济打了通电话。
<br/>
<br/>        电话刚一接通，他便焦急地问道：“如果一个女孩子被人下了那种性质的药，该去买什么药给她治？”
<br/>
<br/>        “是谁被下了那种药？”应光济问道。
<br/>
<br/>        “我未婚妻。”明司寒不冷不热地答。
<br/>
<br/>        下一秒，听筒里却传来应光济的笑声：“既然是你未婚妻的话，那就好办了。”
<br/>
<br/>        “快说！”
<br/>
<br/>        “你把衣服脱光了，躺床上就行了！”
<br/>
<br/>        “你大爷的！”明司寒忍不住大爆粗口，顿了顿后，又补充了一句，“她是傅夜沉的女儿，你觉得我敢动她？”
<br/>
<br/>        “……”一听这话，应光济立马一本正经地回答道，“给她多喂水，一定要多喝水！”
<br/>
<br/>        “知道了。”明司寒没好气地应了声，直接就将电话给挂断了。
<br/>
<br/>        走到自己的套房门口，明司寒拿着卡刷开房门后，一丝刺肤的冷意，像簌簌寒风般从门内刮了出来。
<br/>
<br/>        明司寒走进去后，房门自动反弹关上，又是一阵风吹来，他的双臂也因为这股冷风起了鸡皮疙瘩。
<br/>
<br/>        他下意识地瞄了一眼墙上的空调开关，十六摄氏度，怪不得这么冷。
<br/>
<br/>        当他想把空调温度调高的时候，忽然间想起了什么，连忙疾步往室内走去。
