　　第400章
<br/>　　白元飞鬼鬼祟祟，顺着墙角爬上屋顶。随后手脚并用，在屋脊上行走，四肢接触瓦片，轻盈得不发出任何声音。
<br/>　　能在长安城得到飞天蛤蟆的匪号，又可在程大雷手上得到重用，他自然有不凡之处。
<br/>　　但与其说他是一只飞天的蛤蟆，倒不如说他是只在房顶行走的猫。白元飞很满意自己今晚的表现，顺利取得凉州城情报，又取到李行哉身上一件信物，只要平安回到山寨，这次任务就称得上圆满。
<br/>　　然后，无意间向屋顶下扫了一眼，白元飞这只『夜猫』，看见了一头黑豹。
<br/>　　对方同样在夜幕中行走，手脚并用，只不过动作没有白元飞轻盈，但更加简单直接，纯粹高效。
<br/>　　他从回廊跃到墙角，又贴着墙边跃到假山上，四肢永远可以找到最好的落脚点。路上碰到的一个个人，都干脆倒在他的刀下。
<br/>　　白元飞一身寒意，城主府内竟然有这样的高手。
<br/>　　他和福德勒不算熟悉，互相见面的机会少之又少，平素他也不是很喜欢接近福德勒。而此刻二人距离又远，视野又差，白元飞没能认出福德勒。
<br/>　　主要原因，还是白元飞没想到琴川关那个木讷的男孩，竟然是个如此狠辣的杀货。
<br/>　　自己是一名飞贼，靠手艺吃饭，和这样的杀贼不是同一条路上的。这种人惹不起，赶紧撤。
<br/>　　白元飞将脖后的汗珠擦去，没再做任何停留，第一时间就向外转移。
<br/>　　于此同时，福德勒也发现了一件可悲的事实：自己好像迷路了。
<br/>　　关键是城主府太大，大部分建筑又极其类似，他胡乱向前走，渐渐分不清东南西北。
<br/>　　而此时，城主府的卫兵已经对福德勒的行为有了反应，黑色的夜里亮起火把，人声噪杂，李行哉所住的院子第一时间被保护起来，然后开始寻找凶手。
<br/>　　福德勒也没有傻到鸡蛋撞石头的地步，他在躲避追捕，寻找暂时的安身之处，以等待可以出手的机会。
<br/>　　他的速度很快，没有白元飞那样轻盈，但动作却很干净，整个人走一条『之』字形路线，任何可立足的地方，都将会成为他脚下的路。
<br/>　　前方是一座假山，隐隐传来脚步声。福德勒瞬间提起精神，在草原上长大的人，有种类似野兽的听觉。对方速度很快，不是弱手，足以给自己带来危险。
<br/>　　他瞬间握住了刀柄，一路上，他已杀过很多人，都是直击要害，一招致命，以减少不必要的麻烦。
<br/>　　可随之，那脚步声消失了。福德勒的心瞬间提了起来。在自己发觉对方的时候，对方也发觉了自己，都是狠人，都在等待猎物松懈时，一招毙命。
<br/>　　隔着一道假山的对峙，究竟谁是猎物，谁是猎人。
<br/>　　福德勒放松呼吸，猛地向前跨步，手中刀干净利落的挥出。同一时间，他察觉到对方的呼吸也消失，这是一个猎人，挥出猎刀的前奏。
<br/>　　铛！
<br/>　　清脆的碰撞声，同一时间停步，同一时间攻击，二人的刀也同时撞在一起。
<br/>　　短暂几个呼吸间，双方已经撞了数招，平分秋色，谁也奈何不得谁。
<br/>　　福德勒知道城主府有高手，肯定有人的实力在自己之上，但对方出色的反应和力量，还是给他带来不小的震撼。
<br/>　　倒是对方的刀势缓了缓，愣了愣神道：“是你？”
<br/>　　对方穿着夜行服，蒙着口鼻，能看到的只有一双眼睛。福德勒没办法认出对方，可他察觉到了对方的松懈，随即立刻出手，刀切喉咙。
<br/>　　高手相争，一分一毫的松懈都是致命的。
<br/>　　“喂喂，是我啦，自己人。”对方慌的大叫，动作狼狈。
<br/>　　刀刃接近对方脖子时硬生生的停住，福德勒眼睛逼视着对方，等待对方开口，如果话有不对，他可以随时击杀对方。
<br/>　　“我啦，白元飞，飞天蛤蟆。”
<br/>　　人潜意识中，会从危险向安全转移。如今城主府开始搜捕，那些燃烧的火把就代表了危险，而被黑暗覆盖的区域就代表了安全。
<br/>　　不约而同，白元飞和福德勒躲向同一处地方，然后又不期而遇。
<br/>　　福德勒还真不认识白元飞，在琴川关，他不是个很善于交际的人。但同时，福德勒有是个记忆力不错的人，飞天蛤蟆这四个字他是听说过的。
<br/>　　再看这人的做派，很强，但说话有透着一股蠢，蠢强蠢强的，妥妥是琴川关出来的。
<br/>　　“你怎么来了？”福德勒问，见到自己人他也松了口气。
<br/>　　“现在是聊这个时候的么，现在是聊这个的时候么？”白元飞重复了两遍：“我们现在该想想怎么逃命吧？”
<br/>　　福德勒回头看了一眼，白元飞说得果然不错，卫兵已经摸到这里，火光越来越近。现在不跑，或许就没机会离开了。
<br/>　　“我得去救人。”福德勒现在还想着李行哉。
<br/>　　“他不用担心，在这里没人敢动他，我们回到山寨，再想办法救他。”
<br/>　　福德勒点点头，潜意识中他觉得，只要事情让程大雷知道，程大雷总是会有办法的。而福德勒对程大雷的忌惮之深，是任何人都无法理解的。
<br/>　　“快走吧，再不走来不及了。”白元飞焦急起来。其实他很是郁闷，如果没有福德勒，他早已神不知鬼不觉的离开这里，又怎会惹出这样的事。
<br/>　　“已经来不及了。”福德勒直起身，看到四面都有火光，层层推进：“我没你飞檐走壁的本事，离不开这里。你现在走，还来的及，我帮你托住他们。”
<br/>　　白元飞怔怔看着这个年轻的戎族，难以想象，这么小的年纪，他便干净的像一块黑铁。丝毫不拖泥带水，做事干脆果断。
<br/>　　“好兄弟！”白元飞赞了一声：“等回到山寨，我请你喝酒。”
<br/>　　白元飞也不啰嗦，他当然明白，现在福德勒所说是最好的选择，可同时也意味着，他将自己置身在危险中。
<br/>　　宋伯康不会杀李行哉，可不意味着他同样不会杀宋伯康。
<br/>　　白元飞腾身而起，从假山窜到一棵大树上，又顺着树梢爬到大屋的房顶，脱离了包围圈。
<br/>
