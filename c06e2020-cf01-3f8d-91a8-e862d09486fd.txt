　　这一道剑光，猛然穿透空间，化为了一道空间隧道。
<br/>　　其中的剑锋，全部向着这隧道背后的楚风眠，斩杀而去。
<br/>　　“剑师兄的十步杀剑，已经是有所大成了，这十步杀剑的极致，便是十步之内，足有斩杀千里之外的武者，剑师兄将这十步杀剑大成，在这归墟天中，没有任何人会是您的对手！”
<br/>　　一名天剑宗弟子，在一旁急忙的奉承到。
<br/>　　“哼，不错，在外面的时候，我一直是将这十步杀剑隐藏起来，作为杀招，打算用来对付九魔城，圣王宗的那些家伙的，如今一个北域来的小子，居然就敢来挑衅我们天剑宗的威严，今日我就以这十步杀剑，先杀了他！”
<br/>　　剑师兄听到这些弟子的奉承，脸上也不由的露出几分得意。
<br/>　　他手中剑锋抖动，无数剑气便是顺着那空间隧道，全部向着楚风眠斩杀过去。
<br/>　　但是这无数剑气，顺着空间隧道打出去，却并没有出现多少效果。
<br/>　　反而是在下一刻，一声龙吟，突然是从这空间隧道之中响彻起来。
<br/>　　“轰！”
<br/>　　眨眼之间，一道巨大无比的战龙之爪，猛然之间从这空间隧道之中打了出来。
<br/>　　这战龙之爪从中冲出的一刻，直接将这空间隧道，都给震得扭曲破碎，同时这巨大的龙爪，已经是向着那天剑宗四人，席卷过去。
<br/>　　“轰隆！”
<br/>　　这战龙之爪的突然出现，也完全是出乎了他们天剑宗弟子的意料之中，他们先是一楞，随后连连催动无数剑术，准备来抵挡。
<br/>　　“一群天剑宗的废物，楚某还没有来找你们，你们倒是想要杀我？我杀了你们天剑宗的弟子又如何？再敢放肆，你们也一起去陪葬！”
<br/>　　楚风眠的声音，正是从这战龙之爪中，突然响彻而起。
<br/>　　随后这战龙之爪的力量，猛然爆发出来，轰隆一声，化为了巨大无比的爆炸。
<br/>　　那些天剑宗弟子，落入到了这爆炸之中，顿时一个个都被轰飞了出去，口中吐出了无数鲜血。
<br/>　　就连那为首的剑师兄，都是连连后退数步，他的脸色，已经是气的铁青。
<br/>　　“该死！小畜生！我一定杀了你！”
<br/>　　剑师兄脸色气的铁青，疯狂的咆哮着，他再次出剑，想要再度斩杀楚风眠，但是他的剑，却是失去了方向。
<br/>　　楚风眠的气息，彻底消失，令他根本察觉不到。
<br/>　　“剑师兄，怎么办？”
<br/>　　剩下那三名天剑宗弟子的脸色，也是一样的愤怒，他们七大宗门的弟子，向来都是他们杀别人，什么时候居然会被人逼到了如此地步。
<br/>　　剑师兄目光闪烁，思考了一阵才突缓缓开口道。
<br/>　　“走，我们去归墟大陆，除非是这小子不想要归墟天的宝藏，不然他肯定会去归虚大陆的，倒时候我们再去杀了他！”
<br/>　　“是。”
<br/>　　那另外三名天剑宗弟子，也是听见这剑师兄的话，直接回答一声。
<br/>　　他们的身形，便是化为遁光，直接向着归墟大陆的方向飞了过去。
<br/>　　同样在另一边，在无数岩石群中，楚风眠便是在其中，连连穿行。
<br/>　　顺着他感觉到了气息，在寻找着无上剑气的踪影。
<br/>　　“这些天剑宗弟子，看来真是找死。”
<br/>　　楚风眠的眼神中，露出了几分杀机。
<br/>　　等到了归墟大陆，让他在碰到这些天剑宗弟子，他必然要出手，直接杀了这些人。
<br/>　　其实就在刚刚，楚风眠若是想要全力出手，就足以轰杀了这些天剑宗弟子。
<br/>　　不过现在，还不是出手的时候，对楚风眠来说，现在找寻无上剑气，可是要比起杀那些天剑宗弟子要重要的多。
<br/>　　反正到了最后，在归墟大陆上，终归是会有着碰面的机会，倒时候再杀也不迟。
<br/>　　楚风眠在这归墟天之中，只有一个月的时间，可以来寻找那三千道无上剑气，这无上剑气得到的越多，也就代表着，楚风眠的实力将会越强。
<br/>　　现在楚风眠便是已经感受得到，他的实力，已经是开始跟他所创造出的九域剑术，息息相关，可以说九域剑术完善的越多，楚风眠的实力也就会越发的强大。
<br/>　　“就在这前面！”
<br/>　　楚风眠看着前方，突然感觉到了一种锋利的剑芒。
<br/>　　在归墟天中，能够拥有这样剑芒的存在，也就便是只有一个可能，无上剑气。
<br/>　　楚风眠顺着剑芒散发出来的方向，直接一刀遁光，飞了过去。
<br/>　　就在这无数岩石背后，居然是一处极为特殊的区域，是一块漂浮在空中的大陆，这大陆上，没有丝毫生机，只有万里黄沙。
<br/>　　而就在这万里黄沙之中，无数剑光，正从中闪耀这。
<br/>　　“这些是？”
<br/>　　楚风眠目光凝重，看了过去。
<br/>　　这一道道的剑光，不是其他，正是无上剑气，一共是足足有着四十二道无上剑气，如今联合起来，直接布置出了一道阵法。
<br/>　　而在这阵法中间，却是有着五名武者，这五名武者合力之下，在对抗这四十二道无上剑气，组成的阵法。
<br/>　　让楚风眠目光，一眼望过去的时候，最令他注意的，便是那五名武者之中，站在最中间的哪一位。
<br/>　　此人的手中，手持一把赤色灵剑，显然也是一尊剑修，而在他的身边，却是围绕着七道无形剑气。
<br/>　　这七道无形剑气，不是其他，正是无上剑气。
<br/>　　在这归墟天中，居然已经是也有着其他剑修，察觉到了无上剑气的存在，甚至是已经在降服这些无上剑气。
<br/>　　七道无上剑气，已经是足以化为了一道阵法，来加持这那一尊剑修的实力。
<br/>　　“此人的实力，好强。”
<br/>　　楚风眠打量了一眼那拥有七道无上剑气的剑修，不由的感叹了一声。
<br/>　　这一尊剑修的境界，已经是达到了生死境七重的地步，他身上展现出来的气息，更是要远超什么夜天君，冷面魔君等人。
<br/>　　可以说将此人放在北域，若是没有楚风眠的话，只怕是足以横扫整个北域。
<br/>　　绝对是一尊在九域中，都排的上号的天才，此人身上凌厉围绕的凌厉的剑意，将他的实力，展现的淋漓尽致。
<br/>　　能够降服七道无上剑气，已经是足以证明此人的身份。
<br/>　　在对抗这四十二道无上剑气，组成的杀阵的时候，也是以此人为首，才足以抗衡。
