        &lt;h3&gt;第三章第四十一章  子昂，祝你幸福&lt;/h3&gt;
<br/>
<br/>        陈子昂在洛城。
<br/>
<br/>        她并不知道张函的想法，从那次见了张函之后，她的心慢慢的冷下来了，坚持了那么久的信念突然就失去了方向。
<br/>
<br/>        傅海生回到洛城并没有急于和陈子昂见面。
<br/>
<br/>        他得捋一捋，他得想一想。事情可能不会那么简单，在省城帮助夏颜母女的是谁？那些人现在如何了？
<br/>
<br/>        这件事情还是先不能告诉陈子昂。
<br/>
<br/>        他得先打听清楚事情的由来，该如何进行。
<br/>
<br/>        第二天，陈子昂和江俞轩、陈子霖来找他了。
<br/>
<br/>        “俩位陈总、江总，你们来了，快请坐。”傅海生文质彬彬，客气的招呼着。
<br/>
<br/>        “傅总好，真是抱歉，我很久没有回去看父母了，回家心切。”陈子昂也客气的说。
<br/>
<br/>        “应该的，应该的，我国以孝为先。”
<br/>
<br/>        “希望我们这次来没有给傅总增添麻烦。”
<br/>
<br/>        “怎么会有麻烦呢，你们是来帮助我单位发展的，我感谢都来不及。”傅海生招呼他们坐下后接着说：
<br/>
<br/>        “江总和我聊了很久，陈总，你真是让我佩服啊，原来如此，现在也如此。”
<br/>
<br/>        “傅总真会说话，子昂就是一个农民，仗着政策利好，沾了政策的光，要不然还在泉城上弯种地呢。”陈子昂说的是实话，但是听在傅海生的耳里似乎就是话中有话了。
<br/>
<br/>        莫非，陈子昂知道蹊跷了？
<br/>
<br/>        “怎么会呢，陈总永远都是走哪里都是会闪光的人。”
<br/>
<br/>        “傅总，听俞轩说你对信息化发展有独到的见解，我们也想听听你的意见呢。”陈子昂改变了话题，她来不是听傅海生客气和赞美的。
<br/>
<br/>        “我是看你的论文受的启发。”傅海生笑着说。
<br/>
<br/>        “是这样，我们洛城有两个如我这样的单位，我们两家都是公立的。我们准备一起将两个单位的信息化都建起来，项目申请已经审批下来了，资金基本到位，我们虽然是小地方，小地方更是急切的需要发展。”傅海生谈起工作就像变了一个人。
<br/>
<br/>        “我们北方的信息化发展是滞后于南方的，相对来说推动起来也是比较困难的，没想到傅总的思想竟然如此超前。这次我们来的，也是想对我们家乡的发展出一点力，加一块砖。”陈子昂和傅海生就工作上进来了探讨。
<br/>
<br/>        他们不再是同学关系，而是以一种全新的合作方式进行了交流。
<br/>
<br/>        “我们现在是各部门都是单独进行数据统计，各窗口也是人工操作，且，对于电脑的操作仅仅在能用上，也是需要经过培训。我看了你们西晋项目，我其实还有个想法，希望比那个更精进一些，能不能将所有的统计在一起，进行整合？”
<br/>
<br/>        “我们也有这个想法，只是很多软件厂家和硬件厂家的匹配度不是非常的融洽，作为我们来说，怎么样做好软件接口，及和各厂家对接才是我们目前最需要解决的问题，你提到的有包括财务软件，我们会把这个考虑进去的，而且我们也在和做财务软件的有沟通，包括后期科研上面的。
<br/>
<br/>        其实，现在的厂家也如雨后春笋，都很多，作为我们供应商，不但要有实力，考虑软件适合不适合用户，售后服务和后期的维护开发是必须要考虑到的。我们一直在致力与后期的售后和后期的开发工作，后期我们会将管理和临床科研一体化，不过这还需要一个漫长的过程，少则三五年，多则十来年，这要看整个社会的形态发展。”
<br/>
<br/>        “你们不能全部都做起来吗？”傅海生问。
<br/>
<br/>        “社会在发展，信息化其实是一个整体的形态，各行各业都在齐头并进，我们这个行业也是如此，没有一个人或者是一个集体能全部把大大小小的东西做起来，只能是整合资源。”陈子昂笑着说接着说：
<br/>
<br/>        “如何整合资源，那就要看我们的发展眼光和合作能力了。怎么样在众多的厂家里寻找合适合作伙伴很重要。不过......”
<br/>
<br/>        “不过什么？”
<br/>
<br/>        “固然发展眼光重要，有些人为的因数不太好控制。”
<br/>
<br/>        “陈总，我们是老同学不错，但是，我有做事情的原则，我只要适合我单位发展的，别的原因我不会去考虑的。”
<br/>
<br/>        “傅总，这样很好，大家都有公平竞争的机会，你也能找到合适你的方案。那么傅总，也给我们一次机会，我安排人调研一下咱单位的情况，也给出一份方案？”陈子昂含笑看着傅海生。
<br/>
<br/>        傅海生对陈子昂的笑没有任何免疫力，虽然他极力的掩饰着自己那跳跃的、欢快的小激动，以工作的名义和陈子昂聊着天。
<br/>
<br/>        天知道，他是多么渴望和陈子昂多呆一会，他似乎回到了高中时代，陈子昂那红嘟嘟的嘴巴，陈子昂那自信的倔强的表情，陈子昂那温柔的娓娓道来，她还是原来的那个她，似乎什么都不曾变过，可是一切都变了。
<br/>
<br/>        初恋是美好的，只是他傅海生的初恋是单相思，是暗恋，原来不曾说出口，现在也没有办法说出口，他们都不再是十八岁。
<br/>
<br/>        看着美丽如旧却又成熟了很多的陈子昂，她退去了青春的羞涩，宛如一颗熟透的水蜜桃，她睿智了很多，那双明亮的丹凤眼还是清澈，只是多了一丝沧桑的味道。
<br/>
<br/>        他想起了给她涂药的情景，他的脸无端的火烧火燎起来，还好，陈子昂那个时候是懵懂的，什么都不知道，要不然，现在很难堪。
<br/>
<br/>        陈子霖和江俞轩一直都是透明人，他们听着俩个人谈论着，看着俩个人客客气气的说着冠冕堂皇的话，突然看见，傅海生看着陈子昂那温柔似水的眼神，还有，他的脸，竟然红了！
<br/>
<br/>        “啧，人模人样的，还不知道心里想啥呢。”陈子霖狠狠的瞪着傅海生，如果说，眼神能杀人的话，傅海生此时不知道死了多少次了。
<br/>
<br/>        “装，装，终于装不住了，看那羞涩的小模样，说不定一会就又会抱头痛哭。”江俞轩鄙夷的看着傅海生。
<br/>
<br/>        陈子昂也发现了傅海生突然脸红了，她很奇怪，他的脸怎么会突然红了，现在也不热了啊。
<br/>
<br/>        她也用手摸了摸自己的脸。
<br/>
<br/>        电光火石间，她也想到了傅海生给她涂药的情景，那一次，结束了她和他三年的高中生涯，他们天各一方。
<br/>
<br/>        青春已散场，我们都安好，傅海生，谢谢你陪我那三年的高中时光。
<br/>
<br/>        “傅总，那我们就不打扰你了，我安排人过来采集一下咱单位的数据，我们后期给出个方案。”陈子昂站起来告辞。
<br/>
<br/>        “子昂，下午我们一起吃饭吧？”
<br/>
<br/>        “不了，我还要赶回西城，那边还有人等，另外，傅总，后期你这边的项目由陈子霖来对接。”
<br/>
<br/>        “那么，你不来了吗？”
<br/>
<br/>        “我有可能要出长差，会偶尔来。”陈子昂微笑着。
<br/>
<br/>        “子昂......”刚见怎么就要分开啊？
<br/>
<br/>        “海生，后会有期！”
<br/>
<br/>        “后会有期！”
<br/>
<br/>        傅海生失落的送走了陈子昂。没想到，他们见面竟然是这样。
<br/>
<br/>        子昂，祝你幸福！
<br/>
<br/>        过去的已经过去了，陈子昂不愿意纠结在过去的事情上，和傅海生永远就是哥们，既然如此就不要给傅海生任何幻想。
