        &lt;h3&gt;第三卷第三百一十二章  爱心代言人谷焕焕&lt;/h3&gt;
<br/>
<br/>        开完会，陈子昂留下了宁雅和谷焕焕。
<br/>
<br/>        “宁雅，工作都布置好了吗？”陈子昂叮咛着宁雅。
<br/>
<br/>        “放心，不会有任何纰漏，一切准备到位。明天全国联动，而且倪小美和李凌菲已经拿到了好几个单子，明天会在新闻发布会上签下外贸出口的合同。”宁雅看着陈子昂，似乎这一段时间她又瘦了很多。
<br/>
<br/>        “好，这一段时间辛苦你了。”陈子昂笑着对宁雅说，那一展颜，宁雅觉得天都是亮的，她的笑让人莫名的安心。
<br/>
<br/>        陈子昂知道宁雅要走，也知道宁雅在两头跑着，更知道宁雅在宁家受到的委屈，所以她的心里还是很心疼宁雅的。
<br/>
<br/>        “一句辛苦”让宁雅心里又欣慰又感动，这个世界上只要有一个人懂你就够了。
<br/>
<br/>        “最辛苦的还不是你，你看看你都瘦了很多，以后工作再忙你也要照顾好自己。”宁雅切切叮咛。
<br/>
<br/>        “知道了，美女妈婆，你去忙你的去吧，我和焕焕说几句话。”陈子昂笑着赶宁雅走。
<br/>
<br/>        “谷焕焕，甸城谷家，谷老大的侄女，父亲谷修瑾在杭州某部门任老总，你说你富二代一个潜伏在我公司搞慈善，不受委屈吗？”陈子昂笑着说。
<br/>
<br/>        “什么都瞒不过你，哼！我想干我自己喜欢的事情不行吗？”谷焕焕噘噘嘴。
<br/>
<br/>        “行啊，怎么不行啊，只是你隐藏的够深啊，你说你是为了自己还是为了谷家？”陈子昂依然笑眯眯的说。
<br/>
<br/>        “我从家里跑出来不久就到我们公司来了，原本家里不让我在公司干下去，但是他们看我做的是慈善事业，我们公司又搞得很好，而我似乎也成了公司的招牌，所以他们也就不说什么了，相反还很支持我。”谷焕焕低着头。
<br/>
<br/>        父亲的话她还记得：“你做的这个事情很好，你伯伯在甸城是有名的大慈善家，你虽然是在别人的公司，但你也在做这个事情，这几年你的名声已经很亮了，借助了陈子昂的这个平台，你已经成长起来了，不错，不错，比你在有些部门上班要好很多。”
<br/>
<br/>        “焕焕，做慈善不是做给人看的，我们公司不光光是你一个人在做，是整个公司的工作人员都在做，你代表着他们，而他们也是你，我相信你是没有任何目的在工作。”陈子昂一如既往地微笑着。
<br/>
<br/>        “我没有目的，但是不代表别人没有目的，我知道你会怀疑我，但是陈总，自从我加入到这个公司以来，自从我接手这份工作以来，我已经是我们公司的人了，虽然我流的是谷家的血液，我.......”
<br/>
<br/>        “那么着急干嘛？我都说了你是没不带任何目的的在工作，我当然是相信你的啊，要不然那么多资金供你随便的去搞项目？”陈子昂低头看着谷焕焕，她比谷焕焕本身高了很多，现在又穿着高跟鞋，谷焕焕要抬头看着她。
<br/>
<br/>        “我很喜欢现在的这份工作，这让我感觉很满足很幸福，每当我看到得到帮助的人开心的笑着的时候，我觉得我活得有意义，我才觉得我是我自己，陈总，您放心，我不会背叛我们的事业也不会背叛公司的。因为是您给了我这样一个平台，让我找到了自己的价值，谢谢您。”谷焕焕不知道为什么总是想证明自己不会背叛公司。
<br/>
<br/>        “傻姑娘，谁说你会背叛公司的，我只是希望你越做越好，保持你的初心，不被这个世界上有些俗事所干扰。我也希望你永远是我公司最美的代言人！”陈子昂笑着搂了搂谷焕焕的肩膀。
<br/>
<br/>        谷焕焕长得很好看，双眼皮下一双美丽的杏仁眼，瓜子脸，高挺的鼻梁，像樱桃一样泛着丰润光泽的小嘴此时微微的撅着，显得又可爱又娇媚。
<br/>
<br/>        陈子昂看得不由想逗逗谷焕焕：“你这一副小鸟依人，委屈扒拉的神情让我这颗小心脏都在噗嗤噗嗤的跳个不停，要是换了别人，怕不是早都被你俘虏了。”
<br/>
<br/>        “那我怎么就俘虏不了你呢？”谷焕焕也笑了，心里那种隐隐的担忧被陈子昂一逗竟然消失得无影无踪。
<br/>
<br/>        “你不晓得同性相斥，异性相吸吗？我和你一样是地地道道的大美女。”陈子昂似乎心情很好，开起了玩笑。
<br/>
<br/>        “哇，陈总，我就发现这个世界上要说自恋的人那非你莫属。”谷焕焕夸张的笑。
<br/>
<br/>        “自己都不恋自己，你还指望谁恋你？要想别人爱需得自爱，要对自己充满信心，充满爱！”人生导师陈子昂上线了。
<br/>
<br/>        “你说得好有道理。”谷焕焕点点头，表示赞同。
<br/>
<br/>        “对了，听说你琴棋书画都很好，尤其是喜欢作画，是不是真的？”陈子昂漫不经心的的问谷焕焕。
<br/>
<br/>        “还行吧，小的时候家里人逼着练琴跳舞，画画是我自己的爱好，上班之后就留了一个爱好，别的都已经快要丢失了。”
<br/>
<br/>        “有这么好的才艺还是要保持的，你小时候是在甸城长大的吗？”
<br/>
<br/>        “没有，我父亲从小就出来了，一直在外面上班，基本都不回甸城，不过过年过节还是会回去看看的。”
<br/>
<br/>        “那你甸城还有什么人吗？”
<br/>
<br/>        “我大伯啊，甸城最大的慈善家，呵。”
<br/>
<br/>        “那你们兄妹几个人啊？”
<br/>
<br/>        “我大伯只有一个女儿，我父亲也只有一个女儿，倒是大伯几年前收了一个儿子，谷强。”
<br/>
<br/>        “你们关系都很好吧？”
<br/>
<br/>        “一般吧，我不怎么去甸城，关系也好不到哪里去，只是我觉得谷强很傻，明明我大伯是利用他，他还乐此不彼。”谷焕焕到豆子一样啪啦啪啦说个不停。
<br/>
<br/>        “你怎么知道他是利用谷强呢？”
<br/>
<br/>        “因为，我有一次听见........”谷焕焕突然就不说话了。
<br/>
<br/>        “怎么了？”陈子昂问。
<br/>
<br/>        “不怎么了，陈总，你去甸城了还是尽量少和谷强他们打交道。”谷焕焕立马一脸愁容。
<br/>
<br/>        “你知道我这次是和谁合作的吗？”陈子昂笑眯眯的看着谷焕焕。
<br/>
<br/>        “不知道！”
<br/>
<br/>        “谷家，谷强和谷正娟，你的堂哥和堂姐！”
<br/>
<br/>        “反正你注意点就是，你好好的做正柜的生意，我们公司都是合法合规的事情，你能拿出来一部分资金做慈善，这几年我也看出来了，你是实实在在的想做事情，不像有些人道貌岸然。希望我们的公司越来越好吧。”谷焕焕长长的叹了口气。
<br/>
<br/>        “焕焕，你这话里有话啊。”陈子昂还是一直在笑。
<br/>
<br/>        “反正谷强一直是被利用的对象，我可不想你也是被利用的对象！好了，陈总，我还有事情要做，我先去干活了。明天我也有好多事情要干呢。”谷焕焕心思沉重的主动结束了和陈子昂的对话。
<br/>
<br/>        陈子昂看着离去的谷焕焕微微一笑。
