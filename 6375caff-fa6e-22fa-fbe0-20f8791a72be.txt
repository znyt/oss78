富二代的手直接僵在了半空，再也前进不了半分，他抬眸，一下子就看到了走来的顾夜瑾。
<br/>
<br/>        顾夜瑾身上一件黑色大衣，逆着光伫立，他俊美如玉的容颜镀在昏暗的光线里，看不真切，但他清寒的眸子落在富二代的脸上，眸底如扑洒开的墨汁，寒冷如万丈深渊。
<br/>
<br/>        富二代只觉得头皮一麻，迅速收回了自己的手，“顾…顾少，你误会了，我…我对叶美人没有任何想法，刚才只是…只是想她握个手…”
<br/>
<br/>        顾夜瑾走过来，走到了叶翎的身边，他漫不经心的开腔道，“哪只手想跟她握手的，就自己废了哪只手。”
<br/>
<br/>        富二代的脸色“刷”一下全白了，相比低调神秘的陆寒霆，海城真正的簪缨世家是顾家，顾夜瑾作为这一代的少主手段雷厉风行，让人从骨子里的敬畏害怕。
<br/>
<br/>        “你们慢慢玩，我先走了。”叶翎没有兴趣再看下去，所以她拔腿离开了。
<br/>
<br/>        顾夜瑾看着叶翎的俏影，迅速跟了上去，临走时他幽幽道，“自己动手，如果让我派人动手，我会让你知道废你一只手已经是我对你最大的仁慈。”
<br/>
<br/>        很快，身后传来了那个富二代自废右手的惨叫声。
<br/>
<br/>        ……
<br/>
<br/>        叶翎走到转弯时她纤细的皓腕被身后探来的一只大手一把拽住了，“翎翎。”
<br/>
<br/>        叶翎停下脚步，抬眸看向顾夜瑾，勾了一下红唇，“哥，你找我有事？”
<br/>
<br/>        顾夜瑾看着她唇角边的假笑，低声训斥道，“以后不许再来这种地方玩，现在跟我回去！”
<br/>
<br/>        “为什么，哥，我已经成年了，为什么不能来酒吧玩，夏妍妍也来了，你怎么不管管？”
<br/>
<br/>        “你跟她不一样，你长得多招摇，自己心里没点数？”顾夜瑾冷声打断了她的话。
<br/>
<br/>        叶翎垂下了蒲扇般的羽捷，“哦，我就当哥在夸我了，谢谢了，现在可以放开我了么？”
<br/>
<br/>        顾夜瑾将她巴掌大的明媚小脸尽收眼底，视线往下看了看她今日的穿着还有露出来的一双美腿。
<br/>
<br/>        叶翎自然捕捉到了他的目光，他的目光跟刚才那个富二代并没有什么不同，她用力的抽回自己的皓腕，“不要拿一副你想上了我的眼神来看我！”
<br/>
<br/>        顾夜瑾似笑非笑，“那你以后不要这样穿，长得招摇并不是你的错，但是穿的还招摇你就显得很故意。”
<br/>
<br/>        叶翎回了一句，“淫者见淫！”
<br/>
<br/>        “如果刚才我没有赶过来，你打算怎么办？”
<br/>
<br/>        “不会的，你一定会赶过来。”说完，叶翎直接转身离开。
<br/>
<br/>        但是顾夜瑾骨节分明的手指扣上了她莹润的肩头，将她用力的往墙上一推，将她堵在了自己的怀里。
<br/>
<br/>        叶翎纤柔的美背直接撞上了墙壁，很疼，她水媚的眸子望着男人，“疼，你将我弄疼了。”
<br/>
<br/>        这个女孩是他一手养大的，宠大的，没有人比他更清楚她有多娇气，受不得一点的疼。
<br/>
<br/>        现在脑袋里突然闯入她18岁生日那一天，在他的房间里，她软糯的往他怀里钻，一双水眸全是泪，告诉他，她有多疼。
<br/>
<br/>        顾夜瑾上下滚动着喉结，清寒的黑眸染上了一层情愫的猩红，“别对我喊疼，疼，就忍着。”
<br/>
<br/>        叶翎拧眉。
<br/>
<br/>        “翎翎，不要仗着我宠你就为所欲为，恩？”
<br/>
<br/>        叶翎看着他，“我倚仗的就是你对我的宠爱，如果哪一天我看出你是真的不宠爱我了，我自然就不会为所欲为了。”
