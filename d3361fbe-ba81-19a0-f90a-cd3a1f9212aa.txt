　　江凌轩一下子愣住，步纱这话说的理直气壮，让他根本无力反驳。
<br/>　　良久后，江凌轩开始尝试说服：“你救了我，说明你对猎者联盟还是有感情的，可为何不肯传递一下消息？难道猎王跟你有仇？”
<br/>　　“你说反了。”步纱冷漠开口：“我对猎者联盟没有感情，猎王对我有恩，我也不想救你，只是你命大出现在我屋外，我可不想一具尸体就这么腐烂在外，空气会不好。”
<br/>　　这理由让江凌轩无言以对，但也猛然察觉到了什么，惊讶道：“猎王对你有恩？那你不是更应该帮他一把么？”
<br/>　　步纱显然有些不耐烦：“他快死了么？还是他有危险？”
<br/>　　江凌轩噎住，过了好久才冷静下来，就想要挣扎着起身：“那行，我亲自去。”
<br/>　　步纱的笑容灿烂：“可以，大概还没走到一半，你就会死在路上。”
<br/>　　江凌轩的动作顿住，他知道步纱说的是事实，可眼前这个女人根本油盐不进，消息怎么才能传到猎王那里？
<br/>　　总部被毁已经够严重，江凌轩担心猎王会注意不到其中的阴谋。
<br/>　　忽的，江凌轩又猛然想到了什么，双眼噌亮道：“我怎么忘了白家大小姐，她当时也在场，太好了，她应该会去通知的吧？”
<br/>　　步纱偏过头看向窗外风景，声音冰冷：“关她什么事。”
<br/>　　此话一出，任江凌轩脾气再好也忍不住崩溃，他忽然发现自己大错特错还错的没边，竟然跟眼前这个古怪的女人说了这么一通废话。
<br/>　　关我什么事。
<br/>　　关她什么事。
<br/>　　这两句话简直无敌！
<br/>　　也同时让江凌轩清晰的了解到，这消息恐怕在他恢复到能出行之前，都没可能被猎王知道！
<br/>　　这个步纱，还有这个白允儿，到底是从哪冒出来的怪胎？
<br/>　　——————
<br/>　　西部高原上的小基地内，叶永阳已经带着叶家人在这里住下，每日劳作为将来的生活谋划，基地的首领杨飞杰将这一切都看在眼中，也对叶永阳等人颇为照顾。
<br/>　　一个年迈的老人跋山涉水，还带着一家子人，最神奇的是还走到了目的地，哪怕有再多目的这也值得人尊敬。
<br/>　　“首领，有人来投奔。”忽的一人走进屋，想杨飞杰汇报。
<br/>　　杨飞杰并没有太在意，道：“正常流程问清楚后，就安排住处。”
<br/>　　“问了。”那人表情古怪，有些迟疑的说道：“是两个人，其中一人也姓叶，叫叶子博。”
<br/>　　杨飞杰一愣，皱眉道：“跟叶永阳有什么关系？”
<br/>　　“我也问了。”那人回答：“他一来就问叶永阳在不在这里，自称叶永阳是他堂叔。”
<br/>　　杨飞杰点点头：“那就安顿进来。”
<br/>　　“是。”那人应着，忽的又迟疑道：“可是那人提了一个要求，说是不让我们告诉叶永阳他在这里。”
<br/>　　“这么奇怪？”杨飞杰愣了愣：“问原因了么？”
<br/>　　那人道：“说是叔侄俩有过矛盾，叶永阳曾将他逐出叶家。”
<br/>　　“还有这种事？”杨飞杰笑道：“这种家事我们也不好多问，那就让他自己单独住个地方。”
<br/>　　于是在叶永阳毫不知情的情况下，在很多人视野里都失踪的叶子博，就这么进入了西部的这间高原基地。
<br/>　　环境简陋的小屋内，叶子博一脸阴沉：“这么脏这么乱！”
<br/>　　与他一同前往此处的一人谄媚的笑道：“大人，这也没办法啊，谁让上边给我们的任务呢！”
<br/>　　“是我的任务，不是你的！”叶子博怒瞪着他，道：“若不是我废了一条胳膊也没办法提升等阶，我还需要你这个废物一路跟着？”
<br/>　　“是是。”那人低眉顺眼的点着头：“我这不是为了保护您的安全嘛，还麻烦大人您能在白少爷面前，多说两句我的好话。”
<br/>　　叶子博眼神闪烁不定，眼前跟着他的这人是他花了大价钱找来的，就是为了此行能保护他的安全，避免出现意外。
<br/>　　这人名叫张玄，别看他此时对叶子博毕恭毕敬，一副跟班的样子，实际上此人心机颇深。
<br/>　　张玄是他的真名，但叶子博却知道他还有一个身份，就是猎者联盟的高级猎者，只不过相对于大部分猎者来说，张玄这个猎者更加边缘化，也极端化，尤其是叶子博查到的资料上，张玄大部分接下的任务都是杀人，还是杀那些基地内的高层家属。
<br/>　　每一个基地内都有勾心斗角，亦或是权势争夺，发生在小圈子内的阴谋不断，张玄就利用了人类的阴暗面赚钱，做的买卖更是价格极高。
<br/>　　因为此人没有原则也无底线，什么人都杀，只要给钱，哪怕是孕妇、孩童也不放过。
<br/>　　拿钱办事，替人消灾。
<br/>　　如此心狠手辣的人在猎者联盟中并不算多，相对于猎王管理下的那些猎者们来说，张玄的模式更像是个杀手，一个只认数据和金钱的杀手。
<br/>　　既然是任务，甚至还有很多是内容不公布的任务，又没有触犯到猎者联盟的原则，张玄的行·事作风也无人能指手画脚，哪怕有人略知一二他的秉性，也不好多说什么。
<br/>　　猎者联盟本身就是一个自由中立的组织，你能往上爬是你的本事。
<br/>　　而张玄这样的人，反而在各式暗杀任务中相当吃香，找他杀人的任务一项接一项，甚至还有很多人不惜开高价请他去做保密任务。
<br/>　　张玄便因此在猎者圈中·出名，他没有朋友，也不需要朋友，他要的只是不断的杀人，以及不断的谋利。
<br/>　　叶子博正是看上了他这一点，才狠下心将其请来一路护送，早在南部联盟军北上的时候，叶子博就已经察觉到大势已去，好几次险些被狼牙的人活捉后，他便立即想到了之前让顾凉晨前往西部找人的事。
<br/>　　只是顾凉晨这么一去，就彻底没了音讯，叶子博只得再次求上白家，请求白忧给他翻阅情报的权利。
<br/>　　至于叶永阳前往西部的事，叶子博却留了一个心眼，并没有对白忧具体说明。
