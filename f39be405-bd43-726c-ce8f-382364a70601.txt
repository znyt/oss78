　　写字楼顶楼，指挥中心。
<br/>　　指挥中心中所有人员全都停下了自己手上正在忙碌的工作，站到了窗户前，看着传来爆炸的地铁站方向。
<br/>　　爆炸所产生的冲天火光此时已经退散掉，但是夜空中，还是能看到在那片地方有着火光在跳动。
<br/>　　除了这不断跳动，忽明忽暗的火光外，隐隐中还有充满恐惧的惊叫声，撕心裂肺的哭喊声传来。指挥中心中每一个人都能从这些声音中，听出来一种叫做“绝望”的东西。
<br/>　　赵天宇深吸一口气，通过无线电再一次向罗昊询问情况。
<br/>　　无线电中沉默了差不多有十几秒钟的时间，罗昊低沉的声音从无线电中传了过来，“智天使在这个城市中安放的炸药，就在刚才，K国快速反应部队负责监控的狙击手自行向智天使射击，使得智天使引爆了炸药。”
<br/>　　“愚蠢！混蛋！饭桶！”
<br/>　　赵天宇愤怒的将对讲机仍在地上，砸落在地板上对讲机弹跳而起，打在墙壁上又反弹回来，在地面上滚了两圈，对讲机的电池盖板弹飞起来，一节电池也是弹了出来。
<br/>　　云熙弯腰捡起滚落在自己脚边的对讲机，把弹起的电池重新压下去，装上电池盖板，打开对讲机递给赵天宇。
<br/>　　赵天宇接过对讲机，对罗昊吩咐道：“立刻将那个狙击手控制，我会跟K国方面联系，说明情况。”
<br/>　　“明白！”
<br/>　　赵天宇放下对讲机，对李然说道：“妖瞳，你去一趟现场。”
<br/>　　李然点了点头，转身离开了指挥中心。
<br/>　　李然独自一人驾驶着一辆汽车离开了写字楼，朝着发生爆炸的地铁站驶去。地铁站距离指挥中心的写字楼并不远，也就三个街区的距离，只是李然的车子没能靠近地铁站，因为在距离地铁站还有五百米左右的位置，路就被封了，除了特殊车辆，其余汽车全部不能靠近，路上被拉起了警戒线，警戒线外站着几名黑色防弹背心上印着白色“SWAT”字样的SWAT队员，他们手中握着G36自动步枪，眼神警惕的看着来往的人员。
<br/>　　拉开储物格，里面放着一把P226手枪，李然把枪拿了出来，想了想最后还是把枪又放回了储物格中，推门下车。
<br/>　　警戒线前，一名SWAT队员将李然拦了下来，李然拿起挂在胸前，K国政府专门准备的特别通行证，给那名SWAT队员看了一眼，很快就得到了放行。
<br/>　　地铁站附近可以说一片狼藉，满目疮痍，马路中央被炸开了一个大口子，站在旁边探头往下看去，能看到下面被炸断的地铁，还有横卧在站台上，被炸得支零破碎的尸体。
<br/>　　路边停着警车、消防车和救护车，爆闪灯不断的在闪烁着；身上沾着血迹的救护人员用担架把受伤较重的伤员抬上急救车，路边的花坛上也坐着好几个伤员，只是他们伤得并不算重，而且急救车数量紧张，所以只能优先考虑重伤员，这些轻伤员经过简单包扎后被安排在旁边休息，等待后续赶来的急救车送去附近医院接收治疗。
<br/>　　地铁站的电源系统已经被彻底破坏，地铁站内一片漆黑，进入地铁站搜救的消防人员头盔上都亮着灯，他们身上不仅穿着防护服，因为考虑到现场会有有毒气体，所以他们还背上了氧气瓶。
<br/>　　今天晚上，对于这座城市来说，是一个灾难之夜；尽管还没有统计出具体的伤亡人员数量，可是从马路上那个被炸塌的大坑，李然已经可以基本判断出爆炸当量，结合当时地铁站内的人流量，大致对此次爆炸造成的伤亡情况作出一个推断，而李然所作出的推断还是一个保守数值，实际情况，比起自己所做的推断，只会高不会低！
<br/>　　……
<br/>　　目标写字楼外，罗昊还是站在他原来的位置，抬头看着那面被子弹击碎，空出来的玻璃墙，地面上洒落的玻璃碎片，反射着写字楼内的灯光，很刺眼。
<br/>　　那个自作主张开枪射击的狙击手根本不需要控制，在地铁站爆炸发生的那一刻，他就已经被吓傻了，别说是逃跑，他连站起来的力气都没有，整个人就像是被抽空了一样，趴在地上，目光空洞的看着发生爆炸的地铁站方向。
<br/>　　因为自己这一枪，今夜不知有多少无辜的人丧生在这场爆炸中，他们中间有跟自己一样，是孩子的父亲，是父母的儿子，是妻子的丈夫。曾经他为自己是一名狙击手而感到自豪，此时他不住发抖的双手却是连枪都握不住，当然他现在面前已经没有狙击步枪了，就连他大腿快拔枪套中的手枪也是被人给卸掉了。最后一颗子弹，带着他盲目骄傲和自信的这颗子弹在打碎那块玻璃墙的同时，也击碎了他的一切，家庭、前途，还有他最重视的骄傲和自豪。
<br/>　　K国快速反应部队的司令官亲自赶到了现场，跟随他一起来的，还有在半个小时前刚成立的调查小组。
<br/>　　司令官对着罗昊敬了一个礼，说道：“我很抱歉手下的兵，惹了这么大的麻烦……”
<br/>　　“司令官先生，你不应该对我说抱歉。”
<br/>　　司令官的话还没说完，就被罗昊打断了，罗昊指着爆炸发生的地铁站，说道：“你应该对那里的人说对不起，那颗子弹毁了多少无辜的家庭，您或许应该去了解一下，然后对着他们鞠躬道歉，这样或许你晚上不会因为自己良心的不安而辗转难眠。”
<br/>　　司令官没有因为罗昊的话而发怒，他沉默着站在一旁，现场所有的认证都能证明，在事发的时候，他眼前这个中校明确下达了“待命”的命令，但是是他手下那个狙击手擅作主张开枪射击，导致了爆炸的发生。
<br/>　　肇事狙击手被调查小组拖着往押送车的方向走去，他裤子裆部已经完全湿了，并且散发出一阵阵的骚臭味。在从罗昊面前经过的时候，他要求身边架着他双手的调查小组成员停了下来，缓慢的转过头，看着罗昊。
