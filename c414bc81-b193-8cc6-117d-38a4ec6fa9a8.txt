　　青俞皇帝震惊的站了起来，眼眸呆滞的望着苏莫，心中惊骇莫名。
<br/>　　一方面他是震惊苏莫的实力，能轻描淡写的一招秒杀汪晟，估计只有万尊榜前二十名之人，才能做到吧！
<br/>　　汪晟虽然是武王境九重武者，但作为万尊榜排名第六十二的年轻至尊，战力估计不会比他弱多少。
<br/>　　另一方面，青俞皇帝心中惊恐了起来，汪晟是北海冰宫最得意的弟子之一，死在了他青俞皇宫，他青俞帝国根本无法承受北海冰宫的怒火。
<br/>　　青雅宫主同样震惊的傻了，她只是抱着试一试的打算，用苏莫来当挡箭牌。
<br/>　　但是，她哪里会料到，眼前看似平凡的青年，居然是如此恐怖的人物！
<br/>　　一指头，就能秒杀万尊榜排名第六十二的年轻至尊！
<br/>　　这种实力，比她父皇都要强大十几倍都不知吧！
<br/>　　因为，她曾和汪晟是同门，心中清楚的知道，汪晟的那柄蓝色长刀，乃是皇级下品神兵。
<br/>　　一指头洞穿皇级下品神兵，这是怎样的逆天实力？
<br/>　　就算是北海冰宫的大师姐，万尊榜排名第二的慕冰雪，也没有这种实力吧！
<br/>　　一时间，青雅震惊的愣在了原地，脑中一片空白。
<br/>　　短暂的寂静过后，广场上突然沸腾了起来，一片哗然。
<br/>　　“天呐！汪晟死了！”
<br/>　　“秒杀，一指秒杀了！”
<br/>　　“这家伙是谁？难道他不怕北海冰宫吗？”
<br/>　　“好恐怖的实力，太强大了！”
<br/>　　无数的人惊呼出声，形成震天的声浪，冲霄而起，浩浩荡荡。
<br/>　　嗖！
<br/>　　破空声响起，最后那名未死的汪晟同伴，身形拔空，一句话没说便急速逃离。
<br/>　　汪晟都没被秒杀了，此人哪里还敢逗留，立刻开始逃跑。
<br/>　　此人几乎是爆发了有生以来最快的速度，身形如同一道寒流，刮过虚空，瞬息间便消失不见。
<br/>　　“不好！”
<br/>　　少顷之后，青俞皇帝反应了过来，顿时脸色一变。
<br/>　　此人若是逃掉，消息立刻就会传回北海冰宫，到时候他青俞皇室很可能要面临灭顶之灾。
<br/>　　身形一动，青俞皇帝正欲前去追击，顿时又止住了身形。
<br/>　　只见苏莫击杀了汪晟之后，居然径直向看台走了过来。
<br/>　　苏莫没有理会那名逃走的青年，对方没有对他出手，他也懒得理会，逃不逃走对于他没有什么影响。
<br/>　　两步来到战台之前，苏莫凝视着那名鹰钩鼻老者，眸中一片冰冷。
<br/>　　那名鹰钩鼻老者早已站了起来，同样在凝视着苏莫，消瘦古板的脸庞上，充满了凝重之色。
<br/>　　“告诉我你的身份，我饶你不死！”苏莫淡淡开口，声音冷漠无比。
<br/>　　鹰钩鼻老者闻言一怔，眸中闪过一丝阴霾，随即脸色沉了下来。
<br/>　　因为苏莫的言语，太过霸道了，让人不容反抗。
<br/>　　众人见此，均是有些摸不着头脑，这个实力恐怖的青年，这是要干嘛？
<br/>　　怎么击杀了汪晟之后，又对国师发难了？
<br/>　　难道其不想做青俞帝国的驸马？
<br/>　　“这……这位小友，这位是我青俞帝国的国师！”青俞皇帝面带惊疑，为苏莫介绍鹰钩鼻老者的身份。
<br/>　　此刻，青俞皇帝心念急转，他现在一定要稳住苏莫，万不能得罪。
<br/>　　他清楚的知道，面前这位无名青年的实力，比他要强大的太多了，绝对不是他皇室能够抗衡的存在。
<br/>　　再者，这名青年杀死了汪晟，更不能让他一走了之，不然的话，他皇室就要受无妄之灾，为其承受北海冰宫的怒火。
<br/>　　总之，青俞皇帝心念电转，苏莫杀死汪晟，就要承担后果，不能让他们背黑锅。
<br/>　　但是，他也不能和苏莫撕破脸皮，因为苏莫的实力太强了。
<br/>　　“老夫屈覃，正是青俞帝国的国师，不知哪里得罪了小友？”鹰钩鼻老者开口了，向苏莫微微抱拳，声音低沉无比。
<br/>　　显然，他也知道苏莫的实力强大，不敢违逆苏莫的话。
<br/>　　“国师？”苏莫闻言微微皱眉，随即他催动幽冥魔瞳武魂，仔仔细细的打量了对方一番。
<br/>　　在幽冥魔瞳武魂之下，这屈覃的身体本质，被苏莫看的一清二楚。
<br/>　　此人体内玄力浑厚，尸气滚滚，气血也不弱，生机也是相当浓郁。
<br/>　　这就说明对方是修炼了尸功的武者，并不是吸收天地精气形成的尸魁之身。
<br/>　　“你为何会修炼尸功？”苏莫再次问道。
<br/>　　“老夫早年偶得一门尸功，感觉很是强大，便一直修炼这门功法！”屈覃不卑不亢，沉声道。
<br/>　　“是吗？”苏莫双眸微眯了起来，难道对方真的与黄泉魔宗没有半点关系？
<br/>　　“小友若是不信，大可以询问=问在坐的诸位！”屈覃面色镇定，沉声说道。
<br/>　　“这位小友，屈国师的确是偶得的尸功，是在一处遗迹中所获！”
<br/>　　“这不是什么秘密，我等都知晓！”
<br/>　　“屈国师修炼这门尸功也有四、五十年了！”
<br/>　　看台上，一些皇室的强者，闻听屈覃之言，纷纷开口了。
<br/>　　苏莫闻言沉默了，难道真的只是巧合？对方并不是黄泉魔宗尸罗殿之人？
<br/>　　“这位小友，不如我等去宫中详谈如何？”青俞皇帝望着沉默的苏莫，试探性的说道。
<br/>　　他要想办法将苏莫留下来，到时候北海冰宫强者来临，他才好向北海冰宫交代。
<br/>　　苏莫闻言，略一沉吟，便点了点头。
<br/>　　不管如何，既然来了，就要将此事了解清楚。
<br/>　　“请！”
<br/>　　随即，青俞皇帝亲自为苏莫引路，带着青雅公主、屈覃国师，以及一众皇室的强者，返回了皇宫之中。
<br/>　　广场上，无数的围观之人望着离开的苏莫，议论不休。
<br/>　　“这人到底是谁啊？”
<br/>　　“他有什么目的？不是为了做驸马，杀了汪晟，居然还要调查国师！”
<br/>　　“此人有恃无恐，估计是某个超级势力的强者！”
<br/>　　“哈哈！他杀了汪晟，居然还不逃走，待北海冰宫的强者来临，他就是身份再不简单，也吃不了兜着走！”
<br/>　　“我们就等着看好戏吧！北海冰宫距青俞岛并不远，最多三个时辰的时间，北海冰宫的强者，定然会到来！”
<br/>　　嘈杂的声音此起彼伏，广场上聚集了数十万人，没有人离开，均是在等着观看事态的发展。
