<br/>　　若是不堂堂正正、光明正大的将苏莫击杀，如何能洗刷这次的耻辱！

<br/>　　同一时间，内府的另一座山峰上，宫殿之中同样在议事。
<br/>　　“哈哈哈！这一次太史世家派系，真是丢进了脸面！”

<br/>　　殿中气氛非常的火热，笑声不绝于耳，不少人均是面带笑容。

<br/>　　“这个苏莫，倒是有点胆色！”主座之上的青年，嘴角挂着若有如无的笑意。

<br/>　　“殿下，我们现在要不要派人去拉拢苏莫，晚了怕是就被其它派系的人抢走了！”一名青年向古天意问道。

<br/>　　“不用！”
<br/>　　古天意微微的摇了摇头，嘴角挂着淡笑，道：“苏莫天赋虽然不错，但还不值得我们为了他，与太史世家派系正面交锋，再等等吧！看看他接下来的表现！”

<br/>　　还是先静观其变，看看太史世家派系接下来如何应对。

<br/>　　……
<br/>　　洞府之中，苏莫身上气息澎湃，正在静心修炼。
<br/>　　浑厚的五行玄力，如奔腾的长河在他体内奔流，让得他全身的气息不断的涌动。

<br/>　　修炼之中的苏莫，微微有些疑惑，过去了这么久，不仅太史世家派系的人并没有来找他麻烦，连其它派系的人也没有来拉拢他。

<br/>　　苏莫在洞府中端坐不动，嗤笑一声，继续道：“想邀战也可以，同阶武者，来多少我杀多少！”
