　　不周倒，天水现。浩劫降临人间，生灵涂炭，人们该何去何从？
<br/>　　人族所在，以人祖殿为中心，首阳山各山峰之间，赫然可见，一道庞大的光幕浮现，轰然之间，笼罩了整个首阳山地界乃至东海之滨，将人族保护住。好在，这段时间三族大战，人族部落收缩聚拢，都在首阳山附近，如此一来，却是正好借助人祖殿的守护，躲过以这次近乎毁灭的灾难。
<br/>　　不过，巫、妖两族就没那么幸运了。盘古殿附近的巫族倒好，从盘古殿中释放出一道巨大光幕，笼罩了周围的空间，将巫族保护在其中。与人族大战的巫族，也有各祖巫出手，暂时保护起来，但，其他地方的巫族，那可就没那么幸运了。
<br/>　　妖族更不用说了，散落在个洪荒大地上的妖族，却是没人会出手保护，他们如何抵挡这奔流不息的弱水冲击？转瞬间，便有数不清的生灵被淹没在那滚怒涛之中。
<br/>　　大陆之上的生灵在奔走相逐，以求生存。豺狼虎豹，毒蛇猛兽纷纷逃窜，却被身后的巨大浪涛卷入弱水之中，顷刻间便消融。
<br/>　　灾难降临，天道有缺，受天机牵引，但见天际之中六道光华闪现，紫气降临，正是元始三清、女娲娘娘、西方两位教主。
<br/>　　元始一步跨出，来到不周山天宇之上的大窟窿处，撑开体内世界，将这些弱水引进体内世界之中，他的体内世界众多，肉身细胞空间就堪比一个小世界，三百六十五个神窍空间，随着修为的突破，更是扩张到中千世界极限。如今他修为已经混元大罗金仙五重天，这些弱水还伤不到他。
<br/>　　他将这些弱水引进体内世界之中，由黑帝水皇炼化，化为自己的法力，专门有一个世界用来装这些弱水，不怕装不完，只怕不够装。
<br/>　　由天洞降下的无尽弱水，还没来得及落下，便被元始吸收，甚至那些已经流向洪荒大地的无尽弱水，有的都被元始施展大法力，逆流而来，引入体内世界之中。
<br/>　　六圣出手，拯救苍生。
<br/>　　元始负责堵住九天之上的大窟窿，牵引降落的弱水，太清圣人祭出太极图，化为一张巨大的天幕，保护洪荒生灵，通天祭出诛仙剑阵，梳理河道，将那些已经流到洪荒中的天水引入北海海眼。接引的十二品功德金莲，准提的七宝妙树，女娲的江山社稷图，各自有各自的手段，梳理弱水，拯救苍生。
<br/>　　良久，元始觉得吸收的差不多了，毕竟，再多的弱水对自己也没用了。他出手了，只见他双手对着那巨大的窟窿按了上去，玄光绽放，璀璨耀眼，他施展大神通，将那窟窿暂时封印了。
<br/>　　此刻，天河不再往下灌，原先那道巨大的窟窿已经消失不见，却是多了一道闪闪发光的封印。至于天庭之上的弱水，如何祸害天庭，那却不是元始管理的范围了。对他来说，妖族死再多，与他也没有关系。
<br/>　　看着九天之上那道封印，通天性子急，当即就出声问道：
<br/>　　“诸位，如今不周山倒塌，天柱折断，却不知天上这偌大的窟窿裂缝又该如何是好？”
<br/>　　望着天空之上的那个硕大的窟窿，一时之间，就连天道圣人，此时此刻，一个个也束手无策。
<br/>　　元始皱了皱眉头，缓缓道：“本座来将他补上！”
<br/>　　说完，他不再理会其他五位圣人那诧异的眼神，翻手之间，取出一尊古朴小鼎，正是极品先天灵宝乾坤鼎。
<br/>　　“道友真是好运气，竟能得到如此至宝？”
<br/>　　其他几圣眼光自是不错，见到此鼎，顿时便认出这是极品先天灵宝乾坤鼎，纷纷感叹元始好运气，竟然能够遇到这等灵宝。
<br/>　　“好运气？”
<br/>　　元始冷笑一声，便不再多语。真是好运气吗？天地间哪有那么多好运气，这可是他的分身混沌道君当初大战魔祖罗睺之时所收集的。
<br/>　　元始在此刻取了出来，自是不怕被道祖知道他与混沌道君的关系，以他如今的实力，也不怕被别人惦记。更不用说道祖只知道他与混沌道君关系不一般，也未必会识出他与混沌道君是同一人。
<br/>　　在众圣羡慕的眼神下，元始不理会其他，催动法力，霎时间，乾坤鼎在高空中变得巨大，涨大到百丈大小。随即，他取出数百块闪烁着九彩光芒的混沌晶石，那是他当年在混沌之中寻到的，平时他都是以此炼器，此刻，用来补天却要比什么五彩石要高级多了。
<br/>　　接着，元始将数百快混沌晶石投入到乾坤鼎之中。同时，他也祭出自己的神火，神火呈紫色，充满高贵，众人在其中感受不到丝毫的热量，仿佛这只是一朵普通的火焰。但众圣又知道，元始手中的又岂会是一般的火焰。准提圣人试着将一丝元神之力探入到火焰之中，想一窥究竟。
<br/>　　“咦！”
<br/>　　准提惊异一声，满上充满惊诧，赞叹道：“道友此火确实不凡呐！”
<br/>　　却是准提的那一丝元神之力刚入紫火之内，便被烧毁了。要知道，圣人的元神在成圣的那一刻，就已经经过天道之力的洗礼了，其中更是隐藏着一丝天道之力。就这这样一朵紫火，人家还没有用法力催动呢，就能毁灭一丝圣魂元神，真是不可思议！
<br/>　　别看一丝元神之力很少，但可别忘了，这可是圣人的元神，圣人的一丝元神，已经堪比准圣了，但就这样被一朵紫火给焚烧了。那岂不是在说，一般准圣到此，不用别的，只要将这一朵紫火扔了出去，便能将他烧死？
<br/>　　若是元始再使用大法力催动起来，那岂不是就连圣人都能威胁到？
<br/>　　想到此处，准提顿时就倒吸一口气，看向元始的眼神之中，充满忌惮。
<br/>　　其他人听到准提的话，也纷纷将一丝元神之力探入其中，但结局都是一样。顿时，他们觉得元始深不可测，不能招惹。
<br/>　　元始不理会其他，祭出神火后，便扔入乾坤鼎之中。顿时，浑厚的鸿蒙混沌气息自鼎中透出，几百颗混沌晶石在鼎中急速转动，九彩光芒大盛，元始催动造化法则，造化天地，玄光闪烁，如此锻炼，将混沌晶石返本归源，用造化法则不断改造晶石本质，将之炼化成玄清气。
<br/>　　“咦！这是……造化法则？”
<br/>　　女娲大吃一惊，本以为只有自己精通造化法则，谁知元始也精通，而且看样子，一点也不比她弱。这一刻，她看向元始的目光之中，充满警惕，要知道，同一种法则，最后只能有一人证道，如今她以造化法则证道成圣了，却发现元始也精通此道，怎能让她不警惕？
<br/>　　其他几位圣人也是瞳孔一缩，目光闪烁，显然，他们也非常吃惊。不过，谁也没有多说什么？这与他们无关，顶多就是以后少惹元始就是了。
<br/>　　元始不理会众人的惊异，依旧不急不缓，以造化之力在炼化晶石。
