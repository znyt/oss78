　　分身接受到苏莫的吩咐，立刻就开始行动了起来，再次开始散布消息。
<br/>　　没过多久，一个爆炸性的消息，便席卷天池各大界域，并且向其它神朝传播。
<br/>　　这个消息的内容就是，紫霄杀死了拓跋流明、云沐和古剑山。
<br/>　　此消息一出，可谓是轰动四方，震惊了无数人。
<br/>　　因为，无论是拓跋流明，还是云沐和古剑山，均是万古天骄，背后同样有大势力。
<br/>　　但是，这样的三名万古天骄，居然被紫霄杀死了。
<br/>　　据闻，当初秦陨从万道剑冢出来之后，没有多久，便被拓跋流明和云沐、古剑山三人截住。
<br/>　　秦陨虽然逆天，但是毕竟修为较低，根本不是拓跋流明的对方。
<br/>　　但就在最后关头，紫霄出现了，一招就灭杀了拓跋流明和云沐，以及古剑山。
<br/>　　这个消息，以无与伦比的速度，席卷了整个神域，至于消息的来源，以及没有人在意了。
<br/>　　……
<br/>　　“紫霄，欺人太甚！”
<br/>　　轮回神朝地王宫，大殿之中，愤怒的咆哮之声，震动整座大殿。
<br/>　　大殿中聚集了数十人，主座之上端坐着一名身材藏青色长袍，身形矮壮，面色黝黑的老者。
<br/>　　此人不是别人，正是地王宫宫主，人称‘地渊神王’。
<br/>　　此时此刻，地渊神王面色阴沉，眸中爆射出凌厉的精芒。
<br/>　　因为，拓跋流明身死的消息，已经传到了地王宫，而拓跋流明，乃是地渊神王的孙子。
<br/>　　“父亲，此事绝对不能善罢甘休，若是流明真的死于紫霄之手，紫霄必须偿命！”
<br/>　　下首，一名身穿宽大锦袍，身材同样矮壮的中年人，面上露出悲戚之色。
<br/>　　此人，乃是拓跋流明之父，地藏神王。
<br/>　　“宫主，紫霄和终极神王殿，必须付出血的代价！”
<br/>　　“我等应该离开前去终极神王殿！”
<br/>　　“流明不能白死！”
<br/>　　不少人纷纷开口，个个面现怒色，这些人全部都是地王宫的高层，有神王级别的强者，也有八品和九品的真神。
<br/>　　地渊宫主闻言，眸中的怒火稍稍收敛，他略一沉吟，道：“此事是真是假，还有待考证，不能鲁莽！”
<br/>　　地渊宫主并未被怒火冲昏头脑，终极神王殿的实力，虽然不如他们地王宫，但是也不弱，不能贸然开战。
<br/>　　“藏儿，你动手去终极神王殿一趟，必须将此事调查清楚，若是流明真的死于紫霄之手，为父会为流明报仇！”地渊宫主凝视着地藏神王，沉声说道。
<br/>　　他是想要亲自去终极神王殿的，但是，他却是无法离开。
<br/>　　毕竟，拓跋青云在闭关，若是他离开了地王宫，天王宫绝对不会放过这个机会，定然大举来袭，那就糟糕了。
<br/>　　“好，我这就去！”地藏神王点了点头，二话不说，便转身大步向外走去。
<br/>　　“藏儿，切不可鲁莽，免得和终极神王殿交恶！”地渊宫主嘱咐道。
<br/>　　此事不一定为真，在查清楚之前，不能和终极神王殿翻脸。
<br/>　　“父亲，我明白！”地藏神王头也不会，走出大殿之后，便立刻冲天而起，向终极剑域而去。
<br/>　　……
<br/>　　与此同时，天池万化界域，霸主级势力万化神殿，也发生了差不多的一幕。
<br/>　　古剑山，乃是万化神殿，近几万年来最杰出的天才，如今惨死，万化神殿自然不会善罢甘休，同样派出强者，前往终极神域。
<br/>　　雷焱界域，雷焱神王殿，是一个堪比终极神王殿的大势力，也是云沐所在的大势力。
<br/>　　云沐身死，雷焱神王殿，同样派出强者，火速前往终极剑域。
<br/>　　一时之间，整个神域的焦点，都被此事所吸引。
<br/>　　所有的矛头，均是直指终极神王殿，紫霄神王。
<br/>　　……
<br/>　　这一日，终极神王殿，迎来了数位贵客。
<br/>　　地王宫的地藏神王，万化神殿的赤云神王，雷焱神王殿的雷火神王。
<br/>　　终极大殿之中，终极神王端坐在高高的主座之上，下首，紫霄神王、毁灭剑王、风雪神王，依次坐于右侧。
<br/>　　地藏神王、赤云神王和雷火神王三人，端坐在左侧。
<br/>　　三位神王，来到终极神王殿的时间相差不多，终极神王便一起接见了。
<br/>　　“三位大驾光临本殿，不知有何贵干？”终极神王阴沉着脸，沉声问道。
<br/>　　虽然开口询问，但是终极神王，已经知晓了三人的来意。
<br/>　　因为，这段时间的传闻，他也早已听到了。
<br/>　　对此，他之前特意询问过紫霄，是否杀了拓跋流明等人，得到的回答自然是否认。
<br/>　　对此，终极神王也不能确信紫霄之言是否属实，毕竟，他也不能太过逼迫紫霄。
<br/>　　“终极殿主，我们来此，自然是找紫霄宫主！”地藏神王率先开口了，目光死死的盯着紫霄。
<br/>　　雷火神王和赤云神王，同样目光紧紧的盯着紫霄，仿佛是想要从紫霄的身上，看出点什么。
<br/>　　雷火神王和赤云神王，均是中年男子，身材都是颇为高大。
<br/>　　不过，前者面色黝黑，模样略显丑陋，后者肤色暗红，长着一双赤红色的眼眸，显得凌厉摄人。
<br/>　　此时此刻，紫霄的面色，无比的难看，他也早有预料，这三人是来找他的。
<br/>　　毕竟，现在外面的传闻，都已经传疯了。
<br/>　　“三位，拓跋流明和云沐、古剑山，有没有死我并不知晓，也从未见过三人！”紫霄阴沉着脸，环视眼前的三人。
<br/>　　他心中即愤怒又无奈，此事，他能肯定，必然是秦陨在搞鬼。
<br/>　　这一切，显然是有人在推波助澜，而始作俑者，不是秦陨还能有谁？
<br/>　　“你没有见过他们三人，那他们三人去了哪里？”地藏神王微眯着眼眸，想要看出紫霄是否在说谎。
<br/>　　“拓跋兄，拓跋流明三人去了哪里？我怎么可能会知晓？”紫霄皱眉反问道，他心中又气又怒，这简直是笑话，他怎么可能知晓那三个小辈的情况？
<br/>　　他对于眼前的地藏神王，也算是熟识，知晓其是拓跋流明之父。
<br/>　　“紫霄，你难道是要告诉我们，他们三人死在了秦陨手上？”赤云神王冷笑道，他知道紫霄不会承认，而他们的目的不是要紫霄承认，而是从紫霄的身上找到破绽。
<br/>　　“秦陨此人，不可能杀的了他们三人！”雷火神王道。
<br/>　　“三位，我紫霄做事，一向光明磊落，堂堂中级神王还不至于对三个小辈出手！”紫霄起身，面色郑重的环视三人，朗声说道。
<br/>　　“光明磊落？”
<br/>　　地藏神王闻言，冷笑一声，道：“那我且问你，你和秦陨是什么关系？”
<br/>　　“没有关系！”紫霄毫不犹豫的说道。
<br/>　　“没有关系？”地藏神王根本不信，继续问道：“根据各方调查，秦陨曾杀了你紫霄剑宫几名弟子，你为何不惩处他，反而将此事压了下去。”
<br/>　　“这是因为绫裳，秦陨和绫裳关系较好，所以我便没有惩处于他，让此事不了了之！”紫霄徐徐说道，他早有说辞，将此事推到绫裳的身上。
<br/>　　因为，此事，他根本无法解释，总不能告诉众人，他不惩处秦陨，是为了日后要挟绫裳吧！
<br/>　　“绫裳宫主呢？为何不出来一见？”雷火神王问道。
<br/>　　“绫裳在静修，现在本殿主便传唤于他！”终极神王朗声说道，此事，今日必须给予三人一个合理的交代，不然麻烦不小。
<br/>　　言罢，终极神王便命人传唤绫裳。
<br/>　　就在此刻，终极神王城上空，陡然有滚滚雷音，响彻而起。
<br/>　　“终极神王，天尘宇求见！”
<br/>　　朗朗的声音，如同天降雷音，响彻整个终极神王城，也传到了终极大殿之中。
