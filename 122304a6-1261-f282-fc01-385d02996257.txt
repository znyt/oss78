　　被蛇咬了？！
<br/>　　巫师眉头微皱，眼中闪过凝重之色。咬伤周柏的毒蛇必然是出自毒天使之手，而毒天使放出来的东西会是简单之物吗？显然不会！
<br/>　　如果毒天使手里出来的东西还是简单之物，那简直就是辱没了“毒天使”这个称呼，也对不起他出身“黑苗”，拜师“唐门”这样的身份。
<br/>　　“是什么样的蛇？”巫师立刻问道。
<br/>　　被毒天使放出来的毒蛇咬伤，指望普通的抗蛇毒血清是没用的。
<br/>　　“小手指粗细，通体碧蓝带有深蓝色圆环的毒蛇。”李然回答道。
<br/>　　这种毒蛇之前从没有见到过，所以李然记得的很清楚。
<br/>　　听到李然的形容，巫师松了一口气，说了一句“还有救”后，取出两颗药丸，将其中一颗交给周柏让他含在嘴里，把另外一颗捏碎撒在周柏被蛇咬伤的位置。
<br/>　　“他怎么样？”罗昊对巫师问道。
<br/>　　“还好他自己做了应急处理，扩创放掉了一些毒血，不然他现在已经死了。”巫师说道：“咬伤他的是黑苗一族培育出来的毒蛇，这种毒蛇毒性虽然比不上‘小黑龙’，但是也不是普通毒蛇所能比的。而且这种毒蛇的毒牙很细，遭到攻击不易被察觉，而等到察觉的时候，基本已经是无药可救了。”
<br/>　　有了巫师的帮助，在场的所有人都知道，周柏的命算是保住了。在巫师两颗解毒丸的作用下，周柏小腿上黑紫色立刻变淡了下去，就连他的嘴唇也是恢复了正常色。
<br/>　　“看来应该是没事了。”
<br/>　　巫师查看了周柏的情况后，说道：“不过现在他体内的蛇毒还没有全清不能动，否则血液加快流动会让蛇毒流遍他全身，如果心脏受到影响就完蛋了。”
<br/>　　“子午教官，猫头鹰就交给你了。”罗昊对子午说道。
<br/>　　子午不是专业的战斗队员，他的价值更体现在他的技术上。虽然加入龙隐部队后，子午在格斗、射击这些方面也是有针对的进行训练，并且也算得上是一个高手，但是这个高手只是相对于普通人而言。
<br/>　　“没问题。”
<br/>　　子午点头，他自己也清楚，要是论玩电脑，自己一个人玩整个战术中队都行，当然是不把云熙计算在内的情况下；可是要论真枪实弹的战斗，他真的不擅长。尽管不擅长特种作战，可是让他保护一下周柏还是没有问题的，而且在巫师的帮助下，周柏体内蛇毒正在被清除，也不是毫无战斗力。
<br/>　　巫师抽出挂在身后刀鞘中的弯刀，握在手中，对罗昊说道：“毒天使交给我，其他的人你们负责。”
<br/>　　在巫师出现的时候，罗昊就注意到了他挂在身后的这柄弯刀，现在当这柄弯刀出鞘的时候，罗昊从这柄弯刀上感觉到了岁月的沧桑，很显然这柄弯刀已经存在很长很长时间了。
<br/>　　“你回了白苗族？”
<br/>　　李然盯着巫师手中的弯刀，顿时明白了这柄弯刀来自何处了。
<br/>　　“白苗和黑苗很早以前是一体的，只是后来因为各种原因分裂开来了。”巫师看了眼自己握在手里的弯刀，解释道：“这柄弯刀是整个苗族的圣物，自黑苗和白苗分裂后，这柄刀就一直存放在两族交界处大山山洞中，那里供奉着苗族第一代圣女的石像。这次回去，我向族长说明了情况，族长主动带着我去了趟黑苗族，把情况进行了说明。虽然上百年来，两族一直都是水火不容，但黑苗的人也不是完全不讲道理，最后两族族长经过商讨后决定进行合作，清除毒天使这个败类，还特别将这把供奉在圣女石像前的刀交给了我。”
<br/>　　“看来你这次是有备而来。”李然脸上露出一丝微笑，道。
<br/>　　“当然。”
<br/>　　巫师从身上又摸出几颗药丸，分别交给罗昊、李然、子午、周柏、李毅峰、慕容飞六个人，说道：“这是黑苗的避毒丸，他们世代跟各种毒物打交道，这种避毒丸带在身上会让所有的毒物远离你们，如果中毒立刻把避毒丸服下，可以解毒。”
<br/>　　“这么牛！”
<br/>　　慕容飞眼睛发亮。
<br/>　　“虽然苗族世代居住在深山中与世隔绝，但是不要小看他们的传承，有些东西是现代科技都无法媲美的。”李然把避毒丸收起后，说道。
<br/>　　“难怪现在电视广告里，那些个‘苗医’，‘苗族圣药’满天飞，原来是这个原因。”慕容飞若有所思的嘀咕道。
<br/>　　李然闻言，嘴角狠狠的抽了抽，你丫的还能联想到电视广告上去，这思维跳跃也真是够了，玩游戏的就是不一样。
<br/>　　David这边，毒天使脸上的神色也是认真起来了，他知道，这次自己是避无可避了，必须要正面跟巫师一战了。
<br/>　　David本来还是非常兴奋的，看到毒天使扔出几个黑色圆球，然后爆出一团团淡蓝色的烟雾，对面的攻击明显就弱了下去，顿时一扫之前的阴霾，可是这股兴奋劲还没持续多久，飘荡在停车场上的淡蓝色烟雾随着几团白色烟雾的升起就消失了。David还想问清楚到底是怎么回事，就看到身边的毒天使将自己武装了起来。
<br/>　　毒天使摘下挂在腰间的用特种金属打造的手套铠甲，这只金属手套铠甲是“暗月”专门为毒天使打造出来的武器之一，它的造型有点类似中世纪骑士手部铠甲，当然毒天使这个手套铠甲显然不会像骑士手部铠甲这么笨重，打造这只手套铠甲的金属是一种兼备轻便和强度的特殊金属，这种金属即便是正面遭到高碳钢军刀砍击也不会留下任何痕迹，甚至9mm口径的手枪子弹也只能在上面留下一个很浅的凹痕。
<br/>　　当然，这种强度变态而且还轻便的特殊金属是不可能量产的，即便是“暗月”也没有多余的库存，因为在制造毒天使手中这个手套铠甲的时候，因为种种试验就已经消耗殆尽。
<br/>　　加上这种特殊金属令人咂舌的制造成本，让财大气粗的“暗月”都放弃了批量制造的想法。
