        听着八爷的话，童筱颖只觉自己的心情很是复杂。
<br/>
<br/>        过了许久，当八爷还沉浸在回忆的忧伤与愤怒中时，童筱颖缓缓地开口：“如果，一命抵一命，你是不是可以不再与傅夜沉作对？”
<br/>
<br/>        “现在要他命的人不是我，而是冒先生。”八爷嗤之以鼻。
<br/>
<br/>        童筱颖顿时沉默了，放眼望向了车窗外。
<br/>
<br/>        这辆载着她的车子，已经离开了滨江城，目的地是哪里，她完全看不出来，因为车子驶向的是一片荒无人烟的地方。
<br/>
<br/>        八爷见童筱颖很安分，于是将手里的枪收了起来。
<br/>
<br/>        “你倒是一点都不像个人质。”八爷调侃道，“人质不会像你这么冷静。”
<br/>
<br/>        童筱颖看着车窗外，对八爷的调侃，只笑不语。
<br/>
<br/>        事到如今，她还能怎样？
<br/>
<br/>        落以柔被推下车前，对她说：傅夜沉的命比她更重要。
<br/>
<br/>        其实是在暗示她，另外一层意思。
<br/>
<br/>        而她救她，也是在成全她。
<br/>
<br/>        她知道，落以柔很想跟傅夜沉在一起，而她童筱颖是最大的阻碍。
<br/>
<br/>        其实，更多的，她是为了傅夜沉。
<br/>
<br/>        经历过这么多事情之后，童筱颖忽然间想开了。
<br/>
<br/>        她捂着自己的肚子，脸上露出苦楚又无奈的笑意。
<br/>
<br/>        连华生那实验室里的医生，在看了她在医院做的检查结果，又给她重新抽了血做了化验后，神色都变了。
<br/>
<br/>        她虽然不懂他们那脸色是什么意思，但是十有八九是在间接地告诉她，她可能终生不孕不育吧！
<br/>
<br/>        这一切都已经回不去了……
<br/>
<br/>        如果，一开始，她没有遇见过大哥该多好。
<br/>
<br/>        另一边，临海城依山别苑。
<br/>
<br/>        季雅珊端着自己做的点心，打开了楼梯下的一堵暗门，然后只身走了进去。
<br/>
<br/>        她被傅夜沉和十爷冷逸峰里应外合的救出来后，就被傅夜沉安排和十爷冷逸峰一起住在了这里。
<br/>
<br/>        但是，今天，却是一个很特别的日子。
<br/>
<br/>        她不知道他们是从哪里得来的消息，只是听说，冒险者集团的人绑架了童筱颖。
<br/>
<br/>        如果他们在今晚十一点还未抵到“飞鸟断崖”那么他们就杀了童筱颖。
<br/>
<br/>        “飞鸟断崖”是临海城临东海上的一处地势险恶的崖岸，那里只有一条路上去，然后得沿原路返回。
<br/>
<br/>        冒险者集团的人把时间和地点都定在那里，很明显是早已布置好了陷阱，等着傅夜沉和十爷上钩。
<br/>
<br/>        他们若是去救童筱颖的话，无疑是去送死。
<br/>
<br/>        可是，季雅珊知道，傅夜沉和十爷，不会放着童筱颖的生死不顾。
<br/>
<br/>        当她走到地下室设备房的大门口时，就看到傅夜沉和十爷站在工作台前布置营救计划。
<br/>
<br/>        冒险者集团的人不是安防局的人，安防局的人不会随意杀人，但是冒险者集团的人则杀人不眨眼。
<br/>
<br/>        “你们吃点东西吧！有力气了，才能去救筱颖。”季雅珊将手中的点心盘，放在了台面上。
<br/>
<br/>        其实，傅夜沉和十爷冷逸峰现在称得上是最强组合了。
<br/>
<br/>        他们两彼此都有自己的精英手下，而且一个了解安防局，一个又了解冒险者集团，两人算是强强联手，难怪安防局和冒险者集团的两个大佬都会忌惮他俩。
<br/>
<br/>        倘若，他们俩被抓，他们身后的“兄弟”怎么办？
<br/>
<br/>        季雅珊坐在一旁，静默地陪在他们身边，看着他们部署的一切计划，很周密也很详细，但唯一不足的就是无法保证做到百分百的全身而退。
<br/>
<br/>        因为，营救地点的地理环境不允许。
<br/>
<br/>        安防局的人只是想抓到他们俩，但是冒险者集团的人却是想要他俩的命。
<br/>
<br/>        傅夜沉和十爷冷逸峰两人一边商讨着营救计划，一边拿起季雅珊送过来的点心吃了起来。
<br/>
<br/>        两人话题才进行到一半，十爷忽然趴倒在台面上。
<br/>
<br/>        傅夜沉也开始头脑昏沉，他使劲地摇头，想要自己清醒点，但是眼前却出现了重影，一阵天旋地转，最后也晕了过去。
<br/>
<br/>        季雅珊这才从一旁的椅子上起身，将他俩分别挪到了旁边的沙发上躺着。
<br/>
<br/>        希望筱颖能原谅她的自私……
<br/>
<br/>        她不希望十爷去送死，也不希望傅夜沉去送死。
<br/>
<br/>        于她而言，十爷是自己的爱人，傅夜沉是自己的救命恩人。
<br/>
<br/>        虽然，筱颖是她最要好的朋友，但是牺牲一个朋友，能救他们两个人的话，那她宁愿牺牲自己的朋友。
<br/>
<br/>        季雅珊关掉了傅夜沉所有的通讯工具，给他俩一人拿了一条毛毯盖上。
<br/>
<br/>        她下的不是普通的安眠药，而是一种带麻醉性质的迷药，足够他们两睡到第二天早上的时间。
<br/>
<br/>        但是这晚，季雅珊有些难以入睡。
