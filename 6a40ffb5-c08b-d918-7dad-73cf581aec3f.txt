　　在炼制完九九散魄葫芦之后，元始不停，继续炼制下一个葫芦。
<br/>　　第二个葫芦元始将它按上一世的说法炼制成一柄紫白色飞刀，紫白色葫芦蕴含先天庚金本源，擅长杀伐，炼制成的飞刀之后，锋芒毕露，锐气更甚。
<br/>　　简直是有一种“一刀在手，天下无物不可破”的气势，端是厉害。而且，这还不是最厉害的，更厉害的是元始在炼制这柄飞刀的时候，加入了因果律。
<br/>　　他在紫白色飞刀之中制定了“没入喉咙”的果，也就是说，不管发出什么样的因，都会出现这样的结果。换句话说，飞刀只要在出手那一刻，结局就早已注定了，不管它飞出的准不准，它都会没入一个生灵的喉咙之中。
<br/>　　这就非常可怕了！
<br/>　　因果法则，不愧是三千大道之中排名前五的存在，果然深不可测。这还只是因果法则的一个简单运用，就有如此可怕。
<br/>　　不过，因为此物过于厉害，天道之下，万物平衡。为了平衡，此刀却有了极大的限制。这飞刀只能斩杀准圣之下的修士，而且，最为重要的是，你一旦斩杀了他，你就会承受他的最大因果，一旦你没有尽快将那种因果化解，你自己就会因因果反噬而亡。
<br/>　　可以说，这是一件伤人更伤己的必杀之刀，绝杀之刀。
<br/>　　此刀斩仙，斩魔，斩神，斩妖，斩鬼，斩人，更斩自己。若是没有一定的功力，一旦承受不住被斩之人的因果之力的反噬，你自己也会因此而亡。
<br/>　　故而，此飞刀更适用于当自己生命威胁之时，那时，若是命都没了，还顾忌什么因果干什么。所以，不到万不得已，能不用此刀，就尽量不用。
<br/>　　但，就算如此，此刀也极为厉害，简直是对于准圣以下修士们的必杀绝刀啊。
<br/>　　元始对此颇为满意，他第一次将因果之道用于炼器之上，也是收获颇丰了。对于这柄飞刀，他也是极为满意，至少它比起上一世传说中的斩仙飞刀要厉害多了。因为修士对上斩仙飞刀还有解，不至于没有希望，但若是对上这柄飞刀，大罗以下者，十死无生，无解！
<br/>　　元始将之命名为“绝望之刃”！
<br/>　　至于第三个葫芦，元始则是简单祭炼一番，并没有增添什么，因为它本身的功能就十分不凡了，没必要画蛇添足了。弄不好适得其反，那样就不好了。
<br/>　　三个葫芦处理好之后，元始就继续炼化先天至宝混沌钟。
<br/>　　混沌钟原本属于太一的至宝，但太一这些年也并没有完全炼化它，他也仅仅只能够发挥出一部分混沌钟的威力，要不然他也不至于只有那么点战力，被元始玩得那么惨？
<br/>　　先天灵宝是等级的，最高的先天至宝有四十九道禁制，四十道禁制以下三十五道以上的为极品先天灵宝，剩下的为普通先天灵宝。
<br/>　　混沌钟为先天至宝，而且还是最为厉害的先天至宝，足足拥有四十九道先天禁制，可见它的厉害之处。其它三件元始不知，但弑神枪他是知道的，弑神枪虽然也为先天至宝，但它仅有四十五道先天禁制，比不上混沌钟。
<br/>　　而且元始估计，其他的三件先天灵宝，也没有一件能与混沌钟媲美。就算是杀伐第一的诛仙剑阵，估计也只是与它接近，不可能与它相同。因为天道四九，四十九道已经为先天极致了，而这种极致只能存在一种，若是多出来了，那就意味着是变数了。
<br/>　　但，现在它们都还存在，没有被毁，也就意味着它们不是变数。
<br/>　　所以，混沌钟可能是先天至宝中最为厉害的。
<br/>　　一想到这里，元始沉寂多年的心都不免有点变得火热起来。像混沌钟这种先天至宝，已经可以堪比大多数混沌灵宝了，谁也不会嫌自己的宝物多啊，这由不得他不高兴。
<br/>　　不再耽搁，他抓紧时间炼化此钟，此时想太多都是虚的，只有完全炼化，才算是自己的。
<br/>　　但，先天至宝的炼化，并不简单，它至少需要有大罗金仙的境界才能简单炼化，大罗以下者，就算拿到手中，他也没有办法炼化。不到大罗境界，就没有那种属于大罗金仙修士的独特大罗之力。
<br/>　　众所周知，大罗金仙就意味着能够挣脱命运长河了，挣脱命运长河者，他们能够修炼出大罗之力，那是一种更为强大的先天灵力。想要炼化先天至宝的先天禁制，必须要不低于大罗之力的力量，才能炼化。而且修为越高，炼化越快。
<br/>　　所以，太一拿到混沌钟如此长时间，他也没有办法完全炼化混沌钟。
<br/>　　元始炼化之前，还要将属于太一的元神烙印完全去除，之后才能炼化禁制。只有炼化禁制并打上属于自己的烙印，这才算是炼化。
<br/>　　通过去除太一的元神烙印，元始发现太一也仅仅只炼化了三十三道先天禁制，相当于上品先天灵宝，怪不得太一拿着混沌钟如此不堪，原来是他仅仅只能发挥出上品先天灵宝级别的威力。
<br/>　　元始张嘴一吞，顿时将混沌钟吞入腹中。混沌钟顺着经脉来到心神脏道宫之中，赤帝祭出神火，将太一的烙印焚烧毁灭。幸好是之前太一已经将自己的元神烙印抹除，他并没有花费太多力气，便将他残余的元神全部烧毁。如若不然，不仅元始需要花费的时间更长，而且太一也会因此变得元神大损。
<br/>　　烧毁残余的烙印之后，元始则需要静心解析这些先天禁制。每一道先天禁制之中，都包含了不同的道，正是因为有了这些禁制，灵宝才会变得拥有各种神异，各种威能。可以说，这些先天禁制，就是灵宝中所蕴含的道的体现，是灵宝的根本。
<br/>　　元始炼器之时，加入的也是一些禁制，不过那只是后天禁制，而且还是他通过大道演化出来的禁制。所以，那些被炼制出来的灵器，才会拥有种种奇异之处。
<br/>　　而如今解析炼化混沌钟的过程之中，其实同时也是在学习混沌钟中所蕴含的大道，学会了这些先天禁制，才能更好的掌控混沌钟，以后也能在炼器之时加入这些禁制，增加灵器的威能。
<br/>　　元始在拼命地学习这些知识，渐渐地，他沉浸在其中，不可自拔。
<br/>　　时间就在元始学习炼化先天禁制之中流逝，眨眼间，便是三千多年过去了。
<br/>　　此时，元始也从沉浸在学习的美妙之中醒了过来。他不得不醒过来，因为紫霄宫第二次讲道就快要到了。
<br/>　　……
