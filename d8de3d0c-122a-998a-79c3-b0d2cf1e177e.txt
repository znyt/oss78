　　苏莫静静的盘膝坐在地上，整个人仿佛都失去了斗志。
<br/>　　青璇在大祭司的手中，这简直是将他的希望打进了万丈深渊之中，让他无法自拔。
<br/>　　时间缓缓流逝，巫长远已经进入了修炼室中，苏莫依旧怔怔的坐在地上。
<br/>　　青璇怎么样了？
<br/>　　他很担心！
<br/>　　但却是没有任何办法。
<br/>　　到底如何才能救出青璇？
<br/>　　若是等到他拥有武圣的实力之时，都不知道是多少年之后了，那个时候一切都晚了！
<br/>　　大祭司将宏青璇带走，是为了双修吗？
<br/>　　这好像不太可能！
<br/>　　以大祭司武圣境的修为，与青璇双修，不会有任何的效果，除非把青璇培养到拥有武尊境巅峰的修为。
<br/>　　那是大祭司贪图青璇的美色？
<br/>　　这就更不可能了！
<br/>　　以大祭司的身份地位、修为实力，要什么样的女人没有？可以说对方一招手，就有万千女人投怀送抱。
<br/>　　虽然青璇容颜倾城，但还不至于让大祭司都贪图的地步。
<br/>　　那大祭司带走青璇是为了什么？
<br/>　　思索了良久，他发现自己想要了解情况、甚至是救出青璇，只有一个办法，那就是接近大祭司！
<br/>　　但这个办法，危险至极，一个不慎，必死无疑。
<br/>　　危险他倒是不怕，但摆在他面前的是，如何接近大祭司？
<br/>　　他现在的身份地位、修为实力，与大祭司简直是相差十万八千里，根本没有接近的可能。
<br/>　　苏莫仔细思索，并且查找搜魂得来的记忆，寻找能接近大祭司的方法。
<br/>　　很快，他终于找到了一个办法。
<br/>　　天冥星巫族一脉，每过五年便会在全族中选择一名少司，然后由族中四名武圣境的祭司亲自指导修炼，并且传授巫族禁忌巫术，重点培养五十年。
<br/>　　而五十年之后，这些少司就会成为族中核心长老。
<br/>　　这些核心长老，才是巫族最核心的支柱，也是巫族中最有希望达到武圣境界之人，比那些普通长老身份尊贵的多。
<br/>　　而距离五年一次的少司选拔，就在不足一年的时间了。
<br/>　　少司选拔，是天冥星巫族年轻一代的盛会，因为只要能成为少司，就能得到族中武圣境祭司的亲自指导，将来成为核心长老。
<br/>　　这是一步登天的机会！
<br/>　　对于普通的巫族人来说，只要成为了少司，便能改变命运，一飞冲天。
<br/>　　巫族最强大的武学，乃是秘传的禁忌巫术，威力绝伦，只有成为少司，才有可能习得禁忌巫术。
<br/>　　所以，每一次的少司选拔，均是天冥星巫族一脉的盛会。
<br/>　　“少司！”苏莫低声自喃，只要他成为了少司，就能接近大祭司，从而有机会救出宏青璇。
<br/>　　不过，想成为少司，并不是简单的事。
<br/>　　他要在少司选拨中脱颖而出，击败所有竞争者，才能成为少司。
<br/>　　这无关于修为高低，全靠天赋。
<br/>　　他若是使用本体，自然有很大的信心，但现在这具新的肉身，实在太弱了。
<br/>　　此事，还需要仔细斟酌才行。
<br/>　　少倾之后，苏莫发现有了空余的修炼室，便大步的走了进去。
<br/>　　修炼室中空间不大，约莫方圆五丈左右，但用来修炼是足够了。
<br/>　　修炼室的中央，刻画着复杂的阵法纹路，还有投放灵晶的凹槽。
<br/>　　苏莫关上修炼室的大门，在阵法中央盘膝坐了下来。
<br/>　　他现在已经不急着找巫天御了，要为接下来的少司选拔做准备。
<br/>　　不过，外界的人无法查探这修炼室内的情况，在修炼室内却是可以查探到外界的情况，若是巫天御出现，苏莫也能发现对方。
<br/>　　分出一缕心神注意着外界的情况，苏莫没有急着修炼，沉吟了片刻之后，用意念呼唤阎魔鬼圣：“鬼圣前辈！”
<br/>　　“何事？”阎魔鬼圣轻声问道。
<br/>　　“鬼圣前辈，你可有办法在一年之内提升我的实力？”苏莫沉声问道。
<br/>　　“提升你的实力？”
<br/>　　阎魔鬼圣闻言，沉默了少倾，道：“你杀人吞噬不就行了，还用吾帮忙？”
<br/>　　“这……我不是要提升修为，而是要提升战斗力？”苏莫解释道。
<br/>　　想要在少司选拔中脱颖而出，他提升修为是没有用的，因为无论提升到什么地步，比他修为高的人，都有一大堆。
<br/>　　他只有达到同阶无敌，才能够有机会成为少司。
<br/>　　“提升战斗力？提升多少？”阎魔鬼圣问道。
<br/>　　“最起码也要提升十几倍吧！”苏莫道。
<br/>　　阎魔鬼圣闻听此言，差点被没噎死，提升十几倍？武帝境的武者还想短时间内提升十几倍的实力？
<br/>　　这是白日做梦吧！
<br/>　　“吾无法做到！”阎魔鬼圣非常光棍的说道。
<br/>　　“那……提升个七、倍，你总能做到吧？”苏莫又问道。
<br/>　　“勉勉强强吧！”阎魔鬼圣道。
<br/>　　苏莫闻言大喜，急忙问道：“那你快说！”
<br/>　　“一年时间不算长，但也不算太短，吾传汝圣级上品功法《混天魔功》，以汝的悟性，一年之内差不多能提升七八倍战力了！”阎魔鬼圣说道。
<br/>　　“混天魔功？”苏莫闻言眼眸一亮，微微有些激动了起来，圣级上品功法，对方居然传他如此高级的功法！
<br/>　　“嗯！此乃魔族不传之密，在古灵星河能超越此功法的武学已经不多了！”阎魔鬼圣点头说道。
<br/>　　“好，请前辈赐予！”苏莫沉声说道。
<br/>　　随即，阎魔鬼圣意念中传出一股磅礴的信息，直接涌入了苏莫的脑海中。
<br/>　　这股信息真是混天魔功的法诀，深奥玄妙，晦涩难懂。
<br/>　　苏莫很是激动，圣级上品功法啊！比他本体修炼的功法，都强大无数倍。
<br/>　　他心中不仅有些暗叹，如此强大的功法，用这具肉身修炼，未免太浪费了。
<br/>　　因为这具肉身就算再强大，但终究不是他自己的肉身，将来他还是要舍弃的。
<br/>　　但现在眼下这种情况，只能修炼这具肉身了。
<br/>　　不过，这阎魔鬼圣居然将这么强大的功法传于他，难道对方这么好心？苏莫有些不太相信。
<br/>　　但是，眼下他也没有什么办法，只能先修炼再说了。
<br/>　　“苏莫，吾为你详细讲解一番此功法的玄妙！”阎魔鬼圣道。
<br/>　　“多谢前辈！”苏莫沉声说道。
<br/>　　“首先，修炼混天魔功，汝要有心理准备，此乃是魔族禁忌功法，霸道绝伦、至刚至威，修炼此功要承受巨大的痛苦，一旦出现差错，就会走火入魔，魔力反噬，轻则丧失本性，沦为嗜血的杀戮魔头，重则爆体而亡。”阎魔鬼圣先是告诫了一番。
<br/>　　苏莫闻言双眉一跳，对方若是在此功法上动点手脚，他修炼此功法，那不是要完蛋了。
<br/>　　不过，很快苏莫便抛弃了这个顾忌，对方若是要害他，也不会告诉他这些了。
<br/>　　再者，他一旦修炼出现变故，第一时间就能灭了对方，对方应该清楚这一点。
<br/>　　总之，苏莫现在骑虎难下，只能走一步看一步了。
