　　云熙按照罗昊的吩咐，在键盘上敲打了几下，把画面放大。
<br/>　　画面中的五菱宏光有些老旧，银灰色的车身上有好几处掉漆和凹坑，因为监控探头距离车子有一些距离，加上汽车两侧车窗上贴着深色车窗膜，在昏暗的天色下，看不清楚面包车里面的情况。
<br/>　　“画面在放大一些。”
<br/>　　罗昊凑近云熙身边，盯着屏幕中的面包车。
<br/>　　云熙把画面放到了最大，可还是不算清楚，从监控探头的角度，只能看到面包车驾驶座上的人，手里拿着望远镜正在往外看。罗昊自然是不会认为，面包车驾驶座上的人拿着望远镜是在看港口夜景的，比起来看港口夜景这个可能性，罗昊更倾向于对方是李霄力派来踩点的。
<br/>　　“狐狸，港口西北位置，有一辆银灰色的五菱宏光很可疑，把‘探路鸟’放出去，看看汽车里的情况。”罗昊通过耳麦跟已经在港口外树林子里埋伏起来的李毅峰说道。
<br/>　　“收到！”
<br/>　　李毅峰从自己的背包中取出一个旋翼式无人机，跟自己的PDA相连后，抬手把无人机放了出去。
<br/>　　“嗡嗡嗡嗡……”
<br/>　　无人机以低于30分贝的声音，朝着罗昊所说的那辆银灰色五菱宏光飞去。
<br/>　　在港口这种又是海水声，又是货轮装卸货物发出的声响，又是工人的劳动号子声的地方，30分贝的声响几乎就是“静音”。只要李毅峰不是把无人机下降到非常低的高度，根本不会有人发现它的存在。
<br/>　　在光线不足的环境下，李毅峰打开了无人机的夜视功能，无人机的系统不仅跟李毅峰的PDA相连，也跟指挥车相连接，罗昊、云熙他们坐在指挥车中也能看到通过无人机拍摄到的画面。
<br/>　　很快那辆引起罗昊怀疑的银灰色五菱宏光出现在指挥车屏幕中，无人机下降高度，绕到面包车背后，把车牌拍了下来；随后无人机的高度再次拉升，悬停在面包车后车窗的位置，但是因为车窗上贴了窗膜的关系，无人机还是没能看到面包车里的情况。
<br/>　　李毅峰紧接着了开启了无人机的热成像功能，指挥车中屏幕上的画面立刻一变，面包车中的情况立刻就显现了出来，一个个橘红色的身影出现在屏幕中，通过无人机传递来的画面，罗昊可以确定面包车里的人数，总共有五个人。
<br/>　　再根据车厢中四个人手中，在热成像仪下呈现出淡蓝色的物体轮廓判断，车厢中的人是持有武器的，应该是“五连发”或者是土制猎枪之类的东西。
<br/>　　同样坐在指挥车中的周宋捷看着屏幕，暗自感慨，如果刑侦队也能有这样的高端装备，具备夜视仪和热成像仪，就算是嫌疑人藏在屋子里也能被轻易的发现，可以最大程度的减低行动中的伤亡率；但是周宋捷也知道，这也只能是自己想想，这样的东西就是省厅都没有，更何况是市一级的刑侦队。
<br/>　　罗昊让李毅峰把“探路鸟”回收回来后，说道：“这辆面包车应该就是李霄力派来踩点的人，没想到这家伙还很谨慎。”
<br/>　　“他如果不谨慎，早就被我们端掉了，哪里还能让他舒服到现在？”周宋捷说道。
<br/>　　“他的好运到今天晚上就结束了。”罗昊冷冷地说道。
<br/>　　那辆银灰色的五菱宏光始终停在原地没有移动，随着时间的推移，天空已经是如墨一般漆黑，码头上用来搬运集装箱的龙门吊已经不再工作，慕容飞顺着扶梯爬上龙门吊，从那里居高临下，将附近全都监控了起来。
<br/>　　停在港口中的货轮顿时间内暂时不会离港，货轮上的船员也是全都下船，相互结伴去市区或是聚餐，或是消费找乐子，有些家就在杭山市的船员更是迫不及待的打车往家里赶，难得回来一次自然是要回家跟家人团聚，下次靠港回家就不知道是什么时候了。
<br/>　　港口变得冷清起来，涨潮了，海浪也比之前要大了很多，海浪拍打在货轮上，发出“哗啦哗啦”的声响，货轮在海浪的拍打下轻轻晃动着。
<br/>　　时间指向九点三十分，两道汽车远光灯从远处扫过来，一辆银白色的宝马五系，一辆黑色的奥迪A6L，还有两辆面包车从远处驶来。
<br/>　　“狂龙，有车辆进场。”
<br/>　　趴在龙门吊平台上的慕容飞看到四辆汽车驶来，立刻通过无线电跟罗昊取得联系，眼睛紧盯着狙击镜，盯着那四辆汽车。
<br/>　　指挥车中，罗昊、云熙等人也是通过港口附近的监控探头看到了那四辆可疑车辆。周宋捷跟李霄力这根线已经有很长时间，所以在四辆汽车靠近后，周宋捷一眼就认出了那辆银白色宝马是李霄力的车，那辆黑色奥迪A6L是刘三的车。
<br/>　　果然，四辆车停稳后，李霄力和刘三分别从银白色宝马和黑色奥迪A6L上走了下来，那两辆面包车上走下来十几个马仔；银灰色的五菱宏光上负责踩点的马仔这个时候也是走下汽车，李霄力向那几个负责踩点的马仔询问着什么，只是监控探头只能拍摄到画面，没办法听到他们说些什么。
<br/>　　“刘队长，‘关闸’！”
<br/>　　罗昊通过无线电告诉刘东来封锁住港口附近的几个路口，警车拦着路口，警车上的灯光全部被关闭，警车前铺着一道半米宽的拦截带，一根根尖刺在夜色下闪着寒光，荷枪实弹的特警队员握着05微冲趴在汽车引擎盖上盯着港口的方向，狙击手趴在一辆大巴车车顶上，CS/LR4架在他的身前，黑色面罩遮住了他的口鼻，只剩下一双眼睛露在外面，闪烁着冷冽之色。
<br/>　　“周队长，现场无关人员都撤离了吗？”罗昊对周宋捷问道。
<br/>　　“已经完成清场，现在在港口控制室里的，是我们的人。”周宋捷回答道。
<br/>　　“攻击一组，攻击二组准备行动。”
<br/>　　罗昊拿起放在桌子上的56-1，轻轻拉开枪栓，看了眼枪膛中的子弹，推开指挥车的车厢门。
