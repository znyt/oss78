　　“土著，你们很有勇气，不过，勇气没有半点用处！”巫天御的目光，落在了苏莫的身上，朗声笑道。
<br/>　　他的目光中带着一丝好奇，他看苏莫只有武皇境八重巅峰的修为，居然成为了一群武皇境九重，乃至准帝武者的首领，心中倒是有些惊奇。
<br/>　　看来眼前的这位年轻土著，是苍穹世界的顶级天才了，很可能也是最强之人。
<br/>　　“你知道这里是什么地方吗？”
<br/>　　苏莫眸光冷漠，直视巫天御，淡淡的问道，不待对方回答，他又继续道：“这里是死亡之海，预示着这里是你们的死亡之地！”
<br/>　　此刻，苏莫眸光扫视一众巫族天才，心中在暗暗思忖。
<br/>　　这么多巫族试炼者，根本不可能知道谁在巫族的地位高，那就只能尽力擒拿了，擒拿的人越多，到时候底气越足，成功离开苍穹大陆的希望才越大。
<br/>　　“是吗？”巫天御不置可否的笑了笑，摇头道：“你们这些苍穹世界的土著，真的是不知天有多大、地有多广！”
<br/>　　巫天御心中很是无语，一群土鸡瓦狗，居然还想要杀他们？
<br/>　　别说武帝境以下的人了，在外界，就算是一般的中阶武帝，都杀不了他。
<br/>　　“哈哈！这些土著真是井底之蛙！”
<br/>　　“居然还想杀死我们，真是可笑啊！”
<br/>　　“杀死他们吧！抽取他们的武魂！”
<br/>　　一众巫族天才均是嗤笑了起来，个个满脸无语，他们是猎人，对方是猎物，怎么反倒像是颠倒了过来一样？
<br/>　　“天御哥，直接杀了他们吧！不必浪费时间！”巫娴对巫天御说道。
<br/>　　“嗯！”
<br/>　　巫天御点了点头，笑道：“不过，动手之前，先解决一只小老鼠！”
<br/>　　言罢，其手指一点，一道紫色的指芒顿时爆射而出，击向了不远处的海面上。
<br/>　　指芒犀利无比，杀机无限，快到巅峰，几乎肉眼难辨，瞬间便洞穿了无尽的海水。
<br/>　　啊！
<br/>　　一声凄厉的惨叫声，顿时从海底响起，随即，便见一名老者的身影，如同一支利剑般，从死亡之海的海底冲了出来。
<br/>　　老者全身浴血，胸膛有一个拳头大小的血洞，鲜血狂涌。
<br/>　　苏莫等人见此一怔，他们倒是没发现，这死亡之海的下方，居然还隐藏了一名强者。
<br/>　　这位老者苏莫倒是认识，正是曾经臣服于苍穹神宫的一个宗门的门主，武皇境九重修为。
<br/>　　没想到此人，居然隐藏在死亡之海下！
<br/>　　“宫主救我……！”老者乍一出现，便急速向苏莫冲了过去，企图让苏莫护他性命。
<br/>　　然而，老者话音未落，便戛然而止。
<br/>　　只见他的脑袋被随后到来的另一道指芒击中，顿时炸裂开来，惨死当场。
<br/>　　随即，便见老者的武魂急速向巫天御飞了过去，随即进入了其手中的测魂盘中。
<br/>　　这是一种枪武魂，银光闪烁，发出两道金色的光环。
<br/>　　“不错，天级二阶武魂！”收起测魂盘，巫天御面带微笑的点了点头。
<br/>　　“果然不简单！”苏莫见此顿时眼眸一凝，此人出手之快，攻击之犀利，真是恐怖，让他都来不及出手救援。
<br/>　　一名武皇境九重的武者，几乎没有任何的反抗之力，就被轻易击杀了。
<br/>　　“呵呵！土著们，好好享受死亡的滋味吧！”
<br/>　　随即，巫天御轻笑一声，便挥了挥手，示意众人出手击杀苏莫等人。
<br/>　　在他看来，对付面前的这些土著，根本不需他出手。
<br/>　　巫天御话音一落，霎时之间，一众巫族天才，立刻分散而开，个个身上气息涌动，气势冲天。
<br/>　　浩荡的气势，如同万丈狼烟，形成无边的狂风，横扫九天十地。
<br/>　　狂风所过，方圆数千里之内的空间，顷刻间便全部崩碎，化为了一片虚无之地。
<br/>　　苏莫等人见此，也是做好了战斗的准备，虽然他们人数较少，但也没有丝毫惧色。
<br/>　　咻！
<br/>　　就在大战一触即发之际，蓦地，一道刺耳的破空声急速响起。
<br/>　　只见天地的尽头，一道黑色的流光划破长空，急速而来。
<br/>　　唰！
<br/>　　瞬息之间，黑色的流光破空而至，停在了巫族众人数百里之外，显化成了一名瘦弱的老者。
<br/>　　老者不是别人，正是黄泉魔宗宗主，巫森。
<br/>　　“诸位，在下巫森，乃是巫族后人！”
<br/>　　巫森乍一到来，顿时向一众巫族天才抱拳，朗声说道。
<br/>　　一众巫族天才闻言一怔，随即便见到巫森的额头之上，一个巫族的族印缓缓浮现。
<br/>　　这族印乃是血脉之力所化，只有巫族族人才能显化出族印，万万做不了假！
<br/>　　这立刻让一众巫族天才惊喜，这应该就是大祭司的后人吧！他们还没有开始寻找，没想到大祭司的后人就主动找上门来了。
<br/>　　更让众人惊奇的是，这位大祭司的后人，居然有武帝境的修为，也不简单啊！
<br/>　　轰隆隆！
<br/>　　就在此刻，一只三色的玄力大手突然出现，碾碎虚空，破碎苍穹，向着巫森急速抓去。
<br/>　　出手之人，自然就是苏莫。
<br/>　　他没想到巫森会突然出现，但还是毫不犹豫的立刻出手了。
<br/>　　此人虽然换了肉身，但气息的确是巫森，决不会有错。
<br/>　　对于巫森，苏莫早已列进了必杀名单，绝不会有丝毫留情。
<br/>　　三色玄力大手，如同上苍之手，巨大无比，威势浩荡天地，澎湃的玄力波动如同狂涛骇浪，简直强大的无法形容。
<br/>　　一击出，众人皆惊，就算是一众巫族天才，都是心中一震。
<br/>　　他们完全没有料到，这个武皇境八重的土著，居然这么强大！
<br/>　　“什么？”巫森见此大惊失色，好在他早有防备，立刻身形一闪，急速向一众巫族天才之处冲去。
<br/>　　“找死！”巫天御见苏莫出手，顿时眸中杀机爆射，巫森乃是大祭司的后人，自然不能被杀，毫不犹豫，他立刻出手了，一指点向玄力大手。
<br/>　　这一指极为强大，比之先前击杀那名藏身海底之人的指芒，强大十倍都不止。
<br/>　　犀利的紫色指芒，仿佛一柄洞穿虚无的紫色光柱，裹挟毁天灭地之威，速度无语伦比，瞬间跨越千里，击在了玄力大手之上。
<br/>　　轰！
<br/>　　霎时之间，一声爆响整动九天，让得整个天地都猛然一震。
