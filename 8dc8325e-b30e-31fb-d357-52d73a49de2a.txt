<br/>　　他们五人，联手起来，除非是生死境高阶的强者，不然都能轰杀。
<br/>　　如今却在楚风眠的一掌之下，毫无反抗能力，都被震飞了出去。
<br/>　　“你的实力，怎么可能！”
<br/>　　“没什么不可能的，几个杂碎，死吧。”
<br/>　　楚风眠冷哼一声，这虚空五霸，在楚风眠的面前，就跟蝼蚁，没有任何区别。
<br/>　　就在楚风眠打算捏碎这虚空五霸的一刻，白萱突然开口道。
<br/>　　“楚风眠，留他们一个全尸。”
<br/>　　听到白萱的话，楚风眠点了点头，一掌将这五人，全部拍死，倒是并没有将这五人身躯打破，只是以无上玄妙手段，震破了他们的魂魄。
<br/>　　白萱来到这虚空五霸的身边，便是打开一个空戒，直接将这虚空五霸的身躯装了起来，这才向着楚风眠开口道。
<br/>　　“这虚空五霸，在虚空海中有着悬赏，价值不菲，可以去拿他们的身躯换取财富。”
<br/>　　“恩。”
<br/>　　楚风眠手一挥，这虚空五霸身上的空戒，便是落入到楚风眠的手中。
<br/>　　一股灵力注入其中，打开一看，这虚空五霸也是不知道截杀了多少武者。
<br/>　　身上的财富，倒是多的吓人，像是皇品灵石，这样的奇物都有了不少。
<br/>　　在这空戒中，只有一块拇指大小的碎片，引得了楚风眠的注意。
<br/>　　“这是？”
<br/>　　看到这一块拇指大小的碎片，楚风眠目光闪烁，将这碎片拿在手中，在其中，一股熟悉的力量，顿时传入到了楚风眠的身躯之中去。
<br/>　　“果然，是人皇罐的碎片！”
<br/>　　楚风眠心中狂喜，这一块拇指大小的碎片，居然是人皇罐的碎片。
<br/>　　虽然这一块是碎片，只是很小的一部分，但是每一块碎片，融入到人皇罐中，都会产生无与伦比的效果。
<br/>　　“好，这人皇罐的碎片，已经是分散在了北域，甚至是九域的各处，没想到如今居然又被我找到一块，莫非冥冥之中，真的让我有机会将人皇罐碎片全部收集，从新凝聚？”
<br/>　　楚风眠心中也是惊讶无比。
<br/>　　这人皇罐破碎，碎片只怕是会分散在九域的各个角落中，寻常人能够碰到其中一块，都是极难。
<br/>　　楚风眠如今在得到这一块，已经是第三块了，这已经不是运气可以形容的了。
<br/>　　只能够说，这人皇罐，冥冥之中跟楚风眠注定有着缘分。
<br/>　　现在的人皇罐，只能算是一件残缺的极道仙兵，威力也就是比起天级灵器，稍微强大一些。
<br/>　　但是若是人皇罐碎片真的可以全部找到，凝聚的话，一件极道仙兵的力量，足以让整个九域颤抖。
<br/>　　就算是七大宗门，都没有一件真正完整的极道仙兵。
<br/>　　一件完整的极道仙兵出现，足以改变整个九域的格局。
<br/>　　“先将这块碎片，融入到人皇罐中再说！”
<br/>　　楚风眠手一挥，人皇罐便是出现在了他的手中，这一块碎片，感受到人皇罐的力量后，便是直接飞入到了人皇罐中。
<br/>　　瞬息之间，便是融入到了人皇罐其中，令人皇罐的一块空缺，有少了一块，其中的纹路，也变得完整了一些。
<br/>　　“这人皇罐的碎片，一共有着九块，其中最大的两块，我已经找到的，不过剩下的能不能找到，也只能看机缘了。”
<br/>　　这一块碎片融入，令楚风眠对着人皇罐的了解，更深了一步，发现了这人皇罐的破碎，应该是化为了九块碎片。
<br/>　　如今楚风眠一共得到了其中三块，还剩下六块，分散在九域之中的各个角落。
<br/>　　“每得到一块人皇罐碎片，人皇罐的真正力量，便是就足以苏醒一部分，其中的威力都会增强许多。”
<br/>　　楚风眠感觉到，人皇罐的力量，现在又是增加了不少。
<br/>　　只要是得到越来越多的人皇罐碎片，这人皇罐的力量，也就足以彻底的展现了出来。
<br/>　　“人皇罐恢复力量，也足以让我的底牌，更加深厚！”
<br/>　　楚风眠长舒一口气，便是将人皇罐收了起来。
<br/>　　“走吧，我们离开，去虚空海。”
<br/>　　楚风眠对着白萱开口道。
<br/>　　两道身形，再次化为遁光出发，直接便是离开了乱葬海，回到了虚空海中。

<br/>　　一些武者，甚至都常驻在这混元大陆中，毕竟虚空海中有着无数机缘，是修行最好的地方。
<br/>　　白萱带着楚风眠，一路在虚空海中飞行。
<br/>　　不一会，就在前方，一处巨大无比的大陆，却是映入眼帘。
<br/>　　“这里就是混元大陆？在虚空海，这样危险的地方，居然还足以有着一块如此区域？”
