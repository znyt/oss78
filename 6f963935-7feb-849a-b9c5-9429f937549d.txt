        刘婶见连星儿已经起床了，于是只身走过去替她整理床铺。
<br/>
<br/>        当她看到床单上的血迹时，淡定地将床单拆了下来，然后拿来一床干净的床单，重新给连星儿换上。
<br/>
<br/>        连星儿换好衣服后，刚抱着脏衣服从卫生间里出来，只见刘婶朝她走了过来，很自然地接过她手中的脏衣服，塞入了脏衣篓中。
<br/>
<br/>        “阿婆，我还是自己去洗吧！”连星儿忽然间跟刘婶不好意思起来。
<br/>
<br/>        “小时候都没见你不好意思，长大了倒跟我客气了。”刘婶亲切地笑了笑，抱起脏衣篓，接着说道，“我下楼去跟夫人说一声，让夫人给少小姐煮红糖水蛋。”
<br/>
<br/>        “嗯，谢谢阿婆。”连星儿赧然地笑了笑。
<br/>
<br/>        刘婶点了下头，便先下楼去了。
<br/>
<br/>        吃早餐的时候，连星儿没见着傅夜沉，才关问道：“妈咪，爹地昨晚没回来吗？”
<br/>
<br/>        “他去国外谈生意去了。”童筱颖微微一笑，将刚蒸好的红糖水蛋端到了连星儿的面前。
<br/>
<br/>        “那……我高中同学的妈妈，她……”连星儿欲言又止。
<br/>
<br/>        关于朱小溪她母亲的事情，童筱颖从傅夜沉那儿听说了。
<br/>
<br/>        童筱颖只是淡淡地应道：“那位太太已经没事了。”
<br/>
<br/>        “那我需不需要去探望她？”连星儿弱弱地问道，顿了顿后，又补充了一句，“那天在咖啡馆的包厢里，我真的没有推她。”
<br/>
<br/>        童筱颖看着连星儿，接着说道：“以后，你就不要再跟你的那位同学来往了。”
<br/>
<br/>        “噢！”连星儿应了声，拿起勺子，一边吃，一边想起了什么，又问道，“妈咪，那你有给我做佛跳墙吗？”
<br/>
<br/>        “带给你外公吃的？”童筱颖十分淡定地反问。
<br/>
<br/>        连星儿立马摇了摇头：“不是！是、是带给明司寒吃的！我跟他吹牛，说你做的这道菜最好吃了！他不信，非得要尝了后才知道。”
<br/>
<br/>        “你不知道，你撒谎的时候，是会脸红的吗？”童筱颖瞥了连星儿一眼，淡定地拿着抹酱刀，将蓝莓酱抹在吐司片上。
<br/>
<br/>        连星儿赧然一笑：“好吧！果然什么事都瞒不过我家母亲大人！我昨天去看外公了，外公说，他怀念你做的佛跳墙。”
<br/>
<br/>        “待会你给他带过去吧！”童筱颖淡淡地说道。
<br/>
<br/>        连星儿不禁嘴角微扬。
<br/>
<br/>        童筱颖又补充了一句：“不要说是我知道给他做的。”
<br/>
<br/>        “嗯嗯，我知道了啦！反正，你和外公都是口是心非。”连星儿啧啧嘴，打趣道。
<br/>
<br/>        童筱颖没再接话，只是这么多年来，她对于连中凯的事情，心里仍旧跨不过那道坎。
<br/>
<br/>        连星儿用保温盒装着童筱颖做好的佛跳墙，带去了连中凯的实验室里。
<br/>
<br/>        在连中凯吃得正香的时候，连星儿找了个借口要去上洗手间，趁机离开了连中凯的视线。
<br/>
<br/>        这个实验室，她小时候来过几次，只不过活动范围有限，实验室里很多暗门她没去过。
<br/>
<br/>        外公连中凯是搞基因研究的，所以连星儿笃定，这实验室里一定有狼。
<br/>
<br/>        而且还是一只变异的狼，不然体型不会有一个成年人那么高大。
<br/>
<br/>        连星儿好奇地侧耳偷听着每张紧闭的门的门内动静，有几张门内，她听到了鸟叫虫鸣，还有几张门内也有些怪异的声音。
<br/>
<br/>        当她来到走廊尽头的那张银色合金门前，身后却突然传来一个女人的呼唤：“郎安。”
<br/>
<br/>        连星儿惊了一下，本能地转过身去，迎面被一抹黑影所笼罩。
<br/>
<br/>        她惊愕地抬起头来，只见面前突然站在一个留着灰黑色斜庞克发型，拥有如同刀凿般立体的五官，身材高大结实，穿着一袭银灰色休闲装，打着赤脚，看起来与她年纪相仿的男生。
<br/>
<br/>        “郎安，你在这里做什么？”那个女人的声音再次响起，语气十分严厉。
<br/>
<br/>        连星儿好奇地探了探身，声音的女主人从这男生身后走了过来。
<br/>
<br/>        “裴奶奶？”连星儿唤了声。
<br/>
<br/>        裴冉看到连星儿后，面容慈祥，声音也没刚刚那么严肃：“是星儿啊！来看你外公的吗？”
<br/>
<br/>        “嗯。”连星儿点了点头，然后又看向身旁的这个男生。
<br/>
<br/>        裴冉微笑着介绍道：“他叫郎安，是我亲戚家的孩子，生病了，来我这儿治病的。”
<br/>
<br/>        “你好，我叫连星儿。”连星儿看了郎安一眼，微笑着向郎安伸出了手。
<br/>
<br/>        郎安微微垂着眸子，看着连星儿伸过来的这只手，脸上没有任何表情。
<br/>
<br/>        裴冉连忙解围道：“星儿，你外公正找你了，你还是回你外公身边去吧！”
<br/>
<br/>        “好！”连星儿莞尔一笑，然后向郎安挥了挥手，疾步跑开了。
