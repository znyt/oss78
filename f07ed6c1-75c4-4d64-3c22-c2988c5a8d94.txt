　　“这藏地之中，真正巨大的藏地，还未开启，现在不是出手的时候，等到开启的一刻，我们再去对付那小子。”
<br/>　　转轮和尚说完，身形一闪，便是向着一处平原飞去。
<br/>　　小耀王寒夜深深的看了一眼楚风眠离去的遁光，也是身形一动，跟随转轮和尚而去。
<br/>　　“真正的巨大藏地？难道是这佛寺之主，净土佛尊的藏地？”
<br/>　　楚风眠的身形，在这藏地之中飞行这。
<br/>　　刚刚小耀王寒夜跟转轮和尚的交谈，都被楚风眠听得一清二楚，随着实力的提升。
<br/>　　现在楚风眠的灵识所可以探察的距离，也已经是达到方圆二十里的范畴。
<br/>　　在加上万念佛珠的佛门之力，可以抵挡这佛门禁制的压制，他的灵识已经是足以探察方圆五十里。
<br/>　　这二人的交谈，都被楚风眠听在耳中。
<br/>　　甚至是早在刚刚，楚风眠离开那佛寺的一刻，这二人的行踪，都被楚风眠察觉到的。
<br/>　　不过楚风眠也并没有出手。
<br/>　　这转轮和尚，身上的底牌众多，还懂得那轮回宗的秘术，六道轮回，以现在楚风眠的实力，想要生擒他，也并不容易。
<br/>　　如今在这藏地之中，无数宝藏开启，时间紧迫，楚风眠也不愿意拿出大量的时间来对付这转轮和尚。
<br/>　　倒不如趁此机会，多多掠夺一些宝藏，来提升实力。
<br/>　　只要是楚风眠可以踏入九劫古帝境界，这转轮和尚哪怕是什么秘术在多，楚风眠也有把握，足以生擒此人。
<br/>　　楚风眠在这藏地之中，一路飞行，一眼望去，不少的宝光，皆是在这一处藏地之中，冲天而起。
<br/>　　这些宝光，每一道便是代表着，在这藏地之中，有着一尊佛门强者的陵墓开启。
<br/>　　楚风眠粗略看了一眼，便是在看见至少数十道宝光，这还只是藏地之中的一部分。
<br/>　　而且进入到了这藏地之中的武者，妖神，魔族也越来越多。
<br/>　　不时发生着因为争夺宝藏，而进行的战斗。
<br/>　　炎撼世等人，如今也离开，去寻找宝藏了，如此之多的宝藏，几乎是引得所有人疯狂。
<br/>　　各方势力之中的武者妖神，都在混战。
<br/>　　只是楚风眠却是突然发现，没有任何一位净土星域弟子的身影。
<br/>　　之前在外的时候，楚风眠一眼看到了一群来自净土星域的佛门弟子，至少上百位。
<br/>　　在这藏地之中，应该也算是一道极为强横的势力了，可如今楚风眠一个净土星域弟子，都没有看到。
<br/>　　这些净土星域弟子，仿佛是故意的躲藏了起来，在寻找着时机。
<br/>　　“看来这些净土星域弟子，对着净土佛寺之中，极为了解，看来这一次净土宗的人前来，算是有备而来的，必须要防备起来。”
<br/>　　楚风眠心中突然想到那转轮和尚刚刚的话，还有他足以控制这金身佛尊陵墓之中的佛门禁制。
<br/>　　这转轮和尚的实力，还不是最大的威胁，但是他一旦是可以控制这些佛门禁制，这些佛门禁制，才是巨大的威胁。
<br/>　　在这藏地之中，各种陵墓之中的佛门禁制，数不胜数，一旦是这些禁制，可以被转轮和尚所用的话，那可就麻烦无比了。
<br/>　　“必须要小心才行。”
<br/>　　楚风眠自言自语道。
<br/>　　在金身佛祖陵墓之中，楚风眠就被这转轮和尚用金身佛尊陵墓之中的佛门禁制，给阴了一次。
<br/>　　虽然楚风眠化险为夷了，但是有着这一次的例子在，以后楚风眠在探索其他的佛门藏地之中，也必须要更加小心才行。
<br/>　　就在楚风眠一路飞行，在这藏地之中寻找即将开启的陵墓之时。
<br/>　　“那是？”
<br/>　　楚风眠的身形，突然从空中停了下来，看向前方。
<br/>　　前方四十里左右的地方，数道遁光，突然飞了过去。
<br/>　　这数道遁光之中蕴含的力量，极为强横，其中的九劫古帝，至少都有着三尊，剩下的两人，虽然不是九劫古帝的境界，但是一身实力，已经是接近九劫古帝了。
<br/>　　这样的实力，在这藏地之中的无数势力之中，都算是无比强横了。
<br/>　　可引起楚风眠注意的，却是在那一道遁光飞过去的一刻。
<br/>　　在楚风眠的身躯之中，人皇罐的力量，突然是苏醒了过来，一种渴望的感觉，充斥着楚风眠的脑海之中。
<br/>　　“人皇罐的力量，是严家的严君！”
<br/>　　楚风眠猛然灵识探察过去，这五人之中，其中的一名年轻男子，身着华衣，满脸傲气，正是那严家的严君。
<br/>　　如今此人得实力，比起上一次楚风眠见到的时候，要强横了不少。
<br/>　　一身实力，已经是足以媲美九劫古帝了。
<br/>　　看来是此人，在这藏地之中，也大有收获。
<br/>　　而在他的身旁，则是站着另外一名年轻男子，此人身上的气息，不弱于严君，而且看起来地位也不在严君之下，跟严君平起平坐。

<br/>　　他们一路飞行，缓缓降落在了一处平原之上，去抢夺在这平原上开启的陵墓宝藏。

<br/>　　“这算什么，这里可是曾经净土宗的藏地，净土宗曾经可是称霸九域的宗门，其中诞生出来的强者，不可计数，单单是天人境之上的佛尊，都不知道有多少。”
<br/>　　“这如今开启的陵墓，不过只是其中的一部分，我们若是可以得到一些强大佛尊的宝藏，对我们更是有着大益处。”
<br/>　　站在严君身旁的另一名年轻男子，来自慕家的慕九，出声说道。
