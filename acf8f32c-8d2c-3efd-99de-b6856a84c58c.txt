　　闻听天尘宇之言，蟒袍人沉默不语，密室内陷入了寂静之中。
<br/>　　少倾之后，蟒袍人开口了，淡淡的道：“声东击西，混肴视线！”
<br/>　　“哦？你认为，这是苏莫在故意暴露身份，以转移我们的目光？”天尘宇问道。
<br/>　　“不排除这个可能！”蟒袍人道。
<br/>　　“那我们怎么办？你能否衍算出他大概的位置？”天尘宇沉声问道。
<br/>　　“不能！他修为越高，越难以衍算，想必，如今他早已成神，天机都探查不到！”蟒袍人轻轻的摇了摇头。
<br/>　　他心中也有些无奈，好在，如今他修为占据极大的优势，完成目标只是时间问题。
<br/>　　“宁可找错，不可放过，你派人去这几个地方去查吧！”蟒袍人又道。
<br/>　　“好吧！”天尘宇点了点头，如今，只能是将所有可疑之人，可疑之地，都查一遍了。
<br/>　　“切记，不能让地王宫的人，发现你的行动！”蟒袍人又叮嘱道。
<br/>　　这是目前，他最大的忌惮之处，苏莫的存在，目前地王宫还不知晓。
<br/>　　在拓跋青云的眼中，轮回之身只有他们两人。
<br/>　　而苏莫的存在，只有他和天尘宇清楚，连天王宫的高层，都不知晓。
<br/>　　所以，查找苏莫的消息，不能泄露出去，不然，若是让地王宫的人，先一步找到苏莫，那就有些麻烦了。
<br/>　　而他的行踪，都在帝王宫的监视之下，他轻易不能离开天王宫，一切只能交由天尘宇去办了。
<br/>　　“好吧，不过我现在人手不够，混沌神朝那边的人，要不要撤回来？”天尘宇道。
<br/>　　“不，苏莫最想去的地方，肯定是混沌神朝，那里不仅不能懈怠，还要加派人手！”
<br/>　　蟒袍人摇了摇头，随即又道：“另外，人王宫、鬼王宫、魔王宫、兽王宫，都要安排一些人手，严密注意可疑之人！”
<br/>　　“哦，你认为，苏莫会去那四大王宫？”天尘宇闻言，惊讶的问道。
<br/>　　这好像不太可能吧，那苏莫会如此大胆，深入虎穴。
<br/>　　“一切皆有可能！”蟒袍人淡淡的说道，这个可能性虽然不大，但是，却也存在着几分可能。
<br/>　　“好吧！”
<br/>　　天尘宇点了点头，略一沉吟，道：“现在六宫之间，局势复杂，以前的那些人都不能用了，我只能想办法去收买一些那四宫的弟子，让他们留意着可疑之人。”
<br/>　　“嗯！”
<br/>　　蟒袍人点了点头，长叹一声，道：“尘宇，辛苦你了！”
<br/>　　“谈不上辛苦！”
<br/>　　天尘宇摇了摇头，郑重的道：“帝一魂，我并不是帮你，我之是为了神域、为了这个宇宙、为了天地苍生，亿万生灵！我们都需要未来！”
<br/>　　“呵呵，如此便够了！”蟒袍人轻笑一声道。
<br/>　　“告辞了！”天尘宇微微抱拳，随即转身大步离去。
<br/>　　少倾，密室的大门，再次无声的关闭了，只剩下蟒袍人独自盘坐。
<br/>　　“苏莫啊苏莫，你会发现，你所做的一切，都毫无意义！”
<br/>　　蟒袍人低声自喃，随即，他再次闭上了眼眸。
<br/>　　……
<br/>　　轮回神朝，一片苍茫的大地上空，一道黑衣身影，快速疾驰。
<br/>　　此人不是别人，正是苏莫。
<br/>　　他从八卫城离开之后，在周围几个神朝逛了一圈，然后又回到了轮回神朝。
<br/>　　这一圈逛下来，耗费了四年多的时间。
<br/>　　这四年多里，在他几个地方，都故意做出了一些暴露身份之事，然后立刻就离开。
<br/>　　四年多的时间，他大多都是在赶路和修炼之中度过，现如今，他的修为，已经达到了一品中级，距离一品高级也很近了。
<br/>　　主要是，资源匮乏，让他不能最大限度的提升。
<br/>　　当然，这几年在外行走，他也收集了一些源石，但并不太多。
<br/>　　目前，他的身上，还有三十多万下品源石。
<br/>　　他之所以回来，一是完成自己的计划，二是在人王宫能更快的提升。
<br/>　　人王宫的资源，以及各种体系，还是比较好的，远非在外的散修所能比拟。
<br/>　　本来，他是想去一趟终极剑域的，去找绫裳。
<br/>　　但是，天池神朝太大了，路程较远，他就暂时放弃了这个打算。
<br/>　　这一次回来，他不准备再一心蛰伏了，而是要立刻试探出人王宫的态度。
<br/>　　当然，实在保证安全的前提之下。
<br/>　　远远的，苏莫看到了人王城，他便停了下来，落在了一处山林之中。
<br/>　　随即，苏莫找到了一处隐秘的山洞，暂时躲藏了起来。
<br/>　　山洞之中，他意念一动，身前光芒一闪，出现了一道青衣身影。
<br/>　　这青衣身影，赫然也是苏莫的模样，除了衣着不同，其它各方面都一般无二。
<br/>　　“你去人王宫吧！”苏莫挥了挥手道。
<br/>　　青衣身影点了点头，便大步走出了山洞，飞身而去。
<br/>　　这青衣身影，是苏莫的分身，是他前段时间，炼制的一具分身。
<br/>　　在神域之中，危机重重，他不得不炼制一具分身。
<br/>　　因为，就算分身遇到了危机，陨落了，也不会伤及他的本体。
<br/>　　他要用分身，去试探人王宫高层的态度。
<br/>　　这分身极不简单，战力不比苏莫的本体差多少，因为苏莫将万化神剑元神和不败战气元神，留在了分身的体内。
<br/>　　这具分身的肉身，是苏莫在天池无双界域击杀之人。
<br/>　　此人修为一品中级，当初仗着修为比他高，居然想要抢夺他的储物戒，被他吞噬了，然后炼制成了分身。
<br/>　　少倾，苏莫闭上了眼眸，在山洞中静修了起来。
<br/>　　分身离开山林之后，很快，便回到了人王宫中。
<br/>　　人王宫中，弟子极多，人来人往，分身大步行走的宫中的石道上，并没有人在意他。
<br/>　　虽然苏莫当初因为熔岩窟一事，引起了不小的轰动，但是人王宫天才太多了，也没有人会过多在意他。
<br/>　　“姬月松真是厉害，连续击败其它五宫十一名天才！”
<br/>　　“是啊，让我们人王宫威风了一次！”
<br/>　　“这也是挫了其它五宫天才的锐气！”
<br/>　　“不过，姬月松虽然妖孽，比起当初的那位冷无敌，还是有所不如啊！”
<br/>　　“那冷无敌根本不是轮回神朝之人，早已消身匿迹了！”
<br/>　　分身的耳中，听到了不少议论之声，多是在议论姬月松。
<br/>　　不过，苏莫的化名‘冷无敌’，居然也被人提起了。
<br/>　　这不禁让分身有些无语，这都过去了几年，风头居然还没完全过去。
<br/>　　分身没有回惊天府，而是向沁儿的‘月沁园’走去。
<br/>　　他要通过沁儿，见到其父亲，人王宫的十大长老之一，玄天神王。
