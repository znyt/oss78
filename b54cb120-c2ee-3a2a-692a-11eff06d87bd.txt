        傅夜沉见状，也礼貌性地端起餐桌上的高脚杯。
<br/>
<br/>        季思妍主动迎过去跟傅夜沉碰杯，笑容柔和，声音温婉：“夜沉哥哥，傅氏集团在国外就享有名誉，我毕业后做的第一份简历，意向的就职单位就是傅氏集团。而且，我的生日也快到了，希望，夜沉哥哥能实现我的生日愿望。”
<br/>
<br/>        瞧瞧她这话，来之前都已经打听好了他姓甚名谁。
<br/>
<br/>        这左一句“哥哥”，右一句“哥哥”，叫得亲热又矫情。
<br/>
<br/>        听得坐在一旁的郑忠威都忍不住在心里嘲讽：“我家大少爷也只不过比你大一岁而已，哥哥、哥哥什么的，好像我家大少爷很老了一样。”
<br/>
<br/>        郑忠威曾是傅老爷子身边的人，自然不懂现在这些矫揉造作的女人的心思。
<br/>
<br/>        而其实，也就只有季思妍心里知道，傅夜沉就是傅昼景的大哥。
<br/>
<br/>        傅昼景是傅家私生子这件事情，季董事长并不知情，更确切地来说是外界的人都不知情。
<br/>
<br/>        “傅氏集团向来是由人事部统一考核招人，季小姐若是有心来为我们傅氏集团做贡献，明天可以去带着简历去人事部。”不等傅夜沉开口，郑忠威连酒桌上都不忘自己的本分，敬职敬业。
<br/>
<br/>        傅夜沉淡漠地浅笑，默不吭声。
<br/>
<br/>        季思妍只觉自己有点儿尴尬，不得不找台阶给自己下：“那我回去做份简历，有夜沉哥哥在，傅氏集团人事部的人就不会为难我。”
<br/>
<br/>        她这话说得，连郑忠威都不知道要怎么开口了。
<br/>
<br/>        傅夜沉只是回礼性地和季思妍碰了碰杯，并将高脚杯里的红酒一饮而尽。
<br/>
<br/>        季思妍却只是微微仰头，优雅地小啜了一口，一边放下手里的高脚杯，一边轻轻地坐了下去。
<br/>
<br/>        季董事长连连迎手，将餐桌上最名贵的菜转到傅夜沉的跟前，笑容满面地奉承道：“来来来，傅总尝尝这道菜，这道菜可是酒店里大厨的招牌菜！”
<br/>
<br/>        不等傅夜沉拿起刀叉，季思妍便主动献殷勤地拿起公用刀叉，叉了一块法国黑松露放在了傅夜沉的碗里。
<br/>
<br/>        当她给傅夜沉夹菜的时候，身子还故意往前倾了倾，圆润撞上桌边的那一刻，弹性十足地露出了一条美丽的鸿沟。
<br/>
<br/>        就连坐在傅夜沉另一边席位上的郑忠威都能看得一清二楚，可想而知，季思妍这动作是有多大。
<br/>
<br/>        这女人是想明目张胆地勾引他家大少爷啊！
<br/>
<br/>        郑忠威心里揣测性地看了傅夜沉一眼。
<br/>
<br/>        而傅夜沉的目光压根就没在季思妍身上，他只是淡定地看着自己碗里的法国黑松露，剑眉微微蹙动了一下。
<br/>
<br/>        法国黑松露做成菜肴后，那种与生俱来的一股“荷尔蒙”气味，会令人产生一种想一口吞下它的欲望。
<br/>
<br/>        很明显，这次的晚宴，季董事长是别有用心。
<br/>
<br/>        傅夜沉对于季思妍这个女人，只是有种似曾相识的感觉，单纯地只是认为在哪里见过，但这一时间又想不起来了而已。
<br/>
<br/>        “夜沉哥哥是不爱吃吗？那我给你切块鹅肝吧！”季思妍似乎懂察言观色，立即将自己面前那个干净的白瓷盘和傅夜沉面前那个放了黑松露的白瓷盘换了个位置。
<br/>
<br/>        如此体贴入微的女人，又有哪个男人不为之动心呢？
<br/>
<br/>        傅夜沉却除外，他不会暗自嘲讽她，但也不会轻而易举地上她的鱼钩。
<br/>
<br/>        不在心里嘲讽她，是表示对她作为女性的尊重，不轻而易举地上她的鱼钩，是他对自己应有的自制力。
<br/>
<br/>        季思妍可是个尤物，就连郑忠威都忍不住多看了她几眼后，便开始下意识地咽口水。
<br/>
<br/>        “谢谢，我自己弄就行了。季小姐，请慢用。”傅夜沉礼貌性地拒绝，象征性地吃了几口。
<br/>
<br/>        其实这一桌子的高档法国料理，不是很合他的胃口。
<br/>
<br/>        季思妍自然识趣，适合而止，她还是懂，倘若做得太过了，她就是倒贴。
<br/>
<br/>        男人不会真爱一个倒贴型的女人，这是常识！
<br/>
<br/>        中途，季思妍起身去了一趟洗手间。
<br/>
<br/>        良久后都未见人回来，身为父亲的季董事长这才开始担忧起来。
<br/>
<br/>        傅夜沉见状，便命令身旁的特助郑忠威出去寻人。
<br/>
<br/>        五分钟后，郑忠威回来了，但是仍旧没见季思妍跟进来。
<br/>
<br/>        “大少爷，我让服务员去卫生间里找了一圈，没有看到季小姐的人影。”郑忠威如实禀报。
<br/>
<br/>        季董事长顿时就着急得坐立不安，眉心挤成了一个“川”字，双手放下刀叉后，立马掏出手机给季思妍打起了电话。
<br/>
<br/>        但是，电话打通后响了许久都无人接听，这让季董事长主动挂断电话后又接二连三地继续打。
<br/>
<br/>        他一边握着手机不停地拨打过去，一边大手还焦虑不安地大腿上来回摩挲。
<br/>
<br/>        傅夜沉见这位季董事长不像是装出来的，于是从餐桌前站了起来，安抚道：“季董勿慌，我出去找找。”
<br/>
<br/>        “这……”季董事长一脸难为情地看向傅夜沉。
<br/>
<br/>        傅夜沉却偏头看向特助郑忠威，吩咐道：“你留下来陪季董。”
<br/>
<br/>        “是！”郑忠威毕恭毕敬地点了点头。
<br/>
<br/>        傅夜沉离开包厢后，季董事长忐忑不安地看向郑忠威，担忧又不放心地问：“傅总……他真的能把我女儿找回来吗？”
<br/>
<br/>        其实，他是后怕了。
<br/>
<br/>        之前，歹徒绑架过他的小女儿季雅姗，还威胁过他拿钱去赎。但毕竟，季雅姗只是他的私生女，“正宫娘娘”在家看着，他哪敢真正拿出钱来去赎那私生女，所以他最终只能选择报警了。
<br/>
<br/>        但是，思妍不同。
<br/>
<br/>        思妍是妻子的宝贝女儿，思妍要是出了事，他回去要怎么跟妻子交代？
<br/>
<br/>        季董事长暗自为自己捏了把冷汗，最近这些歹徒怎么就盯上他了？！
<br/>
<br/>        “放心吧！有大少爷在，季小姐不会有事的。”郑忠威安慰性地回了一句。
<br/>
<br/>        傅夜沉直接去了酒店的监控室，调出了监控，得知了季思妍的去向后，便给自己的秘密助理阿凯打了通电话：“阿凯，给车牌号是XX的车主开绿色通道。附近的警车，全部为车主保驾护航！”
