间白纸上就跃然而出一串晦涩难懂的几行字。<br/>　　<br/>　　    “好字！”二师爷没看内容，倒是先夸了王惊蛰那一手亢进有力的字体。<br/>　　<br/>　　    王惊蛰把纸掉了过去，推到两人面前，手指敲着桌子说道：“归藏里有有一段是关于相山水的前言，你们先看看，这算是我先答应你们给出的引路砖，待到五年之后你们答应我的两件事都办齐了，剩下的内容我再写给你们，你们也别怕我反悔，出来混的就讲究·······呃，不是”<br/>　　<br/>　　    二师爷一摆手，说道：“你不用说了，这世上的承诺没有比你们赊刀人落下的话更靠谱，我们信了”<br/>　　<br/>　　    大师爷点头说道：“一字千金不为过”<br/>　　<br/>　　    四人脑袋顿时齐刷刷的凑到那页之前，眼珠子贪婪的盯着王惊蛰写的几行字：“开山，山川起伏，落水，风生水起，风水一道源于山川河流之脉络，从鲧禹治水起，一则开山二为导水·······”<br/>　　<br/>　　    洋洋洒洒不过百来字而已，玄门相字脉几人却看的浑身直起劲，差一点就老泪纵横了。<br/>　　<br/>　　    “前言，这还只是前言？”大师爷不可置信的说道：“难以想象啊，现在流传下来的风水之术，连前人十之一二都没有达到啊”<br/>　　<br/>　　    王惊蛰撇嘴说道：“要是世间全是这东西，那岂不是乱套了？”<br/>　　<br/>　　    “也是，也是！”大师爷连连点头。<br/>　　<br/>　　    王惊蛰靠在椅子上，两脚在桌子下翘起了二郎腿，晃悠了起来。<br/>　　<br/>　　    “啪”脚尖不知道踢在了谁的腿上。<br/>　　<br/>　　    易苏苏一低头就看见了王惊蛰的一只布鞋在晃荡着，心里忽然有点狐疑了：“他好像在得意？不是应该上火发愁的吗？”
