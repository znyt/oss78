<br/>				陈默一走进父母这里，就见到了七大姑八大姨之类的。众人见到陈默过来，也是笑着招呼。<br/>　　<br/>　　这个喊二娃，那个叫侄儿，还有些叫叔，或者叫表舅的。反正就是能扯上关系就的就叫，扯不上关系的就拉关系。<br/>　　<br/>　　他倒是笑脸相迎，因为这里还有父母在，所以就笑着点点头，但是却很少开口说话说话。<br/>　　<br/>　　“二娃回来啊！正说你呢！”<br/>　　<br/>　　“二娃啊，你现在可是牛了啊！”<br/>　　<br/>　　“二娃，你怎么不到伯哪里坐坐呢，天天念叨你，只能亲自过来看看了！”<br/>　　<br/>　　“哎呀！二娃，你回来了，我正和你爸说你的事情呢！你看你……”一个陈默能叫声姨的人，对着陈默就亲热的说道。<br/>　　<br/>　　但是陈默却直接打断，对大家说道：“各位，我回来有事情和我爸我妈说，还请大家先各回各家吧！如果有事，等闲了再聊!”<br/>　　<br/>　　他才不相信，这个些人能够有着急的事情。要是有着急的事情，那么也就不会是在这个时候说了。<br/>　　<br/>　　听到陈默说的话，也明白他的意思是什么，有些无关的人也就缓缓走了出去，不过在临走的时候，还不不住的往口袋里装些瓜子花生什么的。<br/>　　<br/>　　“那个，二娃……”<br/>　　<br/>　　“你们先回去吧，有事明天说，我现在有急事和爸妈商量！”陈默直接打断。<br/>　　<br/>　　“这个，那好吧，你先商量事情，等闲了我再过来!”<br/>　　<br/>　　陈默还在人群中见到自家的婶子，真的是稀客，不过他却没有上去喊叫，就那么看着手，好像手上有什么好看的东西一样。<br/>　　<br/>　　父亲陈建国就蹲在墙根的位置，抽着烟，也不说话。本来乱糟糟的场面，自从陈默进来后，三言两语的将人赶走，他也没有站起来。<br/>　　<br/>　　好多人听到陈默的说词后，本来不想走的，但是看到陈建国也不说话，只能挪开脚步朝外走，虽然嘴上不说什么，毕竟有求于人，但是却心里却有些暗骂着陈默和陈建国。<br/>　　<br/>　　等人都走的差不多了，小婶子就走了过来，笑着对陈默说道：“二娃，你闲了也不到我家里坐坐，你叔整天都念叨你呢！”<br/>　　<br/>　　“没有啥好坐的！再说了，这么多年了都没有去过你家了，都忘记你家的家门朝那里开了！”陈默一撇嘴，说道。<br/>　　<br/>　　小婶听到这个话，也是噎的不行。但是想到自家的事情，也就干笑了两声，说道：“二娃，瞧你说的。都是亲~亲的叔侄，那能这么生疏。<br/>　　<br/>　　你看你现在有了些前景，这不也要靠亲戚伙里帮衬么？要知道，靠外人终究是不放心的，还是亲戚用着放心，至少和你是一条心。你看你那里都没有个帮衬的人，你哥在家也没有事干，就让他去帮帮你，你看是不是个理？”<br/>　　<br/>　　“是不是将我哥的媳妇也安排一下，顺便将我堂弟他也安排到我哪里？”陈默有些无语的问道。<br/>　　<br/>　　“二娃，哈哈!瞧你说的，要是你有这个心，婶子我也高兴啊！”小婶当然听的出这是说的什么话，所以也只能勉强笑着说道。<br/>　　<br/>　　“呵呵!还是不用了。我哪里暂时不缺人手，你还是给哥找其他活吧！”陈默有些平静的说道。对于自己的两个叔叔，他们姐弟三人都是心里凉透的，没有任何的亲情可言了。但是他也至于去辱骂小婶，毕竟都是亲戚，你辱骂她也就是骂自己。<br/>　　<br/>　　“二娃，你这话说的！你哥又不是别人，这打断骨头连着筋的亲戚，他可是能帮助你的人啊！”小婶说道。<br/>　　<br/>　　“我哪里真不缺人手，再说了以前小婶你不是说过么，这人啊各过各的就好！有本事就自己去闯荡一下，没有本事就在家里伺候地就是了。我想你可以给堂~哥这样说说么！”<br/>　　<br/>　　小婶的嘴角明显的抽抽了一番，盯着陈默说道：“二娃，咱们可是亲戚，难道这么点小忙都不能帮帮么？”<br/>　　<br/>　　“亲戚？婶子，还是不要这么说了，你我心里有明白，有些话还是留在心里的好，要是说出来，那么大家面子上都说不过去了！”<br/>　　<br/>　　“你！”小婶顿时气结，感觉心里堵的厉害。<br/>　　<br/>　　“他大伯，你也不说说你家二娃，这么和我说话，难道上了个大学白上了？”小婶直接转移火力，质问陈建国。<br/>　　<br/>　　陈建国将手中的烟吧一扔，然后站起来用脚碾灭，然后转头看着小婶，说道：“我家二娃幸亏上过大学了，不然他说的话可能更加过分。再说了，我家的孩子也不需要你来说什么！”<br/>　　<br/>　　陈默听到这个话，都有些不可置信的看着自己的父亲，没有想到老爸能说这么样的话。一个平时老实，尽量不得罪人，与人为善的老爸，这么怼自己的弟媳妇，他只有心中双手点赞，厉害了我的父亲！<br/>　　<br/>　　“你们！你们……有钱了不起啊！”<br/>　　<br/>　　“不错，有钱就是了不起！”陈默接过话说道。<br/>　　<br/>　　“我不和你们说了，有钱你们都带着去棺材里吧！呸！”小婶一口吐沫，直接吐到地上，然后扭头走人。<br/>　　<br/>　　朝别人家里吐口水，这是非常恶劣，可以说是断交的表现。陈默一家三口，就这么看着小婶走了出去，却没有任何的说话。<br/>　　<br/>　　陈默也没有上前，这一口吐沫，也算是两家关系真正地断绝了，以后小婶子也不会再来家里了，不然那就是打脸！<br/>　　<br/>　　“爸，妈！你们不要往心里去。”陈默拿出拖把，将地上的痰拖干净，并且劝慰道。<br/>　　<br/>　　“没事！”父亲坐到板凳上，有掏出一根香烟点上抽了起来。母亲在边上，一直没有说话，但是这一阵却落泪了。<br/>　　<br/>　　陈默赶紧找来毛巾，递给老妈。<br/>　　<br/>　　对于父亲陈建国这么一根接一根的抽烟，他倒是没有什么劝阻。自从父亲喝着陈默配置的药酒，还有吃着乾坤珠内的蔬菜，还有蜂蜜等等，另外就是陈默配置的增元丹，父亲身体非常的好，就连个小病什么都没有了。<br/>　　<br/>　　至于说抽烟这点污染肺部的小毛病，也早就不在话下，抽多少下去，都能通过身体排除体外。反正就是多洗洗澡的事情，只要父亲抽着开心就成。<br/>　　<br/>　　不过现在，陈建国抽烟可不是开心，而且心中积郁。两个亲兄弟啊，真的是让他非常的伤心。这都是做了孽，让亲兄弟走到这一步！<br/>　　<br/>　　“爸！”陈默看着父亲，也只能在旁边喊了一声，他知道父亲的心情此时非常的差，但是却不知道怎么开口。<br/>　　<br/>　　“没事！我出去走走！”陈建国甩甩手，站起身后，就低头走了出去。剩下陈默和母亲两个人。<br/>　　<br/>　　“妈！老爸他……”陈默有些担心的问道。<br/>　　<br/>　　“没事！让你爸静静，这么多年了，该生的气早就生完了。剩下的，也就是那么点点念想了，他不会有事情的，等会就能回来！”母亲付慧丽有些叹气的说道。<br/>　　<br/>　　陈默也只能点点头，母亲是最了解父亲的！今天要不是自己过来，还不知道会怎么滴呢！<br/>　　<br/>　　人啊！陈默除了感叹一声以外，其他的也不好说了。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
