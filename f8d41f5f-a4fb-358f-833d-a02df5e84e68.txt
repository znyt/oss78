        虽然，这只是个意外，大哥不知道被谁下了那种药，并且大哥还误以为她是苏漫雪，但是……
<br/>
<br/>        童筱颖心里疙瘩着，这件事，就像一根刺，从此扎进了她的心头，再也无法自拔。
<br/>
<br/>        另一边，苏漫雪思前顾后，决定再去一趟傅夜沉的房间，探看傅夜沉的状况。
<br/>
<br/>        “夜惑”里的那个男公关说了，他的那种迷香，除了跟女人上床以外，压根就无药可解，就算连华生在，他也无济于事，更何况，连华生今天还不在私宅里。
<br/>
<br/>        傅夜沉如果不要她的话，他打算怎么替自己解决？
<br/>
<br/>        难道他一个人在书房里“打飞机”？！
<br/>
<br/>        苏漫雪好奇不已，当她再次来到傅夜沉的房间里时，身后突然袭来一阵冷风，让她不寒而栗。
<br/>
<br/>        她还未弄清楚是什么状况，就被一道力度掐住了脖子，狠狠地往墙壁上撞去。
<br/>
<br/>        “苏、漫、雪！你一次又一次地挑战我的极限，是活腻了？”傅夜沉那冷酷的声音，在苏漫雪的耳畔响起。
<br/>
<br/>        苏漫雪定睛循声抬眸，对上了傅夜沉那双冷冽的黑眸。
<br/>
<br/>        此刻的他，双眸清澈，眉宇间却透着一股戾气。
<br/>
<br/>        他的头发上有水珠顺着他的额角滚落，干净的白衬衫，不松不紧地贴着他健硕的胸肌，身上散发着沐浴乳的清香。
<br/>
<br/>        看样子，这男人体内的药效已经消散了。
<br/>
<br/>        那种事情做完后，男人都爱去洗个澡……
<br/>
<br/>        苏漫雪红唇微扬，学着童筱颖的声音，柔柔地说道：“大哥，你这是做什么呀？想掐死我吗？”
<br/>
<br/>        反正她已经没有退路了，不如破罐子破摔。
<br/>
<br/>        傅夜沉剑眉微蹙，手下的力度渐渐地变紧。
<br/>
<br/>        苏漫雪慢慢地感觉到呼吸有些困难，心悸地瞪大了眼睛，直勾勾地看着傅夜沉。
<br/>
<br/>        他……莫非真的想杀了她？
<br/>
<br/>        不，不会的！
<br/>
<br/>        他当过兵，是军人，军人的职责是什么？
<br/>
<br/>        保家卫国，护国护民！
<br/>
<br/>        傅夜沉一定不会做杀人犯法的事情！
<br/>
<br/>        苏漫雪在心底笃定，但是傅夜沉手下的力度一直在加重，她几乎快要无法呼吸了。
<br/>
<br/>        “啪嗒、啪嗒、啪嗒……”
<br/>
<br/>        房门外忽然传来不慢不快的脚步声。
<br/>
<br/>        傅夜沉这才想起筱颖的行李箱还在自己房间，于是他放开了苏漫雪，直接将她扛到肩上，带进了书房。
<br/>
<br/>        到了书房里，他将苏漫雪往地上一扔，还未来得及去关上书房的门。
<br/>
<br/>        苏漫雪立即从地上爬了起来，见傅夜沉突然如此反常，心想着会不会是童筱颖过来了，索性不管三七二十一，迅速把自己脱了个精光，紧接着往傅夜沉身上一跳。
<br/>
<br/>        童筱颖洗完澡后，蹑手蹑脚地再次回到傅夜沉的房间，只是为了拿回自己的行李箱，好离开这里。
<br/>
<br/>        她记得，她把自己的行李箱落在大哥傅夜沉的书房门口了。
<br/>
<br/>        大哥当时好像是做晕了过去，应该不会这么快就醒过来吧！
<br/>
<br/>        童筱颖刚走到书房门口，就听到书房里传来苏漫雪的娇喘声。
<br/>
<br/>        书房的大门仍旧是敞开的，她并未定睛朝书房里看去，但余光里，一具一丝不挂白花花的胴体，在一个高大的身影上起起伏伏。
<br/>
<br/>        童筱颖快速提起行李箱，悄悄地疾步离去。

<br/>        傅夜沉见童筱颖拿走了行李箱后，伸手便就苏漫雪从自己的身上推了下去。
<br/>
<br/>        他衣着整洁，她却一丝不挂。
<br/>
<br/>        刚刚她趁机在他身上那么诱惑地乱蹭，他仍旧心如止水，对她没有一丝反应。
<br/>
<br/>        苏漫雪拾起地上的连衣裙，慢条斯理地穿上。
<br/>
<br/>        刚刚她因为背对着门口，所以她并不知道童筱颖进来是做了什么。

<br/>        又或者，她童筱颖是想来说什么，结果看到她苏漫雪和他傅夜沉的这种香艳画面，便不得不打道回府。
