        &lt;h3&gt;第三卷第二百五十一章  爱是相互依恋，不是相互折磨&lt;/h3&gt;
<br/>
<br/>        江俞轩回到家里，张倩楠从新年到现在一直都在西城，但是却没有和往常一样回到他们的家里，而是住在了单位。
<br/>
<br/>        张倩楠在的那一段时间，这个家里是有烟火气息的，充满了温馨，现在家里也有保姆，总感觉缺点啥。
<br/>
<br/>        “倩楠，你在忙吗？”江俞轩拿起电话，给张倩楠打电话，电话响了很久才被张倩楠接起来。
<br/>
<br/>        “嗯，最近一直在忙，在查一些事情。”张倩楠在电话那头回答，好像在走路。
<br/>
<br/>        “你不回来吗？”江俞轩迟疑了一下问。
<br/>
<br/>        “不过去了，事情太多，住单位方便。”张倩楠干脆而果断的回答。
<br/>
<br/>        “那我明天去看你吧，函哥也在吧？你们最近好不好？”
<br/>
<br/>        “我们最近很忙，函哥的身体好像不太好，有些消瘦。”江俞轩听到了电话里面的开门声。
<br/>
<br/>        “他身体本来就不怎么好，让他注意身体，有了好身体才能守护他要守护的人。”江俞轩说到这里感觉无比心酸，张函其实爱得很卑微。
<br/>
<br/>        这么些年张函都是在后面默默的付出，对陈子昂的爱他是看得真真切切。只是陈子昂的那一句：“你们根本不知道我想要什么。”将大家所有的付出都否定了。
<br/>
<br/>        陈子昂，你想要什么？
<br/>
<br/>        “我知道了，我会照顾好他的，你也保重，我们明天见。”张倩楠挂掉了电话。
<br/>
<br/>        江俞轩觉得他和张倩楠从张家举行简单的仪式起，他们的心理就发生了奇妙的变化，按说他们现在是名义上的夫妻了，应该是比之前更加亲密，相反，他们不但不亲密，反而还有意的疏远对方。
<br/>
<br/>        是因为家庭的原因，还是他们本身的原因？
<br/>
<br/>        没等江俞轩深究，电话打断了他的沉思。
<br/>
<br/>        郝景文欢快的语音传过来了：“三哥，你还在西城？”
<br/>
<br/>        “文文，我一直都在西城呀。”
<br/>
<br/>        “嗯，我要去西城，西城那边租赁运营需要启动了。我得过去看看。”
<br/>
<br/>        “来吧。我给你接风。”
<br/>
<br/>        “接风就不必了，大哥过几天才能回来，你得陪我主持西城的工作。”西城也算是一个大城市，这几年的旅游业呈上升趋势，他需要尽早启动计划。
<br/>
<br/>        “我可太难了。”江俞轩吐槽了一句，他现在分身乏术，真羡慕孙大圣的七十二变。
<br/>
<br/>        “难什么难，我又没有要求你具体负责什么，你得露露面啊。啊，对了，陈子昂最近在放大招，研究了一套很厉害的软件，估计是为了她家里的那俩个萌宝准备的。”
<br/>
<br/>        “什么软件？”
<br/>
<br/>        “好像是什么安保防护系统，可太先进了，而且俩个孩子也参与了，你说陈子昂那个怪物是什么物质组成的？连生的孩子都那么厉害，我严重怀疑他们是从火星来的。”
<br/>
<br/>        “怎么个先进法？”江俞轩是知道陈子昂的能力的，但陈子昂没有说要研究什么安保防护系统，这短短的时间就开始进行研发了？
<br/>
<br/>        “不能给你说，陈子昂要我保密。咦，陈子昂竟然没有给你说？你们俩人出什么问题了？”郝景文像是发现新大陆一样。
<br/>
<br/>        “看你那嘚瑟的样子，就你知道陈子昂的秘密？我早就知道了。”打肿脸充胖子他江俞轩不是不会，只是不屑，况且陈子昂什么事情他不知道？只不过最近太忙，没有联系而已。
<br/>
<br/>        “算了，不给你说了，对了，我已经在和思宇商量要两个孩子，嘟嘟和团团太可爱的。”
<br/>
<br/>        “文文，你这八字还没有一撇呢，就准备要孩子了？这比高速还高速啊。”
<br/>
<br/>        “谁说我们八字没有一撇？我们都见过双方家长了，甜甜的日子指日可待。”郝景文那嘚瑟的语气从电话那头溢出来，冲击着江俞轩的耳膜。
<br/>
<br/>        “你这进展的太顺利了，小心翻车，到时候可别让我看大型的翻车现场。”
<br/>
<br/>        “你还是不是哥哥了，我和思宇这一生都不会翻车，我们西城见！”郝景文现在是无时无刻都想去撒狗粮，喜悦是需要分享的。
<br/>
<br/>        ........
<br/>
<br/>        江俞轩挂了电话，打开电脑，电脑邮箱里静静的躺着的一些有关夏颜的资料，从夏颜顶替陈子昂开始，一直到现在她做的一些事情，江俞轩挑选了一些感觉有价值的东西进行打印和装订，明天他要将这些东西交给张倩楠和张函。
<br/>
<br/>        .........
<br/>
<br/>        张倩楠挂了江俞轩的电话，推开门，张函还在灯下翻阅着资料，手中不停的做着记录。
<br/>
<br/>        “这些事情你没有必要亲力亲为，让人做不就行了。”张倩楠看着张函消瘦的脸庞说，这一段时间张函看着日渐消瘦，她的心都在疼。
<br/>
<br/>        “我们调查的这些事情都是秘密的，我要知道他们背后的关系网到底有多大，牵扯到那些方面，如果让别人做就会引起怀疑，到时候什么都干不成。”站函头也不抬的说。
<br/>
<br/>        “我来吧，你休息会。江俞轩给我打电话了，他让我告诉你，只有有了好身体才能守护你要守护的人。”张倩楠心里有一丝莫名的悸动。
<br/>
<br/>        “倩楠，你真的爱江俞轩吗？”张函抬起头看着张倩楠，但是他没有等张倩楠回答接着说：“我是真的爱陈子昂，所以我愿意付出我的一切来守护他们的喜乐安康，哪怕为他们付出我的生命我也在所不惜。我现在所做的一切是为了他们母子，但是也是为了还这个世界一片清白，让清者更清，浊者无处躲藏。”
<br/>
<br/>        张倩楠静静的看着张函，她追随着他的脚步，他爱的人她也会去守护，他想做的事情她也会去做，从小他就是她崇拜的对象。
<br/>
<br/>        “你没有爱过江俞轩，在你温柔后面都是假象。爱是相互依恋，相互想念，而不是相互折磨，更不是利用！我以为你是爱着他的，可是你没有！如果说当初没有你的帮助，我可能不会那么清楚的知道陈子昂的一切，如果说当初你不是表现的那么爱江俞轩的，我也不可能利用江俞轩来探听陈子昂的一切，我会有别的办法。”
<br/>
<br/>        “我，我现在爱上他了。”张倩楠被戳破心思，低声的辩解。
<br/>
<br/>        “爱他为什么要逃避他？”
<br/>
<br/>        “你们心里都装着陈子昂，何时会装过我？我和你们在一起的时候，你们都是说的陈子昂，陈子昂！”张倩楠几乎是喊出来的。
<br/>
<br/>        “陈子昂是我的爱人，你是我的妹妹，我心里都装着你们！而，江俞轩和陈子昂在一起同甘共苦六年，这等感情不是一般人能插入进去的，无论他们是相爱的还是朋友关系，我从来都没有嫉妒过江俞轩和陈子昂的感情，相反，我很庆幸陈子昂身边有这样的人陪着她，替我守护她爱着她。倩楠，爱，不是占有，不是利用！你要明白你的心，如果你真的爱江俞轩，你就要接纳他心里藏着别人，用宽容和体谅去和他过一生，如果你不爱他，就不要相互折磨，犹豫不决了。”
<br/>
<br/>        张倩楠看了很久张函：“我明白了！”
