　　在曾经古老的时代，净土宗还存在的时候，不知道多少强者，都是被真言所惑，成为了佛门奴隶。
<br/>　　楚风眠都差一点中了这一招。
<br/>　　不过楚风眠这也是因为第一次听到这佛门真言，没有防备，如果说有所防备的话，这佛门真言，也难以动摇楚风眠的道心。
<br/>　　“想要用佛门真言来蛊惑我？破我的道心？可笑，天下间还没有人，可以破了我的道心！”
<br/>　　楚风眠冷哼一声，在他所有防备之后，当即心神之中那一丝慌乱，一瞬消散，楚风眠身上被束缚的灵力，也全然打开。
<br/>　　于此同时，楚风眠越是猛然张嘴，一声大吼，从楚风眠口中吼出的，不是人言，却是阵阵龙吟。
<br/>　　这龙吟，尊贵无比，哪怕是真正的远古战龙的龙吟咆哮，都无法与之媲美。
<br/>　　天龙八音！
<br/>　　随着楚风眠血脉的精粹，这天龙八音的威力，已经是不亚于真正战龙的龙吟。
<br/>　　龙吟，足以破除一切鬼神妖邪。
<br/>　　“此人的道心，怎么会如此坚固？我的真言，居然是都难以影响到他？”
<br/>　　转轮和尚看到楚风眠如此之快的清醒过来，都是吃了一惊。
<br/>　　以他的佛门真言，九劫古帝，如果没有防备，都可能直接中招，而且哪怕是有所防备，也会被影响道心。
<br/>　　只要是道心一破，他便是足以趁乱出手，一击必杀，可没想到，楚风眠只是受到了一瞬之间的影响，就当即是苏醒了过来。
<br/>　　楚风眠道心的坚固，超出了这转轮和尚的意料之中，他已经是身躯逼近了楚风眠，本是想要趁机来偷袭楚风眠，对楚风眠一击毙命的。
<br/>　　现在楚风眠却是突然反应过来。
<br/>　　一声龙吟长啸之下，只见楚风眠身边，佛门真言所化的古老梵文，当即破碎，瞬间都被凌空打散。
<br/>　　“不好！”
<br/>　　转轮和尚看到这无数古老梵文被破的一刻，都是脸色大变，急忙退后，却是有些来不及。
<br/>　　天龙八音的力量，在空中凝聚，向着转轮和尚轰击而去，几乎不亚于一尊龙爪，重重的轰击在了这转轮和尚的胸口之上。
<br/>　　琉璃色的光芒，在这天龙八音轰击的瞬间，迸发出来，足以是将天龙八音九成的力量，都给化解。
<br/>　　可纵然是一成的力量，轰击在这转轮和尚的身上，对他也是极大的打击。
<br/>　　碰！
<br/>　　只见转轮和尚的身躯，被当即打飞了出去，他的脸色都是变的苍白无比，身上的灵力无比混乱。
<br/>　　虽然是没有重伤，可承受了这一击，那转轮和尚都是消耗了大量的灵力。
<br/>　　“大自在，涅槃重生！”
<br/>　　转轮和尚怒吼一声，在他的背后，浮现出一片虚无，只见大量的佛门之力，凝聚在了他的身躯之上。
<br/>　　他那身上的伤势，在这源源不断的佛门之力下，居然都在逐渐的恢复。
<br/>　　“这转轮和尚，居然可以汲取这佛寺之中的力量，在恢复伤势？”
<br/>　　楚风眠看到这一幕，脸色有些阴沉。
<br/>　　这佛寺之中，密布的佛门之力，几乎是难以想象的海量，要是现在这转轮和尚，可以一直汲取这佛寺之中的佛门之力，来恢复伤势的话。
<br/>　　那这转轮和尚，在这佛寺之中，就已经是堪称不死之身了。
<br/>　　除非是楚风眠足以将这佛寺之中的佛门之力，全部毁灭，但那几乎是不可能的事情。
<br/>　　这转轮和尚，本就是无比难缠，实力强悍，要是在外，也许楚风眠还有着几分机会可以杀了他。
<br/>　　但是在这佛寺之中，楚风眠想要杀了这转轮和尚的机会，几乎为零。
<br/>　　“必须要想个方法。”
<br/>　　楚风眠在逼退了转轮和尚之后，也没有乘胜追击，而是目光在扫视这四周，考虑这对策。
<br/>　　“凰公子，小耀王，我们一起联手，杀了此人如何？”
<br/>　　另一边转轮和尚恢复了伤势，又看向了楚风眠，不过这一次，他没有再度出手，而是对着旁边的凰公子跟小耀王寒夜开口道。
<br/>　　他在单打独斗之下，他已经是跟楚风眠交手，吃了不小的亏，自然不愿意一个人在跟楚风眠对战。
<br/>　　“好。”
<br/>　　“一起出手！”
<br/>　　小耀王寒夜跟凰公子二人，对视了一眼，同时点了点头，凝聚灵力当即向着楚风眠轰击而去。
<br/>　　他们二人，本是打算想要坐山观虎斗，坐收渔翁之利的，这一次请转轮和尚出手，他们可是开出了这金身佛尊陵墓之中五成的宝藏的报酬。
<br/>　　所以从一开始，他们二人就没有打算出手，只是坐了坐样子。
<br/>　　可现在看到这转轮和尚，都不是楚风眠的对手，他们二人便是立刻改变了注意。
<br/>　　他们的仇敌，还是楚风眠，楚风眠的性格他们也曾经听闻过。
<br/>　　曾经为了圣院宗的弟子罗康，楚风眠就敢踏平了圣龙秘境的两宗，甚至是跟圣龙秘境的高层为敌。
<br/>　　瑕疵必报。
<br/>　　今日他们已经是跟楚风眠撕破了脸皮了，要是让楚风眠活着离开，死的就是他们。
<br/>　　“武道至尊！”
<br/>　　小耀王寒夜身形闪烁，只见在他的拳风之上，古老的力量凝聚了起来，这小耀王寒夜身上的气势，跟刚刚突然之间的截然不同。
<br/>　　小耀王寒夜，能够成为圣院宗的圣子领袖，甚至是足以跟圣院宗的少宗主无道抗衡，他的实力自然没有那般简单。
<br/>　　上一次出手，只是试探，这一次施展而出的，才是小耀王寒夜的最强手段。
<br/>　　“远古！中古！上古！”
<br/>　　轰隆！
<br/>　　小耀王寒夜长啸一声，在他的这一道拳风之上，凝聚了三个时代之中，不同的力量。
<br/>　　以三个时代的武道，集齐大成，所化的这一拳，武道巅峰的一拳。
<br/>　　“果然有着几分实力！”
<br/>　　楚风眠看到这一拳的轰击而来，也看的出来，这一拳跟刚刚小耀王寒夜展现出来的实力，截然不同。
<br/>　　不过任他拳法千万玄妙，楚风眠都以一剑破之。
<br/>　　楚风眠剑锋一挑，顿时施展出一道剑光，只见这一道剑光，冲天而上，化为了漫天剑影。
<br/>　　在这漫天剑影之中，每一道剑影之中，都蕴含着这截然不同的力量。
<br/>　　一剑斩出，宛如万道剑术，同时打出。
