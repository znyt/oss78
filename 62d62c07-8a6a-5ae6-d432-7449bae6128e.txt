　　将另一半玉衡收进储存空间中，来碧山的任务总算是完成，瞥了眼废墟中的叶沉香，叹息一声。
<br/>　　前世今生，恩恩怨怨，该由他们自己去解决。
<br/>　　此时，叶沉香已经恢复正常，眼眸虽空洞却不迷茫，看着满目疮痍的后院，抬头看了眼慕清霄，心中满是恐惧。
<br/>　　她虽承载怨恨，在这极阴之所修习数十年，终于修成鬼丹，却也不过相当于修真界的金丹期。
<br/>　　叶沉香心里非常的清楚，面前看似人畜无害的青年，其实是一尊极其恐怖的存在。
<br/>　　院落中如此破败，却不见主人的踪影，想必是凶多吉少。
<br/>　　“大人已经获得想要的东西，可否放小女子一条生路？”
<br/>　　叶沉香将自己的态度摆在很低的位置，语气非常恭敬，心中执念没有达成，她不想现在就灰飞烟灭。
<br/>　　“你们本不应该存在于世间。”
<br/>　　平静无比的声音如同惊雷一般，令得叶沉香身躯一颤。
<br/>　　整个自闲山庄里，可以说是鬼魂遍地，虽不全是叶沉香这种开启灵智的厉鬼，可一旦放出去，绝对是生灵涂炭。
<br/>　　毕竟，某些没有开启灵智的鬼魂，可不像叶沉香知道思考，见到活人，它们就会化作饿狼，毫不犹豫的扑上来，就像先前那般。
<br/>　　瞥了眼叶沉香，慕清霄道：“普通鬼魂修出内丹来之不易，本座今日就放你一条生路，好自为之。”
<br/>　　闻言，叶沉香如获大赦，压在心间的石头总算落了下来，只要后者不杀她，哪怕让她侍奉也不是不行。
<br/>　　慕清霄自然不知道叶沉香心中所想，否则，非得一巴掌把她拍的魂飞魄散不可。
<br/>　　“也是时候回天墉城了。”
<br/>　　当叶沉香抬起头来，只见周围空间轻微波动了一下，慕清霄的身影就自原地消失的无影无踪。
<br/>　　离开自闲山庄，慕清霄率先前往一趟翠萍村后，告知村长不会在发生先前的情况，便径直离开碧山。
<br/>　　...
<br/>　　...
<br/>　　时光荏苒，岁月如梭，五年的岁月，眨眼间就悄悄溜走。
<br/>　　昆仑山，天墉城。
<br/>　　天墉城乃是天下第一大修真门派，自古流传尊清抑浊的修炼之法，今日正是五年一度，招收新弟子的重要日子。
<br/>　　天墉城方圆数百里内，无数人慕名而来，纷纷希望能拜入天墉城门下修习成仙之术。
<br/>　　昆仑山脚下，天墉城大门口，此时人山人海，人头攒动，喧闹无比，来者都排起一条条长龙，紧致有序。
<br/>　　哪怕温度再高，让他们顶着烈阳站上数个时辰，只要能拜入天墉城，他们也没有任何怨言。
<br/>　　在长龙的前端，摆放着几张桌椅，椅子上坐着数名天墉城弟子，负责给来者登记信息。
<br/>　　在天墉城，弟子分为三种，外门弟子，内门弟子，亲传弟子。
<br/>　　外门弟子，相当于最普通的弟子，一般都是门派中修为资质最差的那种，成就有限，没有修习高深剑术的资格。
<br/>　　只有通过考核后，方能进入内门，接触到一些颇为高深的剑术。
<br/>　　至于内门弟子，等级自然是比外门弟子高级，每月所获得的资源也会多出许多。
<br/>　　亲传弟子，顾名思义，拥有最顶级的修炼资源，最高级的心法。
<br/>　　只有像陵越那种，资质卓越的弟子，运气好可能直接被各堂长老收入门下，这便是亲传弟子。
<br/>　　目前，紫胤真人门下有两名亲传弟子，陵越和百里屠苏，涵素真人门下也有两名亲传弟子，凌端与芙蕖。
<br/>　　“叫什么名字，年龄，籍贯?”
<br/>　　“郝健，十八岁，来自临安...”
<br/>　　“下一个，名字，籍贯？”
<br/>　　“在下朱友荣，二十岁，琴川人士，还请师兄往后多多关照。”说罢，青年从怀中掏出一锭银子，偷偷的朝登记弟子塞了过去。
<br/>　　“嗯，不错，我记住你了，过去吧。”
<br/>　　“多谢师兄，多谢师兄。”
<br/>　　登记弟子悄无声息的接过银子，眼中闪过一丝笑意，顶着太阳暴晒，替这群普通人登记本就烦不胜烦。
<br/>　　当然，识趣的人也不少。
<br/>　　“下一个，姓名，年龄，籍贯？”
<br/>　　“在下云杰，还请师兄多多关照.......”
<br/>　　青年皮肤黝黑，笑起来露出一口黄牙，学着先前朱友荣的语气说道。
<br/>　　登记弟子不耐烦的吸了口气，给他点好处也就算，不给好处，谁肯听你唧唧歪歪个没完没了。
<br/>　　“我问你姓名，年龄，籍贯，在啰嗦就给我滚蛋！”
<br/>　　“.....是是，在下云杰，十九岁，洛阳人士。”
<br/>　　“下一个，年龄，籍贯。”
<br/>　　“风晴雪。”
<br/>　　此时，一名身穿绿裙的女子走上前来，女子看似只有十六七岁左右，乌黑亮丽的长发披在香肩，青涩柔美。
<br/>　　其中一名登记弟子，顿时被她迷得晕头转向，说话都口齿不清。
<br/>　　未等另一名弟子说话，坐在左侧不耐烦的弟子就道：“本门新弟子考核中，从来没有收过女子，这种情况不能收，从哪来回哪去吧。”
<br/>　　风晴雪柳眉一皱，道：“没收过，不代表不能收吧。”
<br/>　　“凌端师兄，你不能因为眼里只有芙蕖师姐，就不让别的女孩子入门吧？”另一名弟子顿时低声开口求情，显然两人的关系很不错。
<br/>　　“你懂个屁啊，闭嘴。”
<br/>　　被成为凌端的师兄低喝一声，顿时吓得另一名弟子不敢在开口求情。
<br/>　　就在这时，风晴雪身后一名容貌俊逸的青年开口道：“这位小兄弟，天墉城并无明文规定，不招收女弟子吧？”
<br/>　　“这里让谁报名我说了算，在多一句废话，你也不用报名了。”
<br/>　　“......天墉城不是修仙的门派么，那有你这样仗势欺人的？”
<br/>　　风晴雪有些气急，她来天墉城是为了找人，要是连天墉城都进不去，更不可能找到自己想要找的人了。
<br/>　　“你说谁仗势欺人啊？”
<br/>　　正当凌端想要开口，背后陡然响起一道声音，来者是一名女子，身穿紫色道袍，年龄与风晴雪相仿，娇俏可爱，宛如带露荷花清雅动人。
<br/>　　看到女子的瞬间，凌端立刻站起身来，脸上堆满微笑的走上前去。
<br/>　　女子不是别人，正是天墉城掌门的掌上明珠，也正是他心中喜欢的女子，芙蕖。
