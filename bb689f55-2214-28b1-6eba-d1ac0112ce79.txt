夏夕绾勾唇，“夏妍妍，你知道么，这十年…我每一天都在想你。”
<br/>
<br/>        “当然相信，我也知道绾绾你总有一天会回来的，所以这十年我不敢有一丝的懈怠，努力的上进，让自己变得优秀，为的就是…再一次的将你打入深渊。”
<br/>
<br/>        夏妍妍作为海城第一名媛，生的娇美可人，声线也好听，哪怕说到“将你打入地狱”时依然带着温柔的笑意，不禁让人有些毛骨悚然。
<br/>
<br/>        夏夕绾看着窗外璀璨的晨曦，缓缓道，“我从深渊而来，从来不畏惧深渊，反而是你，这十年一直生活在天堂里，我保证从你回来的这一刻起已是你和天堂的永别，容你十年已是我对你最大的仁慈。”
<br/>
<br/>        那端的夏妍妍默了片刻，然后挂断了电话。
<br/>
<br/>        夏夕绾收了手机，然后起身下床进行洗漱，不得不说李玉兰虽然生了夏小蝶那个废物，但是夏妍妍却是不容小觑的敌手。
<br/>
<br/>        她不会忘记十年前的夏妍妍是如何的伏小做低，那时她和叶翎是最好的闺蜜，夏妍妍就经常偷偷的站在角落里一脸艳羡的看着她们，当她把她叫过来一起玩时，夏妍妍总是紧张的拽着自己的衣角，害羞又柔怯的样子。
<br/>
<br/>        她和叶翎会聊天，夏妍妍就安静的当个小尾巴，当她回头看她时，她总是腼腆一笑，柔声细语的说自己不会说话，当个倾听者就好。
<br/>
<br/>        甚至有一次夏妍妍为了救她，竟然扑过来被狗给咬了一口。
<br/>
<br/>        现在夏夕绾觉得夏妍妍这上位之位真是走的隐忍又心酸，那时的夏妍妍才是十几岁的孩子啊。
<br/>
<br/>        叶翎是最不喜欢夏妍妍的，那时的叶翎在她耳边就说过夏妍妍是绿茶圣母婊，还叮嘱自己小心。
<br/>
<br/>        只可惜那时的她真的将夏妍妍当成了好朋友，一点都没有防备她，以至于她将自己打入了深渊。
<br/>
<br/>        夏夕绾今天没有出门，陪老夫人在后花园里除草浇水，她在静等。
<br/>
<br/>        下午时分，夏夕绾打开微博，夏妍妍已经上热搜了。
<br/>
<br/>        夏妍妍发了一条微博，配文道听闻噩耗，我匆匆回国，母亲已经犯下大错，不奢求原谅，只求给她一个生的机会了此残生，我愿意承担母亲犯下的一切罪责。
<br/>
<br/>        夏妍妍还配了一张照片，照片是在医院里拍的，李玉兰躺在重症监护病房里，脸上戴着氧气罩，打着点滴，夏妍妍坐在床头紧握着李玉兰的手，悲伤而哀痛。
<br/>
<br/>        夏夕绾笑了，夏妍妍这一条微博发的实在是精湛，全文以一个女儿的身份全方面的展示了自己的无辜和孝意，堪称危机公关的典范。
<br/>
<br/>        这十年夏妍妍是深入人心的，虽然她不混娱乐圈，但是微博粉丝已经有1800万了，而且大都是脑残粉，战斗力爆表。
