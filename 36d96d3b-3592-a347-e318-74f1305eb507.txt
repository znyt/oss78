　　工地面前，一个穿着一身白色中山装，四十多岁的中年人背着手，站在门口。
<br/>　　他便是白家的捉妖师。
<br/>　　“余大师，没想到你那么快就来了”苏羽向前打招呼，显然已经知道了对方的身份，又介绍道：“给你介绍一下，这位是我的朋友涂小安，也是我请来帮忙的”
<br/>　　他叫余子童，虽然是白家的人，但不是白家嫡系，而是投靠白家的修武者，后职业捉妖师。
<br/>　　余子童看了苏羽，淡淡的道：“苏老板你好，没想到你还请了别人，是不是有点多此一举”
<br/>　　他余光看了一眼年轻的涂小安，有点蹙眉。
<br/>　　苏羽解释道：“这工地鼠灾厉害的很，小安兄弟也是修武者，而且手中还有一条觉醒类的蛇种，对付老鼠非常有帮助”
<br/>　　“你知道白家为什么派我来吗”
<br/>　　余子童淡淡的忽然一言。
<br/>　　苏羽一楞。
<br/>　　“因为我有一只宠物，就是老鼠的克星”
<br/>　　“哦...是吗，也是觉醒类？”苏羽眼睛一亮。
<br/>　　这时，大家才看到，原来余子童的怀中抱着一只安静的猫，此猫浑身布满金黄色斑点，其样子像是披上了一件老虎皮，让人眼前一亮。
<br/>　　不知道是不是众人的视线投放在这只猫身上，让它不舒服，此猫猛然的朝人张牙舞爪起来，发出呼啸的声音，凶煞无比，通灵慑历，一双眼瞳既然炯炯有神，散发黄金光芒。
<br/>　　将苏羽吓的都倒退了好几步。
<br/>　　“苏老板不好意思，吓到你了”余子童故作道歉，：“我这只猫，可是觉醒变异猫种，拥有老虎之威，狮子之势，百万猫种中，难处其一，不但是老鼠的克星，还是蛇类的克星”
<br/>　　余子童说着看了涂小安一眼，像是说觉醒类蛇种有什么了不起的。
<br/>　　我这只猫克鼠，还克蛇呢。
<br/>　　涂小安到不在乎对方的贬低，或许是自己的存在让他觉得不爽，既然苏羽请了他，为什么还请别人，是不相信他余子童？
<br/>　　这其实就是一场误会，苏羽也没别的意思。
<br/>　　不过对方手中的那只猫，但是让涂小安有点吃惊，这猫不同凡响呀。
<br/>　　看样子不会比猫精海棠的血统来的逊色。
<br/>　　而且看上去威风凛凛，性情凶猛，简直就是迷你的小老虎，尤其是一双眼睛，闪闪发金光，跟火眼金睛似的，充满威势。
<br/>　　这只眼睛，让涂小安觉得有点熟悉，顿时醍醐灌顶，这是一双阴阳眼呀。
<br/>　　跟自己的黑耳小八觉醒后一样，拥有了看破世间虚无的阴阳眼。
<br/>　　但凡生物能拥有一双阴阳眼，可是非常的了不起，都是一等一的镇宅之兽。
<br/>　　在古代则有“玄猫辟邪”的说法，直至今日都有黑玄猫镇宅的言论，不比狗逊色。
<br/>　　这也是为什么一般人家中，养的不是猫就是狗。
<br/>　　两种生物，要是灵性，都是镇宅之兽，看宅护院。
<br/>　　也正因为如此，猫跟狗像是一山难容两虎，一个家中，要嘛养猫，养嘛养狗。
<br/>　　猫跟狗在一起，就会起争执。
<br/>　　好像在比谁更厉害似的。
<br/>　　“给我洞察”
<br/>　　“叮...19级变异黄金瞳，高等生物，猫中霸主，拥有老虎之威，狮子之势，内蕴阴阳眼”
<br/>　　好一只觉醒猫，虽然还没有成为二阶猫精，但是这血统非常的高贵，已经在海棠之上了，将来会成为非常可怕的妖怪。
<br/>　　涂小安盯着此猫看，可当这只猫看向涂小安的时候，猛然的发狂了起来。
<br/>　　“嗷...”
<br/>　　它虎啸一声，震耳欲聋，直接从余子童的身上跳了下来，落到地上浑身黄金毛发炸立，威风凛凛，凶煞暴躁，一双阴阳眼光芒大亮，直视涂小安，好像是看到了什么敌人，随时准备攻击。
<br/>　　涂小安面色一变，这猫是看出自己是条蛇精了，敌意明显，通灵慑历。
<br/>　　“宝贝乖，不要坏了小朋友”
<br/>　　余子童笑着将地上浑身毛发炸立的黄金瞳抱了起来，抚摸它柔顺的毛发，安慰起来。
<br/>　　但黄金瞳一双有威势可怕的眼睛就是死死的盯着涂小安，充满的警惕。
<br/>　　苏羽以为这是余子童故意给的涂小安下马威，缓解气氛的道：“余大师，你这猫还真的是厉害，比老虎好吓人，我都被吓了双腿发软”
<br/>　　说完，拍了拍涂小安的肩膀，让他别在意。
<br/>　　看到那么厉害的猫，苏羽是非常高兴的。
<br/>　　自古猫抓老鼠，猫就是老鼠的第一大天敌，看来自己的工地鼠灾可以解决了。
<br/>　　“进去吧，我到要看看是什么鼠精作怪，刚好我的宝贝也饿了，今天可以饱餐一顿了”
<br/>　　余子童横刀立马的朝着工地门口走了进去。
<br/>　　苏羽看了涂小安一眼跟了上去，轻轻的说：“小安兄弟，既然如此，我们就全当来看戏好了”
<br/>　　涂小安耸了耸肩膀，对于这只猫充满兴趣，连自己是蛇精都一眼看出来了，还真的不是凡物。
<br/>　　要知道这只猫血统非常高贵，但还没到二阶呢，连口吐人言都做不到。
<br/>　　越如此，越说明它的可贵。
<br/>　　有这只黄金瞳在，到真不需要自己出手了，不是特别厉害可怕的鼠精，都不在话下了。
<br/>　　众人进入工地之中，一阵静悄悄，由于鼠灾，工地里面一个工人都没有。
<br/>　　走着走着余子童蹙眉道：“我既然没有嗅到妖气，而且这里怎么连一只小老鼠都没看到”
<br/>　　“莫非被我的宝贝吓的都四散而逃了？”
<br/>　　苏羽道：“大师莫急，工地平日也看不到老鼠，但只要有人进去了，不出一会儿，那些老鼠就不知从哪里涌了出来”
<br/>　　“我到希望如此”余子童自负的说，不时的摸了摸自己怀中黄金瞳柔顺的毛发，让它安分了不少。
<br/>　　所谓的妖气，自然是觉醒生物身上的气息。
<br/>　　至于他说没有，涂小安暗暗笑了起来，藏匿在工地上的鼠精可不简单，他已经若有若无的察觉到了。
<br/>　　不是修武者可以对付。
<br/>　　涂小安不知道捉妖师的实力是怎么划分的，但是这个余子童，他洞察过，不过是15级的修武者，就是一阶中期的实力。

<br/>　　这种人，怎么可能让一只如此高贵品种的觉醒猫臣服，这就奇怪了。
<br/>　　能拥有老虎之威，狮子之势，说明此猫是非常桀骜不驯的，性情无比凶猛，很难会放下自尊，臣服在一个人类身上。
<br/>　　而且这个余子童，还那么的弱。
<br/>　　可以这样说，要是这只黄金瞳发狂起来，就是来三五个余子童这样的，都要死在这猫手中。
