　　那个站在大厅柱子边，一条浅蓝色牛仔裤，鹅黄色真丝短袖，白色帆布鞋，同样用一副大号墨镜遮住了大半张脸的靓丽身影，赫然就是云熙。
<br/>　　云熙的出现完全出乎罗昊的意料之外，罗昊摘掉自己鼻梁上的墨镜，使劲眨了眨眼睛，确认自己并没有看错后，朝着靠在柱子边的云熙走去，一把将她抱住，在她额头上亲吻了一下，问道：“你怎么来了？”
<br/>　　“在家里太无聊就出来转转。”云熙难得的展露出调皮神色，说道。
<br/>　　“咳咳！”
<br/>　　站在一旁的林晓阳轻轻咳了两声，说道：“爱而不藏，自取其亡！”
<br/>　　“什么意思？”
<br/>　　罗昊眨巴着眼睛，显然是对林晓阳这两句话感到不解。
<br/>　　“意思就是，秀恩爱，死得快。”
<br/>　　李毅峰经过罗昊身边，拉下墨镜，看了眼罗昊和云熙，随后推上墨镜，继续往前走去。
<br/>　　“听……”慕容飞经过林晓阳身边的时候，哼了一句《听海》中的副歌，但是歌词内容却是跟原来的大相径庭，“狗哭的声音……”
<br/>　　周围人顿时一阵哄笑。
<br/>　　林晓阳指着慕容飞，“你你你”了半天都没能说出来一句反驳的话。他根本就没办法反驳，因为慕容飞不但有老婆了，就连孩子都有了。
<br/>　　“听，狗哭的声音……”
<br/>　　“听，狗哭的声音……”
<br/>　　李然、罗昊和云熙从林晓阳身边经过的时候，也是跟着哼了起来；云熙没哼出声，但是嘴角的笑意却是怎么藏都藏不住。
<br/>　　“靠！你们这些家伙！”
<br/>　　林晓阳追上已经走向候机楼大门的一行人，嘴里不断的抱怨道：“我伤势刚好就来支援你们，你们这么做，难道良心不会痛吗？”
<br/>　　“云小熙，我的心好痛。”罗昊苦着一张脸对云熙说道。
<br/>　　“去把那个让你心好痛的家伙打一顿。”云熙一本正经的说道。
<br/>　　“嗯，好主意！”
<br/>　　“靠！”
<br/>　　林晓阳挠着头，无比郁闷，自己当年在天空中翱翔的时候，那粉丝也是无数，是多少少女的梦中情人。这进入了陆特，怎么就成了单身狗了？这难道就是传说中的“落地凤凰不如鸡”？
<br/>　　候机楼停车场，无比郁闷的林晓阳拉开驾驶室的门坐了进去，开车前往在K国设立起来的指挥中心。
<br/>　　罗昊坐在副驾驶上，对林晓阳问道：“这次我们来了多少人？”
<br/>　　说起正事，林晓阳立刻就收起了自己脸上郁闷的表情，说道：“除了我、公主、墓碑、野牛以外，承影和战备一队也来了。”
<br/>　　“来这么多人？！龙头亲自带队！？”
<br/>　　坐在后排的慕容飞探过头来说道：“搞这么大阵仗？！”
<br/>　　“SHADOW这次针对GHOST的战书已经引起了全世界的轰动，不仅仅是等着看这场巅峰对决的世界各国黑客，就连各国政府也是关注起来。”林晓阳解释道：“SHADOW手中的病毒能让全世界网络瘫痪绝对不是危言耸听，各国政府想不关注都不可能。至于子午教官的身份，其实各国情报部门也是有一些推测的，只是大家都没有捅破这层窗户纸而已，可是现在情况不同了，所以龙头亲自出面跟K国方面取得了联系，K国方面也给了我们最大的便利。说白了，这场会令世界网络瘫痪的危机，除了子午教官，谁也阻止不了，所以K国政府想不妥协都不可能。”
<br/>　　商务车从机场离开后，沿着高速公路一路疾驰，大约一个小时后，驶入了一栋写字楼的地下停车库。
<br/>　　地下停车库的电梯外，两个穿着黑色作战服，戴着黑色头罩，黑色头罩外还挂着一副墨镜的SWAT队员握着G36自动步枪正在警戒，看到一行人朝着电梯走来，立刻按下按钮打开电梯门，随后退到电梯门两侧。
<br/>　　当子午从这两名SWAT队员旁边经过的时候，两名swat队员下意识的往子午身上瞟了两眼。
<br/>　　电梯门关上，电梯往上而去的时候，林晓阳对站在电梯中的罗昊、李然等人说道：“这栋写字楼就是K国政府给我们提供的临时指挥中心，顶楼就是我们的指挥室，这次行动由承影亲自坐镇指挥。”
<br/>　　林晓阳把情况进行介绍的时候，电梯已经抵达了写字楼顶楼。电梯门打开，电梯外同样站着两名荷枪实弹的swat队员，不仅是电梯外，就连通往指挥室的走廊上，也是站着K国的swat队员，直到指挥室大门口，警戒人员才换成了战备一队的人。
<br/>　　虽然K国政府声称，这是出于安全的考虑才如此安排，但是所有人都知道，出于安全考虑只是一小部分，更多的，恐怕是出于监视的目的，而且还是光明正大的监视。
<br/>　　本来K国政府甚至想要让自己这边的人进入到指挥中心，不过却是被赵天宇以“防止行动情报走漏”为理由给拒绝了，最开始K国政府自然是不同意，但是面对态度强硬的赵天宇，K国方面的“不同意”最后只能变成“妥协”！
<br/>　　指挥室大门口，两个负责警戒的战备一队队员看到罗昊、李然这些人，脸上立刻露出微笑，帮忙把门打开，说道：“龙头在里面等着你们呢。”
<br/>　　“你们来了。”
<br/>　　赵天宇手中拿着一支铅笔，抬头看了眼走进房间的几个人，说道：“你们都过来看一下。”
<br/>　　桌子上铺着一张打印出来的卫星地图，赵天宇已经用铅笔在卫星地图上将一栋建筑圈了起来，说道：“这里就是SHADOW要求对决的地方，这栋写字楼是一家美国人的公司，但是经过核查，可以确定，这家公司注册时的法人只是挂名而已，跟这家公司没有任何关系，这家公司很有可能是‘暗月’的产业。”
<br/>　　“‘暗月’？”罗昊皱眉，问道：“SHADOW果然是跟‘暗月’有关系吗？”
<br/>　　“原本也只是怀疑这种可能。”赵天宇放下铅笔，把几张照片扔到罗昊面前，说道：“但是我们现在有足够的证据证明，SHADOW确实跟‘暗月’存在着联系，而且还是很深的联系！”
