　　修炼的斗气功法等级的高低，也是决定日后成就高低的关键，比如修炼玄阶中级功法的人，自然要比黄阶高级功法的同等级的人要强上几分。
<br/>　　斗气大陆，分辨强弱，取决于三种条件，三种条件无疑是自身实力，功法等级，斗技等级，从三种条件上看来，任何位面都是如此。
<br/>　　将复杂的情绪抛之脑后，慕清霄的心神落在系统发布的任务上，第一个任务是谈情说爱，苦笑一声，心神放在第二个任务上，掠夺异火。
<br/>　　说到异火，慕清霄手心窜起一缕暗黑色的火焰，自己的九霄琉璃焱可是凤凰的本命火焰，或者根本不能称之为火焰，而是神通！
<br/>　　而斗气大陆的异火，顶天也就是天地衍生出的灵火，两者根本不再一个层次。
<br/>　　慕清霄倒是知道几样异火的地点，比如药尘的骨灵冷火，古薰儿的金帝焚天炎，但没有足够的实力，却是很难得手。
<br/>　　斗气大陆修炼等级森严，由低至高，分别是：一至九段斗之气，斗者，斗师，大斗师，斗灵，斗王，斗皇，斗宗，斗尊，斗圣，斗帝。
<br/>　　正所谓万千大道殊途同归，修炼体系也是如此：
<br/>　　九段斗气与斗者，相当于练体期，斗师相当于练气期，大斗师相当于筑基期，斗灵相当于金丹期，斗王相当于元婴期，斗皇相当于化神期，斗宗相当于练虚期，斗尊相当于合体期，斗圣相当于大乘期，渡劫期相当于斗帝。
<br/>　　而慕清霄现在的修为就是化神一重初期，也就是相当于一星斗皇，在前期剧情中绝对有自保的能力，而且他还有自己的法宝，一剑在手天下我有。
<br/>　　对于修士而言，法宝是实力的一部分，因为心法的缘故，他比同期修士，本就占据优势，更何况还有九霄琉璃焱，化身成黑凤后，修为更是暴涨。
<br/>　　哪怕对上高阶斗皇，他也有一战之力，不过斗气大陆以斗技为主，威力肯定不凡，而且慕清霄的性格本就是小心谨慎，谋而后定，自然不会乱来。
<br/>　　回过神来，慕清霄沉思起来，也不知道剧情有没有开始了，而且他也不清楚自己脚下是何处，毕竟这片山脉可是看不到边际。
<br/>　　想到此处，慕清霄的视线就落在远方的城墙上，看来得先进城一趟，确认自己所在的位置，至于其他事情再做定夺。
<br/>　　几秒钟后，慕清霄就出现在城门口，望着城墙上的三个大字，愣愣出神：“乌坦城？”
<br/>　　乌坦城，加玛帝国的一座小型城市之一，位于东北省份，靠近北部省份的黑焰城、东北省份的青山镇，魔兽山脉、东部省份的黑岩城之间。
<br/>　　“看来，自己刚才所在的地方就是魔兽山脉中了。”
<br/>　　魔兽山脉，其中魔兽横行，危险重重，单人行走在其中，最容易遭来魔兽的进攻，除了一些艺高人胆大的之外，大多都是成群结伴的进入。
<br/>　　乌坦城虽只是加玛帝国范围内的一座小型城市，但面积广阔，相比地球上的一线城市静海市还大，城墙高达十丈，毕竟此处靠近魔兽山脉。
<br/>　　城门口人来人往，车水马龙，小贩的吆喝声笼络不绝，眼中尽是一片繁荣昌盛的景象。
<br/>　　夕阳西下，落日的余晖照耀着大地，由于容貌的缘故，慕清霄自然招来了许多人的注意。
<br/>　　于是，他只好收敛气息，宛若普通人一样走进乌坦城中，由于并没有带行礼，所以进城的时候也不用接受检查。
<br/>　　进入乌坦城后，慕清霄则悠闲的走在繁荣的大街上，街上是各种商店，比如售卖药材，珍贵的魔兽材料等等，哪怕是傍晚时分，城内依旧人气鼎盛。
<br/>　　忽然，慕清霄止住脚步，踏进一家偌大的药材店中，店中有不少顾客正在选取药材，而他则径直走向柜台。
<br/>　　掌柜戴着一副眼镜，看似普通，其实是一位三星斗者，在这乌坦城中也算是小有实力，察觉到慕清霄的带来，抬起头来，眼中有些诧异。
<br/>　　诧异过后，掌柜将眼镜向上推了推，道：“这位公子，我们店里的药材是城中最齐全的，淬体药材，提升斗气的药材应有尽有，不知道您想买些什么药材啊？”
<br/>　　慕清霄微微一笑，道：“掌柜，我不是来买药的，而是来鉴定药材的。”
<br/>　　“......”
<br/>　　掌柜顿时有些无语，他这里是药材店，这小子是不是走错地方了，但表面还是客气的说道：“公子可否将药材给我看看？”
<br/>　　闻言，慕清霄拿出容器悄无声息的从中取出一朵鲜花，试探性的问道：“掌柜，你可认得这是何种药材？”
<br/>　　他手中的花朵手掌大小，呈鲜艳的天蓝色，散发着淡淡的清香，正是他当初在亿年神木上摘取的，只是不知道有何功效。
<br/>　　见到乌坦城中有售卖药材的商店，他便抱着尝试的心态，走进来看一看，瞬间鉴定下自己手中的花朵，到底是什么灵药。
<br/>　　望着慕清霄手中的花朵，掌柜只是瞥了一眼，随口道：“这只是普通的鲜花，魔兽山脉中随处可见。”
<br/>　　慕清霄微微点头，不动声色的将花朵收了起来，随后向店外走去，虽然看似普通，但他却隐隐觉得，这花朵有猫腻。
<br/>　　亿年神木的存在本就是个谜，生长在上面的花朵怎么想都不可能普通，很可能与天帝宝库的主人有关，绝非凡物。
<br/>　　至于掌柜的鉴定不出来，慕清霄也并不意外，他本就不认为，一个小小的斗者，能够鉴定出鲜花的价值，要真能鉴定出来才有鬼了。
<br/>　　在城中行走了片刻，抬头看了看天色，夜幕已经降临，慕清霄可不想露宿街头。
<br/>　　片刻后，他就来到一家生意不错的酒店，点了一些酒菜，坐在客栈的角落里，听着客人们畅谈着乌坦城中的一些趣事。
<br/>　　“难得回来一趟，正好碰上萧家的年终考核了，明天得去凑凑热闹，不知道萧家今年会不会诞生新的天才。”
<br/>　　“说到天才，萧家有一位资质卓绝的天才，好像十岁就成为斗者，也不知道两年过去，现在是何境界了，真是前途无量啊。”
