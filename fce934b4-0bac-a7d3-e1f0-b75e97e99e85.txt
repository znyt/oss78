
<br/>　　<br/>　　    一晃过去了两天，川中一条充满了林荫，但车辆却很少的道路上，胡胡没开那辆奔驰小跑，而是开了一辆很普通的帕萨特，来到一处有人站岗的大铁门外，车子停到一旁他打了个电话，没过多久门里那边就走过来个四十多岁梳着标准三七分穿着长袖白衬衫的中年，冲着他招了招手。<br/>　　<br/>　　    几分钟后，一个独栋小院里，胡胡把车停好然后从车里拿出两盒茶叶走进了院子，里面有个头发花白的老人正在拎着水壶浇花。<br/>　　<br/>　　    胡胡拿着茶叶站在一旁也没说话，一直就看着老人浇完话，然后又翻了下土，足足过去十五分钟后好像才收拾完，就从白衬衫中年手中接过毛巾擦了擦手，指着一颗老树下的座椅说道：“坐吧”<br/>　　<br/>　　    胡胡把两盒茶叶放在桌子上，笑道：“我爸以前留下的明前茶，特意给您带过来的，知道徐老爱这一口，您过后好好品品”<br/>　　<br/>　　    白衬衫中年接过茶叶走了，片刻后又端了个果盘和一壶水放到桌子上，胡胡挺随意的拿起一块西瓜啃了两口：“这西瓜真甜，也脆”<br/>　　<br/>　　    “小胡啊”<br/>　　<br/>　　    “唉……”<br/>　　<br/>　　    “你这是无事不登三宝殿，借茶献佛来了吧？”徐老端起茶杯抿了一口。<br/>　　<br/>　　    “呵呵，您真是火眼金睛，我一来就知道是有事要麻烦您了”胡胡放下西瓜，擦了擦嘴说道。<br/>　　<br/>　　    “呵呵，没事你也不会来看我的，老了退下来了，就门可罗雀了”徐老放下茶杯淡淡的说道。<br/>　　<br/>　　    “您老可别谦虚了，您要是说想开门见客的话，我估计我都不一定能排得上号，您家的门槛不得让人给急踏破了啊？”<br/>　　<br/>　　    “行了，你可别恭维我了，我一会还要补个觉呢，有事就说吧”徐老摆了摆手，催促道。<br/>　　<br/>　　    “是这样的，徐老我和沈君有点误会，可能不太好解开，在蜀地沈君很跋扈，我怕万一解不开的话跟他会有冲突，所以我想请您出面帮着说和一下，我肯定姿态放的很低，有我做错的地方就赔礼道歉，他就是甩我两巴掌的话，我也肯定笑脸相迎……”胡胡这话说的很实在，而他也确实打算咽了这口气，真没想着要和沈君摆出什么刀枪炮来。<br/>　　<br/>　　    沈君不知道是他插手了沈沉鱼的事那自然就算了，可要是知道了的话，他还真不愿意跟对方撕扯下去，这和利益不符。<br/>　　<br/>　　    徐老皱了皱眉，说道：“你们年轻人，就不能安生一点？这是日子好过了，闲的么？现在的年轻人，太浮躁了！”<br/>　　<br/>　　    “是，是，您教育的对，我有的时候确实太年轻了，欠缺点锻炼”<br/>　　<br/>　　    徐老哼了一声摇了摇头，然后摆手说道：“你回去吧，今天就不留你吃饭了”<br/>　　<br/>　　    “好叻，那我过段时间再来看您”胡胡连忙站起来，恭谨的弯了下腰，然后抬头朝着那中年点了点脑袋，就走出了小院。<br/>　　<br/>　　    胡胡走了以后，白衬衫中年就把徐老搀了起来，他说道：“赵秘书，一会你给沈君打个电话吧，小孩子闹归闹，就别上升到阶级斗争层面了”<br/>　　<br/>　　    “知道了，领导”<br/>　　<br/>　　    沈君还在紧锣密鼓的查着胡胡的行踪，准备出手报复的时候，他这边刚上了发条，一个很突兀的电话就打了进来，是徐老现在的生活秘书，赵秘书给沈君打的。<br/>　　<br/>　　    看着屏幕上的电话号码，沈君有些为难了，他现在和赵老那边，差不多几个月都不会联系一次，但不联系不代表他们没有关系，因为袍哥在蜀地背后有几把大伞在照着，其中比较大的一把就是徐老。<br/>　　<br/>　　    而在这时候对方打电话进来，那意图就不言而喻了，可这电话又不能不接。<br/>　　<br/>　　    “喂？赵秘书，呵呵什么风把你吹了过来？”沈君笑呵呵的寒暄了一句后，瞬间就把话题给扯开了：“哎呀，最近有段时间没联系你了，别见怪啊赵哥，实在是因为家事太难处理了，我妹妹小鱼儿前段时间差点让人给祸害死了，我现在正为这事头疼呢”<br/>　　<br/>　　    电话那头，赵秘书听到沈君的话后就顿了顿，两人都是人精，在这个关键时刻，赵秘书打电话，沈君就用话给他下面的说辞堵住了，话里面全是语言艺术。<br/>　　<br/>　　    赵秘书轻声问道：“没缓了是么？”<br/>　　<br/>　　    沈君皱眉说道：“老哥，这句话是你自己问的，还是徐老问的？”<br/>　　<br/>　　    沈君话说的一点都不隐晦，点的很明白，那意思是如果是你过问的话，这个话题咱就到此为止吧，但要是徐老问的，他就得琢磨一下了。<br/>　　<br/>　　    “君啊，你和老领导都相识多少年了，特别是他退了以后，轻易不会在找你什么了，明白么？”赵秘书不轻不重的拿话点了他一句。<br/>　　<br/>　　    沈君的脸“唰”的一下就阴了，但嘴上却笑道：“您放心，我懂不懂事您和领导心里没谱么？我什么时候给你们找过麻烦啊，受点委屈也没什么啊，我也是吃着苦走到今天这步的”<br/>　　<br/>　　    “你明白就好，那就这样吧……”<br/>　　<br/>　　    “啪”挂了电话，沈君一挥手就把桌子上的东西给扫了下去，站起来手叉着腰咬牙说道：“真他么的窝囊，明明我是受害的，还得打碎了牙往肚子里咽”<br/>　　<br/>　　    沈君站在窗口，抽着烟，无奈的叹了口气摇了摇头，然后给王惊蛰去了一个电话。<br/>　　<br/>　　    “惊蛰啊，那件事暂时只能放下了”<br/>　　<br/>　　    王惊蛰顿时诧异的问道：“这是什么意思呢？”<br/>　　<br/>　　    “对方给了我一个老领导打了电话，他出面了，我就不得不卖一个面子过去，不然不太好交代，他们这种人最反感的就是自己有小脾气，不听话的人，而我却也恰恰不能不听他的话，你明白了么？”<br/>　　<br/>　　    王惊蛰“哦”了一声，接着说道：“那你岂不是亏了，白白送了我一份大礼”<br/>　　<br/>　　    沈君顿时笑了，说道：“送你的我不心疼，朋友得交心，交的又不是买卖，一件东西罢了我没什么舍不得的，你好好笑纳就行了”<br/>　　<br/>　　    两人简单的聊了几句就结束了通话，王惊蛰跟小草耸了耸肩膀，说道：“白捡了个便宜……”
