　　山阳山之东，人族汇聚之地，多了一个讲道的圣者。
<br/>　　此后五百年中，太清圣人传道人族，渐渐在人族有了很大的声望。
<br/>　　这一日，太清圣人心有所感，来到人族一处大型汇聚之地，他手握造化青莲所化的拐杖，腰挂紫金葫芦，头顶三十三层的玄黄宝塔，霞光万丈，好一副仙风道骨的做派。
<br/>　　他口吐大道真言，传遍人族祖地，只听得众人如痴如醉，越来越多的人族武者来到此处，听圣人讲道，这对他们来说，是一场难得的机缘。
<br/>　　太清圣人讲的是他根据炼丹之法所创出来的金丹大道。修士修炼金丹，丹碎成婴，进而修成元神，证得仙道的法门，就如同一个生灵一生的轨迹，从蛋中出生，成为婴儿，之后渐渐成长为童年、少年、青年、中年、老年，这如同生命一般的奇迹，非常适合人族修炼。
<br/>　　老子不歇不停，一连讲道三个月，共合九九八十一天，众人心中震撼，却见太清圣人身上突然飞出一道金桥，直连天地，却是先天至宝太极图所化。
<br/>　　太清圣人端坐金桥之上，声音宏大，片刻便传遍洪荒众生：
<br/>　　“吾为盘古氏太清圣人，号太上道主道德天尊，今于洪荒之中，立下一教，名曰‘人’，吾当为人教教主，以先天至宝太极图镇压人教气运，自今以后，行功德之事，教化人族！”
<br/>　　说罢，已经是圣人修为的老子全身气势大涨，周身紫气环绕，一股浩大的圣人身影屹立而起，直顶九天天，周身法则玄奥异常。
<br/>　　九天之上，金灿灿一片，天道有感，降下无量功德，太清元神之中，盘古开天所留下的开天功德，在这一刻被引了出来，再加上九天之上落下的功德，在太清的脑后，凝聚出一轮金光灿灿的功德金轮。
<br/>　　浩大、辉煌、绚烂、宏伟、璀璨……
<br/>　　太清圣人被这浩大的功德金轮衬托的仿佛诸天神明，一股玄妙的道则在他周身环绕，玄之又玄！
<br/>　　老子再次以无量功德铸就至人法体，成就天道圣人！
<br/>　　洪荒之中，无数生灵尽皆不自然地伏地跪拜，口中大呼：
<br/>　　“圣人无疆！”
<br/>　　如此浩大的威势，自然是惊动了无数大神的瞩目，人们都知道太清已经成圣了，但如今却再次弄出如此大的声势，到底是怎么回事？
<br/>　　人族之中，许多人借助此次太清圣人立教之举，一举突破许多境界，不管是仙道，还是武道，都有很大的进步，武道法相境有数十人突破，人族三大初祖更是突破到了转轮武皇境，堪比洪荒准圣。
<br/>　　人族实力大涨，更胜先前。
<br/>　　就在洪荒众多大神通者们纷纷猜测太清圣人为何再次成圣之时，昆仑山方向再次传来一股浩大的神威，与太清圣人一般无二，轰动洪荒。
<br/>　　“吾名玉清元始，号天罚圣主道印天尊，今于洪荒之中立下一教，名曰‘阐’，阐者，明也，应天顺道，阐述大道。吾为阐教教主，天地万物，有缘者，可入吾教，得吾道！”
<br/>　　九天之上一阵涌动，风起云涌，出现一大片金光灿灿的功德金光，无量功德金光向着元始落了下来，元始并没有将它们炼化，用来增长修为，而是将它们炼成一道浩瀚辉煌的功德金轮，他头顶功德金轮，无法不侵，诸神退避，群邪辟易。
<br/>　　虚空之上，传来一阵阵浩大的吟唱，诵经声，好似诸神的祝福，赞颂。
<br/>　　至于阐教的气运，元始手中还缺少镇压气运的灵宝吗，不说鸿蒙元始印，就是因果莲台、净世白莲、盘古幡等，哪一样不是镇压气运的灵宝，有了它们，还怕阐教气运流逝吗？
<br/>　　仙音阵阵，异香缥缈。混沌从来道德奇，全凭玄理立玄机。
<br/>　　太极两仪并四象，天开于子任为之，地丑人寅吾掌教。
<br/>　　还没等洪荒众生反应过来，随即太清通天的浩瀚圣音便再次传了出来：
<br/>　　“吾为盘古氏上清圣人，号通天剑主灵宝天尊，今于洪荒立下一教，名曰：‘截’，截者，断也，大道五十五，天道五十，天衍四九，遁去其一，截教取意截取那一线生机，吾为截教教主。道法自然，天下万物，无物不可成道！三界之中，芸芸众生，若是有缘，皆可入我截教。”
<br/>　　天上同样降下无量功德，功德一入通天体内，便被他炼化，用来增长修为，刚刚突破圣人一重天不久的通天，修为再次突破，一直来到圣人二重天中期，才缓缓停了下来，最后停在圣人二重天中期巅峰，只差一步，便可进入后期。
<br/>　　又是一道混沌色光芒向着三十三天外的混沌中冲去，洪荒世界中出现阵阵仙音，异香弥漫。
<br/>　　鸿钧生化见开天，地丑人寅上法台。
<br/>　　炼就金身无量劫，碧游宫内育多才。
<br/>　　继上次成道后，三清再次立教得无量功德，成就功德至人，惊动洪荒。洪荒中人，谁也不知三清为何会如此，他们只知道，三清早已成圣，今日之事，他们猜测三清立教，恐怕是为了传下圣人道统。其中，女娲多少有些猜测到其中的缘由，因为她自己就是以功德成圣，三清这番作为，与自己当初何其相似！
<br/>　　不由得，她的脸色变得更加难看了，圣人也分三六九等，如今，自己恐怕就要落到最底层了。
<br/>　　三清成圣，圣威轰动洪荒，传到西方灵山。接引道人与准提道人面面相觑，脸上都不由得露出苦涩的笑意：
<br/>　　“如今东方除了红云之外，具已成圣，通过今日三清立教，我也得以窥探到一丝成圣之机，我等也该成圣了。”
<br/>　　听闻接引所言，准提自然没有意见，当即应道：“大善！”
<br/>　　于是，灵山脚下在这一日也传来两道宏大的声音：
<br/>　　“吾为接引（准提），今日有感洪荒生灵涂炭，遂立下一教，名曰‘释’。导众生向善，无苦无忧，为极乐之地。”
<br/>　　天道有感，降下无量功德。二人虽然修为上涨，但并没有突破到圣人境界。二人毕竟没有如三清那般，为盘古元神所化，身具盘古开天功德。立教气势不过只是一个引子，功德不够，何以成圣？
<br/>　　二人眼睛一瞪，顿时就急了，难道要让洪荒中其他人看笑话吗？不，不行！此时一定不能怂！
<br/>　　“师兄，你我如今该怎么办？”处于这等尴尬的境地，准提不由得急声道。
<br/>　　“无妨，跟着我念即可，只不过，我等以后可就有得忙了。”接引眼神一定，大声道。
<br/>　　接着，二人忙向天道发出大宏愿，阵阵梵音唱声，传遍洪荒大地：
<br/>　　“立教第一愿，我作佛时……”
<br/>(本章完)
