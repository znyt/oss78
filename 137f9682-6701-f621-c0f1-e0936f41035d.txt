　　第444章
<br/>　　一路向北，逃不知有多久，但见身后没有追兵赶来，程大雷才算稍稍松口气。
<br/>　　他骑着黑牛，在一块大石前停下，回头问：“军师，你没事吧？”
<br/>　　程大雷最担心的就是徐神机，至于银眸，本不用程大雷担心。
<br/>　　从战斗开始的时候，徐神机一直被银眸搭在马背上，提心吊胆，实实在在从生死线前走了一遭。此刻他才算是恢复自由，站在地上大口大口的喘气。
<br/>　　“不行了，大当家，我实在撑不住了。”
<br/>　　程大雷皱起了眉头，根据眼前的情况判断，继续往下走，情况只会越来越危险。程大雷也没想到，天下那么多人想杀他。他和银眸都有自保的实力，唯有这徐神机……实在是一个很大的累赘呐。
<br/>　　“大当家，你可千万不能抛下我啊，我这把老骨头还不想死在这里。”
<br/>　　程大雷撇撇嘴：“我还以为你为全大事，甘愿自我牺牲呢。”
<br/>　　情况的确变得严峻，程大雷忍不住去想，自己孤身到并州，是否有些太轻敌了。若真的是孤身前往，程大雷也不担心，关键是还带着徐神机这废物，当真是令人的心情好不起来。
<br/>　　正在此时，耳边忽听得马蹄声急响。程大雷扭头往过去，只见从前方杀出一捎人马，前后左右，都有敌人赶到，以大石为中心，将程大雷三人围在当中。
<br/>　　“我擦，又来！”程大雷一激灵，对方咬得真紧呐，一丁点喘息的机会都不给。
<br/>　　而这次针对程大雷的围杀行动，没人打算单对单，按照江湖规矩来。每人出动，都携带着骑兵队伍，就是想用人海战术将程大雷斩成肉酱。
<br/>　　此刻风吹草动，显出一根根朝天的长矛。
<br/>　　程大雷举目四望，见四面八方已经被骑兵包围。看他们身上的穿着……呀，老熟人，来自西北的凉州兵。
<br/>　　“带队的是哪个呀，出来打声招呼吧。”程大雷大喊。
<br/>　　“程大雷，你还认得我么！”
<br/>　　一人骑马立在队伍前面，咬牙切齿，双目充满恨意。
<br/>　　还真是老熟人，乃是在长安见过的屈九江。三年不见，他似乎没多大改变，依旧一张苦瓜脸，像每个人都欠他二百块钱似的。
<br/>　　当年，程大雷同时搅黄了柳轻名与屈九江的好事。屈九江和柳轻名的心情还是不同，柳轻名是世家出身，得不到这武状元，自然还有一份前程等着他。但屈九江则是寒门子弟，一路向上爬，爬得很艰辛，任何一个机会都是不能放过的。
<br/>　　程大雷当初断了他的前程，这三年他都没找到翻身的机会。今日仇人见面，屈九江份外眼红，只有杀了程大雷，才能洗刷干净三年前的耻辱，重新做人。
<br/>　　今日，他并不是没有报仇雪恨的机会，为了执行这次任务，相府给他派了三百轻骑兵。这还是次要的，神龙见首不见尾的十杀士，相府派出了六位。
<br/>　　如今，程大雷已经是案板上的肉，任人宰割。
<br/>　　一股快意从心底泛起，报仇雪恨就在今日。
<br/>　　“程大雷，你的死期到了，明年的今日，就是你的死期。”
<br/>　　“喂，小心！”
<br/>　　屈九江身后一人突然喊道。屈九江下意识抬起头，就见天空中一个黑影袭来，距离自己越来越近，在视线中也是越来越大。
<br/>　　然后，咚地一声，从天而降砸在屈九江身上，将他连人带马砸个稀巴烂。
<br/>　　程大雷也是吓了一跳，回过头，就见银眸拍了拍手，似乎什么事情都没有发生过。
<br/>　　就在刚才，银眸竟然抓起那磨盘大小的石头，整个抡了出去，不偏不倚，直接砸了过去。怀揣一腔仇恨的屈九江，还没来得及施展本事，就被砸成了肉酱。
<br/>　　这姑娘的力气，还真是令人恐怖呐。
<br/>　　就连程大雷，心底也是泛起一阵寒意。
<br/>　　但此时此景，已顾不得思索太多，翻身上了牛背，提着大斧头，口中爆喝一声。
<br/>　　“杀呀！”
<br/>　　刚喘匀气的徐神机，又一次被银眸拎了起来，他挣扎着大叫：“放开我，放开我……”
<br/>　　当然，在银眸的控制下，他这种挣扎也起不到任何作用。
<br/>　　程大雷当先锋，带着银眸杀进了敌军中央。
<br/>　　相比其他人，凉州与蛤蟆城的距离颇近，关于程大雷的恶名也听说过太多。今天又看到，银眸一石投死了屈九江，更是未战先怯。
<br/>　　二人在队伍里大杀特杀，当然，程大雷又借此狠狠收割了一波恐惧值。
<br/>　　……
<br/>　　夜，杀虎口前，京州军营。
<br/>　　山羊胡老者听着手下的汇报，一张脸越来越难看。
<br/>　　“启禀相爷，屈九江率领的凉州兵与程大雷相遇，双方发生交战。屈九江牺牲，十杀士死了四个，程大雷突破包围，向北方逃了，现在下落不明。”
<br/>　　“京州兵三百，加上十杀士，都没能留住他！”
<br/>　　手下跪在崔相面前，头低着，口中道：“之前，他们还遇着扬州的星字军，双方打过一仗。”
<br/>　　“扬州派出多少兵？”
<br/>　　“有三百人。”
<br/>　　“程大雷带了几个人？”
<br/>　　“好像只有两个人，加上程大雷一共三人。”
<br/>　　“三个人对三百人，也能叫打仗？”崔相冷哼一声，脸上表情已经十分难看。
<br/>　　手下人心惊胆战，不敢说话。崔相叹了口气，轻轻摆了摆手。
<br/>　　大帐内只剩下两个人，一个是崔相，另外一个就是李行哉。他坐在椅子上，双目微闭，似乎在打盹，好像听不见刚才的话，也好像发生的情况并没有出乎他的预料。
<br/>　　崔相顿了顿，随之笑了笑道：“不曾想那程大雷如此了得，摆下天罗地网，依旧是困不住他。说起这程大雷，殿下一定是熟悉的。”
<br/>　　“哈，确实是见过几面。”李行哉打了个哈欠，这才睁开眼睛：“比一般人知道得多些。”
<br/>　　“那依殿下所观，这程大雷究竟是怎样一个人？”
<br/>　　“怎样一个人？”李行哉像是沉思了片刻，道：“坏人，恶人，十恶不赦之人。崔相有所不知，我当初就是被他囚在蛤蟆寨，可以说是受尽虐待呐，提起他，我就恨不得生食其肉。”
<br/>(本章完)
