　　将复杂的情绪抛之脑后，慕清霄将心思放在掠夺任务2上，其目标是掠夺剧情中的魔剑。
<br/>　　魔剑乃是景天的前世龙阳所铸，因姜国公主龙葵的室女之血而成。
<br/>　　春秋末年，姜国民女“离”以刺绣名动天下，被姜王纳入皇宫为后。次年，产子名“阳”；五年后，产女名“葵”。
<br/>　　越十年，杨国挥师进攻姜国，姜国不敌，求助于齐，齐王以离后手绣一“江山社稷图”为条件，姜王应允，齐国派兵保卫姜国。
<br/>　　两年后，离后积劳成疾，绣品未成而呕血而死。齐国撤兵，杨国围困姜都城，姜王染疾，朝政暂由太子阳代理。
<br/>　　古姜国太子龙阳依历代相传的手卷铸造魔剑。
<br/>　　据载，魔剑可将人的怨气变成自身的灵力，仇恨、战意、怨念等，都是魔剑力量的源泉。
<br/>　　龙阳希望可以凭借魔剑打退强敌杨国，但剑未铸成，国已先破。
<br/>　　公主龙葵投身铸剑炉，杨国士兵将龙阳及姜国皇宫中的诸多守卫投入铸剑炉焚烧，由此产生的极大怨念使魔剑天成。
<br/>　　魔剑脱出铸剑炉，以自身强大的灵力将方圆百里化作废墟，杨国国力因此役而受重创，不久国灭。
<br/>　　后于西晋末年，姜国故宫（剑冢）被发掘，魔剑落于江湖人手，引起武林腥风血雨。
<br/>　　蜀山仙剑派第三代掌门人降服魔剑，并将其镇于锁妖塔顶，然而魔剑并不能被塔所制，后又出塔，被心怀不轨者带至不周山，被慕容紫英寻得并尝试净化。
<br/>　　慕容紫英居于剑冢，偶遇九州散人，将魔剑转赠于他，慕容紫英随后离开剑冢。
<br/>　　九州散人举办散剑大会后拜入了蜀山仙剑派，魔剑也因此重归蜀山，且因为净化之故，留于锁妖塔顶。
<br/>　　若干年后，重楼闯入锁妖塔顶，寻飞蓬佩剑镇妖剑未果，得『魔剑』，赴渝洲交与景天。
<br/>　　也因此，锁妖塔因重楼损坏，以邪剑仙为首的大量妖魔逸出。
<br/>　　而此刻，剧情并未开始，魔剑依旧留在锁妖塔内，蜀山乃是修仙重地，与古剑奇谭中的天墉城类似，可也不是说闯就能闯的。
<br/>　　最起码，以慕清霄现在的实力，略有不足。
<br/>　　既不能强闯蜀山，慕清霄现在就唯有等待，等重楼将魔剑从蜀山带出来，未必就没有机会。
<br/>　　现在的景天并非前世的飞蓬，只是个彻彻底底的普通人，魔剑在他手中，根本不可能发挥出全部的威力。
<br/>　　到时候，他想要获得魔剑，轻而易举。
<br/>　　在弱肉强食，强者为尊的世界中，明知不可为而为之，并非勇敢，而是愚蠢，失败即使死亡！
<br/>　　至于最后的掠夺任务，土火雷风水，五颗灵珠。
<br/>　　其中第一颗土灵珠在古藤林中，林中有强大的妖族强者看守，想必也不会太容易获得。
<br/>　　第二颗火灵珠则在火鬼王手中。
<br/>　　火鬼王本是上古时期遭受天火焚身而死的人类，死后化为火鬼，因为贪图鬼界的自由，而不愿意投胎转世，经过数千年的修炼，具有极深的道行。
<br/>　　她居住在鬼界深处，名义上从属于鬼王，实则不受鬼王的限制，而且能够利用天然岩浆阻止鬼王的进入。
<br/>　　火鬼王法力高深，所放出的火焰是来自地狱的冥火，可直接将魂魄焚为灰烬，但却对肉身没有伤害。
<br/>　　这些年来，火鬼王一直居住在极乐世界酆都，乃是极乐世界的统治者，靠着火灵珠来维持她千年不变的美貌。
<br/>　　拥有鬼王称号的鬼物，实力恐怕都不会弱。
<br/>　　当初在古剑奇谭中，慕清霄在自闲山庄中遇到的梦魔也是属于鬼怪的一种。
<br/>　　它们拥有的手段时常很多，当初若非他恰巧掌握了一点空间能力，根本不可能将梦魔留下，所以他清楚，这种东西极其难对付。
<br/>　　当然，火鬼王并非梦魔。
<br/>　　想到此处，慕清霄眼底闪过一抹精芒，火鬼王释放出的火焰乃是来自地狱的冥火，可以焚烧魂魄，可对肉身却没有伤害。
<br/>　　就是不知道，当火鬼王的冥火，遇到自己的本命神火‘九霄琉璃焱’时，会露出如何精彩的表情。
<br/>　　第三颗雷灵珠在雷州，自云霆出生时，便依附与他的体内，第四颗风灵珠在神界，神树掌管者夕瑶手中。
<br/>　　最后一颗水灵珠就在剧情女主紫萱手中。
<br/>　　有传闻五灵珠，乃是女娲将水、火、雷、风、土，五种巨大的自然力凝聚而成的五颗灵珠。
<br/>　　五灵之力被女娲补天遗下的彩石所吸附，成为五灵珠，是人间最强之力。
<br/>　　想到女娲，慕清霄不禁想到，在古剑奇谭中，那股窥探自己的神秘人物。
<br/>　　微微摇头，慕清霄自嘲一笑，自己不过是个合体期的小人物，怎么可能招那种层侧的人物关注，估计是自己想多了吧。
<br/>　　将复杂的情绪抛之脑后，五颗灵珠中，无疑只有土灵珠与雷灵珠最容易获得。
<br/>　　古藤林中的妖族强者在强，最多也就不超过合体期。
<br/>　　剧情中，景天持有魔剑闯入古藤林寻找土灵珠，当时靠的完全是魔剑。
<br/>　　而景天本身实力并不强，如果硬要找个原因，那就是主角光环占据一定因素，而妖族本身估计也不会强大哪里去。
<br/>　　雷灵珠在云家云霆体内，想到云霆的遭遇，慕清霄就忍不住唏嘘，这种人真是倒霉到了姥姥家，估计喝口凉水都塞牙。
<br/>　　因为雷灵珠的缘故，云霆自带雷电之灵力，刚一出生，他的娘亲就被雷电给劈死。
<br/>　　在十八岁时，与刺史府的丫鬟雨舒成婚，结果在新婚之夜，云霆灵力失控杀了自己的妻子。
<br/>　　微微摇头，慕清霄就没听说过比此人还要悲剧的，如果帮他取出雷灵珠，想必后者会非常乐意吧。
<br/>　　土灵珠与雷灵珠，其次就是火灵珠了，对付火鬼王，慕清霄有先天的优势，他拥有神凤血脉，最不惧怕的就是火焰。
<br/>　　最后就是水灵珠与风灵珠了。
<br/>　　风灵珠暂且不用考虑，在神界夕瑶手中，没有绝对的实力，根本不可能接触。
<br/>　　至于水灵珠，将前面三颗灵珠拿到手后再做考虑。
