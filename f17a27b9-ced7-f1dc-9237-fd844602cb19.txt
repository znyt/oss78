　　广场中还有很多少女，都泛着兴奋的美眸看着涂小安走上去，他神情自若，嘴角噙着淡淡的微笑，容貌俊逸，看的少女们俏脸微红，芳心萌动。
<br/>　　要是有这样一个男朋友守护，在古学院的日子就好混了。
<br/>　　走到中年教官的面前，涂小安一个洞察丢过去，教官是二十五级的实力，算是二阶五段吧。
<br/>　　三阶要渡劫，并不是随处可抓，二阶普遍算是高手。
<br/>　　当当教官也是有资格的。
<br/>　　将录取通知函交给他，教官古波不平的脸上，竟然浮现出淡淡的笑意，虽然他之前被涂小安跟白君然的战斗殃及池鱼了，但他没记仇，因为他心里有数，这种学员是得罪不起的。
<br/>　　实力比他还强。
<br/>　　其实超等天才班的学员，普遍实力都比一般的教官要强，这是公认的事实。
<br/>　　他柔声的对涂小安说道：“去吧，去测一下实力”
<br/>　　涂小安没有动，而是问：“怎样才能有进入超等天才班的资格”
<br/>　　这种话，要是换了别的新生说，一定会被大家认为是哗众取宠，超等天才班里面不是一等一的天才，不是二阶以上的高手，想都不要想。
<br/>　　里面都是各方势力家族中的天才人物，这种班级是你想进就能进？
<br/>　　但涂小安此时问，没人取笑，甚至很多人觉得他是有这个资格进入超等天才班的，原因很简单，他跟白君然打成了平手。
<br/>　　虽然这种打不过是热热身，都没认真出手，但也足可以证明很多的问题。
<br/>　　白君然是谁，白家少主，超等天才班排名前五的高手。
<br/>　　你在他面前撑着一招，都可以出去吹一年的牛。
<br/>　　“你要是真的有潜力，够资格，晶石会给你答案的”中年教官笑着开口。
<br/>　　涂小安懂了，冷淡无波的看了白色晶石一眼，将手放了上去。
<br/>　　同一时间，所有人瞪到大眼睛看着，看着晶石会显示出什么。
<br/>　　安静，现场反而一时之间鸦雀无声，变的没有一点声音出现，静悄悄。
<br/>　　当手放上去的时候，涂小安诧异的发现，晶石中有一股强大的吸力，好像在吸自己的力量。
<br/>　　这就是测量实力。
<br/>　　根据你的元力强度，还划分你的实力。
<br/>　　但涂小安是条蛇精，而且这也不是他的本体，他不知道这晶石最终会测量出什么来。
<br/>　　为了安全无患，涂小安噙着坏坏的笑，体内三种元素属性的力量一口气灌输到了白色晶石的里面。
<br/>　　你想吸，我就让你吸个够，就怕你撑死。
<br/>　　“你们看，白色晶石好像动了”
<br/>　　场外有眼尖的新生发现了不寻常之处。
<br/>　　“能让晶石动，说明测量之人的实力是非常的强”
<br/>　　“曾经我听说有一届的新生测量实力，让晶石足足跳起一米高才落下来，后来那人进了超等天才班”
<br/>　　“哈哈，超级新生就是出手不凡，晶石现在分明在颤抖了，而且看样子越来越厉害”
<br/>　　中年教官见到，欣慰的点了点道：“恩，不错”
<br/>　　“咦...我怎么感觉地面也跟着在震动呀”
<br/>　　广场上的新生们一个个摇摇晃晃起来，在看白色晶石，已经在剧烈的摇晃起来。
<br/>　　涂小安肆意的让自己体内的属性力量灌输进去，他到要看看，这晶石怎么测量自己的实力。
<br/>　　这个过程既然足足五分钟。
<br/>　　众人期待晶石会蹦跳起来，可最后这一幕并没有发生，反而...
<br/>　　澎...
<br/>　　一道爆炸声惊天而起。
<br/>　　须根之间，现场的人没有动一分，宛如石化，呆若木鸡。
<br/>　　这尼玛什么情况，白色晶石爆了？
<br/>　　没错，是爆了，成了稀碎的晶块，散落一地。
<br/>　　旁边的中年教官笑容凝固了，彻底凝固了，慢慢的变成了铁青。
<br/>　　这白色晶石可是古学院的宝物呀，蕴含神奇的力量，可是无价之宝，现在既然在自己的面前爆了，他甚至感觉天空都在塌陷了下来。
<br/>　　这个责任谁来负？他反正貌似付不起。
<br/>　　“教官，晶石都爆了，这是不是间接说明我有资格进入超等天才班”
<br/>　　淡然的声音从涂小安的嘴中说出，对于晶石的爆，毫无别的情绪。
<br/>　　“我...”中年教官张了张嘴，不知道说什么，因为他现在别的话都不想说，只想爆粗口。
<br/>　　我累个草！
<br/>　　“哈哈...厉害...牛逼...大神，我崇拜你”
<br/>　　猛然间，现场响起了震耳欲聋的呐喊声，一个个新生莫名的激动了起来，像一壶刚烧开的沸腾的水一样，激动得要溢出来。
<br/>　　天哪，白色晶石都爆了，这说明什么，说明他的实力太强，强的让晶石都承受不住。
<br/>　　不可想象呀，这到底是怎样的实力，他还是一个刚报道的新生。
<br/>　　这就好比一个大胖子站在体重器上称重，然后体重器直接爆了。
<br/>　　“这一届最强的新生诞生了，舍他其谁，无人能超越”
<br/>　　不知是谁在人群中说了这么一句话，不是超级新生，而是史上最强新生。
<br/>　　忽然，一个个你看看我，我看看你，都没有人反驳，甚至有许多人激动的符合。
<br/>　　他们都明白了，眼前这个清秀男子的名字将会传遍整个古学院，晶石爆炸，历届以来，从未有过。
<br/>　　他有绝对资格进入超等天才班，同时也代表着会有一人会从超等天才班级中滚蛋。
<br/>　　当然，前提是涂小安必须打败他。
<br/>　　但这妥妥的没问题，没人会怀疑，晶石都爆了，他的实力完全牵制了大家的想象。
<br/>　　“不..这届还有一个最弱新生诞生，打破了我们古学院不招普通人的规矩”
<br/>　　悄然间，不少人也将目光投放到了东方袁这个胖子身上，想不通，他到底是怎么拿到古学院的录取通知书。
<br/>　　今天新生报道，最强新生，跟最弱新生同时前后诞生，而且他们还是一起来的。
<br/>　　这很让人觉得不可理解的组合，强弱两人组。
<br/>　　不是说物价类聚，人以群分吗。
<br/>　　这两个人怎么凑到一块去的。
<br/>　　突然，有不少人一下子醒悟了另一件事情，晶石爆了，那么他们这些还没测量实力的人怎么办。
<br/>　　实力没测量，就没办法分班级。
<br/>　　这貌似是一个很严重的问题。
