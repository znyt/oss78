　　幽暗的囚室之中。
<br/>　　此时与以往有所不同，以往只有一个刑架，现在则是有七、八个之多。
<br/>　　七、八个刑架之上，绑缚着巫玛、巫幽兰等一众巫族天才。
<br/>　　“欢迎你们来到这里！”苏莫站在众人前方，一脸的冷漠。
<br/>　　“这是哪里？”
<br/>　　“怎么回事？”
<br/>　　“我们为什么会在这里？”
<br/>　　一众巫族天才发现自身的情况，个个大惊失色，纷纷惊呼了起来。
<br/>　　“梦魔空间？怎么会是梦魔空间？”巫玛看清此地的情况，顿时面带不可思议之色。
<br/>　　他曾经领教过巫魔梦魔空间，所以，一眼便认出了此地，正是巫魔的梦魔空间。
<br/>　　难道眼前的青年，是巫魔？
<br/>　　这不可能，气息完全不同，而且巫魔据说已经死了！
<br/>　　但是，天下间难道还有第二个人，拥有梦魔战魂。
<br/>　　“嗯？”苏莫闻言双眉微挑，没想到巫玛居然认出了梦魔空间，不过这也无妨。
<br/>　　“你们既然来了，就全部死吧！”苏莫冷冷的说道，言罢，他直接催动了吞噬战魂。
<br/>　　霎时之间，强大的吞噬之力，横扫整片梦魔空间。
<br/>　　“大家联手，破了梦魔空间！”
<br/>　　巫玛放声大喝了起来，而后，众人个个将精神力量催动到了极致，想要挣脱梦魔空间的束缚，
<br/>　　众人都是强者，合力之下，精神力量强大到极致，瞬间便让的梦魔空间疯狂震动，开始了崩溃。
<br/>　　“这么强？”苏莫心中一惊，但他丝毫没有停止催动吞噬战魂。
<br/>　　轰隆隆！
<br/>　　在梦魔空间崩溃的刹那，七八名巫族天才之中，除了巫玛和巫幽兰两人，其他人全部消散，精神意识被吞噬战魂直接吞噬了。
<br/>　　梦魔空间一崩溃，巫玛和巫幽兰瞬间掌控了身体，而后身形急速暴退，瞬间和苏莫拉开了距离。
<br/>　　而另外几名巫族天才，则是精神意识被吞噬，一个个呆立当场，成为了行尸走肉。
<br/>　　苏莫收回梦魔战魂，而后一挥手，瞬间便将几人收进了储物戒之中。
<br/>　　“大胆，敢在天冥星杀我族人，你简直找死！”巫玛暴喝出声，冷漠的脸庞上露出森然的杀机。
<br/>　　他非常的愤怒，从来还没有其他种族敢在天冥星撒野，而眼前的人，居然瞬间便杀了他几名族人。
<br/>　　巫幽兰同样是脸色冰寒，眸中杀意涌动。
<br/>　　“巫玛、巫幽兰，今日便是你们的死期！”苏莫的身形缓缓升腾而起，手中出现了一柄长剑，体内五枚神丹缓缓旋转，磅礴浩瀚的玄力在体内涌动。
<br/>　　“你认识我们？你到底是谁？”巫幽兰娇斥道。
<br/>　　“杀！”
<br/>　　回答她的是一声冰冷的厉喝，只见苏莫手臂一挥，瞬间连斩两剑，两道璀璨夺目的三色剑气撕裂长空，向巫幽兰和巫玛两人袭杀而去。
<br/>　　这两道剑气极度强大，因苏莫修为的提高，两道剑气中蕴含的力量，如同火山沸腾，威势无比磅礴。
<br/>　　与此同时，苏莫立刻释放出了吞噬战魂，浩大的吞噬旋涡漂浮在他的身后，强大的吞噬之力瞬间笼罩八方。
<br/>　　巫玛和巫幽兰两人，遭到吞噬之力的笼罩，身形微震，不仅体内气血和玄力一阵絮乱，还感觉自己仿佛身形泥潭之中，身形不受控制的向苏莫的剑气飞了过去。
<br/>　　“这是什么战魂？”
<br/>　　“好诡异！”
<br/>　　两人心中大惊，急忙压制体内絮乱的玄力，勉强稳住身形，纷纷出手攻击。
<br/>　　“灭！”
<br/>　　两人几乎同时出手，只见巫玛大手一张，瞬间击出了一只宫殿般大小的掌印。
<br/>　　掌印威势滔天，强大的玄力波动让得空间粉碎，大地崩裂，发出惊天动地的巨响。
<br/>　　而巫幽兰同样不弱，其双手结印，而后向前一扬，立刻幻化出一只长达百丈的白色神鸟。
<br/>　　啾！
<br/>　　神鸟口中发出一声高亢的鸣叫，双翅一扇，如同一道白色的闪电，急速冲向迎面而来的剑气。
<br/>　　轰！轰！
<br/>　　掌印和神鸟，纷纷与剑气相击在了一起，发出震天的爆响。
<br/>　　而后，一切尽皆毁灭，猛烈的爆炸震荡四野，滔天的气劲横扫八方，逆卷九天。
<br/>　　“怎么会如此强大？”巫玛和巫幽兰脸色凝重了起来，一个武帝境四重的武者，居然有不弱于他们的实力，简直不可思议！
<br/>　　两人并没有认出苏莫的身份，他们虽然听说过苏莫之名，但对于苏莫完全不了解。
<br/>　　他们近几年时间，一直在外历练，也就是少司选拔之前，才回到天冥星。
<br/>　　苏莫的脸色亦是稍稍凝重了起来，一击试探，他便知道这两人实力有多强。
<br/>　　不过，在他的吞噬之力下，两人只能勉强稳住身形，绝对不可能逃走。
<br/>　　嗡嗡嗡~~
<br/>　　苏莫手中长剑嗡嗡震动，剑芒缓缓增长，磅礴的三色玄力不断的涌进长剑之中，璀璨夺目。
<br/>　　“无论你是谁？无论你有什么目的？你杀我族人，今日必须要死！”巫玛和巫幽兰对视一眼，随即冷视着苏莫，眸中强大的战意升腾而起。
<br/>　　“你们两人都是巫族顶尖天才，希望你们不会让我失望！”苏莫淡漠的说道。
<br/>　　“死人不会失望！”巫玛身上杀意冲天，全身玄力滚滚，气势无限拔高，磅礴的气势冲天而起，直达云霄。
<br/>　　而巫幽兰身上，则是缓缓变色，全身皮肤都变得晶莹剔透，闪烁着宝玉一般的光泽。
<br/>　　瞬息之间，她整个人完全改变，仿佛变成了一具流光溢彩的玉石。
<br/>　　巫玛和巫幽兰丝毫不敢小觑苏莫，因为两人在吞噬之力下，实力受到了极大的削弱。
<br/>　　他们不仅要费力抵挡吞噬之力的吸扯，自身的战魂威力，亦是受到了很大的压制。
<br/>　　两人能清晰的感觉到，自身的战魂在体内不断的颤抖，仿佛是遇到了天敌一般。
<br/>　　这让两人心中震惊不已，甚至都不敢释放出自己的战魂。
<br/>　　“琉璃宝体！”苏莫望着巫幽兰，眼眸微眯了下来，这种体质虽然没有天玄武体那般强大，但也非常不凡，能增加武者战力的同时，还能赋予武者超强的防御力。
<br/>　　“杀！”巫玛怒喝一声，率先出手了，只见其手掌高高举起，向苏莫猛然一拍。
