　　萧逸就这般释放着星光力量。
<br/>　　体内元力，已悉数调动。
<br/>　　星光力量，进入顾怜星体内后，几乎是一瞬间便消散无踪。
<br/>　　也不见顾怜星有半分的修为增加，更无半分的气息增强。
<br/>　　唯一有的，只是那属于生机的微弱气息，逐渐恢复。
<br/>　　这不得不让人惊疑，如此庞大的星光力量，凭空去了哪？
<br/>　　但事实上，萧逸很清楚这些星光力量去了哪。
<br/>　　整整半个时辰后。
<br/>　　“呼。”萧逸轻呼出一口气。
<br/>　　擦了擦额头上的汗水，停下了元力的转换。
<br/>　　这半个时辰，他已是全力爆发元力，将星光力量催发到极致。
<br/>　　而此时，顾怜星脸上的苍白，已然褪去。
<br/>　　虽仍旧柔弱，可已无生命之虞。
<br/>　　“嗯？”顾怜星缓缓地睁开了眼睛，仍旧是一副云淡风轻的表情。
<br/>　　“萧逸公子不是走了吗？”顾怜星疑惑问道。
<br/>　　“忽然改变主意了。”萧逸淡漠地说了一声。
<br/>　　事实上，他已经能确定顾怜星和那个家伙的关系了。
<br/>　　来十八府历练的这些天，他早已知晓各天骄的名字，还有一些有名的事迹等等。
<br/>　　十八府天骄中，排在最后，而且姓叶的，也就只有那个家伙。
<br/>　　不过萧逸未明说，只是淡淡道，“想起来，我是个炼药师。”
<br/>　　“怜星姑娘的这病，倒是奇特，也颇有难度。”
<br/>　　“权当在下挑战一下难度吧。”
<br/>　　“哦？”顾怜星淡淡一笑，随后反应过来，脸色陡然一惊。
<br/>　　“好强的星光效果。”顾怜星看了看自己的双手，语气惊讶。
<br/>　　原本，她那白泽的双手中，毫无血色。
<br/>　　可现在，却明显带着一丝气血充盈的红润。
<br/>　　顾怜星站了起来，闭目感知了一番，当睁开眼睛时，脸上惊色，已无以复加。
<br/>　　她刚才在感知自己的身体。
<br/>　　一番感知下，竟是气息平稳，毫无半分不适之感。
<br/>　　要知道，半个时辰前，她那虚弱的身体，还如风中残烛，脆弱不堪。
<br/>　　“萧逸公子所修的武道，必定是极强的星光一道吧。”顾怜星美眸中带着惊讶，以及肯定。
<br/>　　“即便是我爹亲自为我灌输星光力量，都不曾有这般效果。”
<br/>　　“你爹？”萧逸挑了挑眉头。
<br/>　　顾怜星的父亲，自然是天星府府主。
<br/>　　“天星府主修的是什么武道？”萧逸问道。
<br/>　　“冷星一道。”顾怜星回答道。
<br/>　　“原来是冷星一道。”萧逸点了点头。
<br/>　　冷星一道，同样是属于星光一道中的武道。
<br/>　　不过，冷星一道，只是一般的顶尖武道。
<br/>　　其本体，乃是星辰所散发的冷异光芒。
<br/>　　而星幻剑道，乃是顶尖武道中位于巅峰的层次。
<br/>　　由星幻一道，与剑道结合而成。
<br/>　　星幻一道的本体，乃是隐匿于星空之中的神秘幻火，位列世间强悍火焰。
<br/>　　自然的，冷星一道，层次上远不如星幻剑道。
<br/>　　以前，在东域时，他所知晓的武道最高层次，便是顶尖武道了。
<br/>　　但在中域，那些在顶尖武道中，位于巅峰层次，凌驾其它一般顶尖武道的武道，还有一个称谓——至强武道。
<br/>　　无论是他的霸道剑道、寒冰剑道，抑或其他各道强悍火焰的武道，尽皆可踏入至强武道的范畴。
<br/>　　这时，公孙火舞看着顾怜星无恙，擦了擦脸上泪水，喜悦道，“怜星，你没事就太好了。”
<br/>　　“没事？”萧逸瞥了眼公孙火舞，随后看回顾怜星。
<br/>　　“我想，怜星姑娘应该知晓自己的情况吧。”
<br/>　　“嗯。”顾怜星点了点头，轻笑一声，并无什么不妥。
<br/>　　公孙火舞眉头一皱，她总觉得，顾怜星的轻笑，带着苦涩，而萧逸的语气，则带着凝重。
<br/>　　“你们在说些什么？”
<br/>　　公孙火舞紧张地看着顾怜星。
<br/>　　顾怜星不语。
<br/>　　公孙火舞看向萧逸，“萧逸小贼，不，萧逸公子，你告诉我。”
<br/>　　“说吗？”萧逸看向顾怜星，淡漠的语气中，带着一丝询问。
<br/>　　这是顾怜星的事，是否要说，自然取决于顾怜星是否在意。
<br/>　　顾怜星笑笑，“随萧逸公子吧。”
<br/>　　公孙火舞，认真地看着萧逸。
<br/>　　萧逸脸色淡漠，轻声道，“我的星光力量，只是在为怜星姑娘续命罢了。”
<br/>　　“续命？”公孙火舞身躯一颤，“你是说怜星她还会…”
<br/>　　她的话，没有说下去，但意思却很清楚。
<br/>　　顾怜星现今看着无碍，实则要不了多久，就会再次如之前那般，虚弱至极，香消玉殒。
<br/>　　“你…你…”公孙火舞颤着声，“你实力如此强，炼药师手段亦是不俗。”
<br/>　　“你所修的星光武道，又效果惊人；你一定有办法救怜星的对不对？”
<br/>　　公孙火舞急切下，扯住了萧逸的衣衫。
<br/>　　萧逸皱了皱眉，淡漠道，“我之前便说了，我只能救一时。”
<br/>　　“倒是怜星姑娘。”萧逸看向顾怜星，“能否告诉我为何你体内会出现这等情况？”
<br/>　　“若是我知晓缘由，说不定能想到办法，当然，也只是可能。”
<br/>　　“哦？”顾怜星脸上，再次露出惊色，“萧逸公子知晓？”
<br/>　　萧逸点了点头，“若非你出了那等变故，此刻的怜星姑娘，怕是早已成中域有数的绝世妖孽之一了吧。”
<br/>　　“身怀天星血脉，外加觉醒深紫色武魂紫耀星辰，一身天赋，以绝世天资四字来形容也毫不为过。”
<br/>　　“只可惜，同为星光属性的血脉和武魂，理应相辅相成，效果惊人，现今却是血脉和武魂互相冲突，危在旦夕。”
<br/>　　不错，原本顾怜星的血脉和武魂，皆是绝世。
<br/>　　随便一样，都足以让顾怜星成为当世一等一的妖孽。
<br/>　　而她的血脉和武魂，又同为星光属性，理应珠联璧合，让得她天赋恐怖。
<br/>　　可现在，她体内的血脉和武魂，竟在互相冲突，甚至达到水火难容的地步。
<br/>　　大好天赋，就此尽废，连性命都危在旦夕。
<br/>　　但这一点，是萧逸怎么都想不明白的。
<br/>　　正常情况下，她的血脉和武魂，绝不会有此冲突。
<br/>　　“不愧是萧逸公子。”顾怜星惊讶地看着萧逸。
<br/>　　以前，即便是那些中域有名的炼药强者，都需费好些手段才知晓她身体虚弱的原因。
<br/>　　可现在，萧逸竟是一番查探便轻松知晓个中冲突。
<br/>　　“不过。”顾怜星微微蹙眉，“为何会产生冲突，我也不知。”
<br/>　　“不知？”萧逸眉头一皱。
