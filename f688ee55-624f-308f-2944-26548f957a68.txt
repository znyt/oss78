　　车开了一个多小时，总算到了风景区，打车费一共两百多，给叶少阳心疼坏了。
<br/>　　因为是冬季，风景区人并不多，只有三三两两的香客在山路上走着，两人跟着一路上山，仔细辨认着。
<br/>　　“感觉跟千年前一点都不一样了。”叶少阳感慨起来，随后感到自己这句话有点怪，好像自己活了一千岁似的。
<br/>　　国清寺就在山顶，显然是重修过的，跟千年前那一座压根不一样了，气派了很多，但也失去了古韵。
<br/>　　叶少阳假装是香客，走了进去，一直逛到大雄宝殿，还是那三尊佛像，但模样跟昨天看的有点不一样，叶少阳正在仔细辨认，道风扯了他一把，转身出去，叶少阳赶紧跟上，来到左边的一个院门前，抬头一看，院门上挂着一块牌匾，写着“天台木雕佛像文物展览馆”。
<br/>　　走进去看，里面一排有三个长厅，两人从最左边的进去，厅的一大半用红线隔断，里面摆着若干佛像，大的小的都有，全是木雕，有佛有菩萨有罗汉。每一个雕像前面都竖着牌子，写着雕像的名字和年代，下面有创作者的姓名和简介。
<br/>　　一个工作人员走上来，对叶少阳介绍这些木雕。叶少阳向来对艺术方面一窍不通，又不好赶人走，就一知半解地听着，一转头发现道风站在一旁，在看贴在墙上的一副宣传字画。
<br/>　　“你……”叶少阳扫了一眼讲解员，为了避免吓到他，将想对道风说的话咽了回去，也装模作样地走过去看。
<br/>　　是有关什么天台木雕的介绍。
<br/>　　“这有啥好看的？”叶少阳低声问道风，结果道风没回答，反倒把讲解员引来了，跟叶少阳介绍道：“我们天台木雕，是一种古老的技艺，尤其是佛像木雕，您看这些佛像，线条圆润，色泽饱满，显得大气，这是我们南方佛像雕刻的特色……”
<br/>　　道风指着墙上一段话，道：“问她，这手艺失传了没有。”
<br/>　　叶少阳看了一下他指的那段话，写的是有关“干漆夹苎”的一段介绍，不明白啥意思，于是问了讲解员，讲解员表示这手艺还有传承人，然后就这门手艺介绍起来。
<br/>　　叶少阳本来不想听，但听讲解员说道，这本制造佛像的工艺，是先用泥胎塑形，然后用特制的麻木蘸上干漆——当地的一种树胶加工而成，刷在特制的麻布上，绕着塑好的形状一层一层裹起来，最终形成一尊姿态饱满的佛像，再用颜料描绘出五官等等，然后再把泥胎取出，这样做出来的佛像内部是中空的，能够长期保存，历经千年也不会毁坏。
<br/>　　叶少阳耐着性子听完这一段，突然想到之前穿越看到那个浓眉和尚，在药师佛的身上撕下的那一条一条的，当时觉得很好奇，莫非就是这什么“干漆夹苎”的工艺？
<br/>　　就算是这样，那跟他们前来找太阿剑又有什么关系？
<br/>　　道风径直往相邻的展厅走去，叶少阳跟着，两人走过中间展厅，直接来到最后一个，道风指了指红线后面的玻璃橱中的一尊佛像，叶少阳看过去，立刻认出来，这就是那尊药师佛！
<br/>　　可真是隔了快一千年啊，这佛像除了外层有点风化之外，其余还是原来的样子。
<br/>　　找到它，叶少阳也稍微放心了一些。
<br/>　　讲解员妹子也跟过来，见他们观察着药师佛，称赞他们好眼力，说这一尊就是用“干漆夹苎”工艺制作出来的佛像，是元朝时期，一位著名的工匠——同时也是天台宗高僧制作而成，之前一直都摆放在国清寺大雄宝殿中间，是镇山之宝。
<br/>　　十年浩劫时期，寺院僧人怕佛像被毁，就把这些佛像一股脑藏了起来，后来经过申报，这座药师佛成了重点文物，本来是要送去京城的，当地极力挽留下来，现今放在这展览馆里，给游客鉴赏。
<br/>　　“这一尊佛像……得不少钱吧？”叶少阳问。
<br/>　　“文物是禁止交易的，是无价之宝，不过之前有专家鉴定，这尊药师佛如果放到市场上，至少价值数千万……”
<br/>　　数千万……
<br/>　　叶少阳瞠目结舌。
<br/>　　讲解员妹子笑道：“文物是不能交易的，这位先生如果有兴趣，后面展厅里还有很多现代制作的天台佛雕，迷你大小，可以放在家里或者汽车里祈福辟邪……”
<br/>　　说着递过来一本精美的画册。
<br/>　　怪不得这么热情！敢情这才是真实目的！
<br/>　　叶少阳只好假装上厕所，在讲解员妹子鄙视的目光下逃之夭夭。
<br/>　　“数千万啊，我勒个去，万一被抓住，这得坐好几年牢吧？”
<br/>　　山门外，叶少阳忧心忡忡地跟道风算计起来。
<br/>　　“你看到渡月法师的手法没有，干漆夹苎，太阿剑藏进去，没一点痕迹。”
<br/>　　“渡月？你说那个浓眉的僧人？”
<br/>　　“我昨天在这里呆了一天，找了他们历代法师的名录，找到了此人，是个尊者级别的强者，也是这佛雕工艺的传承人。”
<br/>　　“哦……喂，我跟你说正经的呢——”
<br/>　　“我也在说正经的，我之前一直纳闷，太阿剑是九段光法器，仓促之间藏在佛像上，如何能隐藏气息，就算用定气符，能躲过一时，灵气耗尽，只要冥河老祖在寺庙蹲点，早晚也能发现，原来他是利用了揭开的麻布，在上面用金漆涂了符文，一层层贴上封好，这就成了永久封印，只要佛像不被毁，永远不会有灵气泄露。”
<br/>　　叶少阳很惊讶道风居然一口气说了这么多个字，怔怔了看了他一会，问道：“就算是这样，跟我们来偷东西有什么关系？”
<br/>　　“没关系。”
<br/>　　“啥？”
<br/>　　“有些事情，既然怀疑了就要搞清楚，万一其中隐藏着关系成败的细节呢。”道风笑了笑，“这是态度问题。”
<br/>　　叶少阳仔细一想，觉得他说得也有道理。
<br/>　　“那咱们怎么办，那是文物，怎么动手？”
<br/>　　“晚上再说。”
<br/>　　两人在山上转了一圈，找到国清寺的厢房，看到几个和尚在走动，有些身上有灵光隐现，表示有法力在身。
<br/>(本章完)
