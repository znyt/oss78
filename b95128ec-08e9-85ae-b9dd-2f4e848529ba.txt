　　给自己留下二百一十点火能，慕清霄将火晶卡递了过去，道：“仙儿，你比我更需要火能，乖乖拿着。”
<br/>　　手中有两百十一点火能，已经完全够用，慕清霄进天焚炼气塔并不是修炼，而是寻找陨落心炎。
<br/>　　只要火能够支撑他寻找异火的时间，不引起怀疑，多少都无所谓。
<br/>　　闻言，小医仙芳心甜甜的，乖巧的点了点头，美眸注视着眼前的青年，只有他，无时无刻在为自己着想。
<br/>　　也只有他才知道，厄难毒体对于自己来说，是多么的无奈。
<br/>　　“在过一刻钟，麻醉散的药效就会过去，你们就坐在这里休息一会吧。”
<br/>　　将火晶卡收了起来，慕清霄转过身来，朝着老生说道，说完，便带着小医仙朝着内院方向走去。
<br/>　　望着两人消失在视线中，领头老生叹了口气。
<br/>　　“哎，回到内院后，我们抓紧做任务，赚火能吧，争取在明年，把今年失去的全都抢回来。”
<br/>　　陡然，慕清霄脚步微微一顿，目光隐晦的朝西边方向看了一眼，他明显察觉到，有两道视线正盯着自己。
<br/>　　整个迦南学院，拥有这种实力的人并不多，而刚才的两位内院长老，正好在这位数不多的人当中。
<br/>　　小医仙小脑袋歪了歪，芊芊玉手拽了拽他的衣袖，不明白慕清霄为何突然就停下了。
<br/>　　收回目光，慕清霄道：“没事，我在想，火能也掠夺的差不多，剩下的估计也不多了，我们立刻赶往内院吧。”
<br/>　　“嗯，你去哪，我就去哪。”
<br/>　　西边方向，盘坐在巨树顶端，苏长老睁开双眼，眼底满是困惑。
<br/>　　“庆老头，你有没有发现，刚才在一瞬间，隐约有一道视线在窥视我们？而且......”
<br/>　　“窥视我们？你还没睡醒吧，院长不再，整个学院里，就只有三位拥有这种实力，他们哪会做这种无聊的事情，是你的错觉吧？”
<br/>　　苏长老若有所思的点了点头：“难道真是我想多了？”
<br/>　　......
<br/>　　......
<br/>　　森林地面上，遍布枯叶，仿佛看不见尽头。
<br/>　　两道身影，闲庭信步的走在其中，一缕光线透过枝叶缝隙照射而下，如梦如幻。
<br/>　　在森林中行走了半个小时左右，两人的速度不快，却也不慢，总算是按照地图上的标记，来到此处。
<br/>　　目光在四周扫视，最后落在数十米开外，一颗有两人合抱粗壮的巨树上。
<br/>　　其树枝上，绑着一块红色布片，布片经历风吹雨打，显得有些残破，一阵微风迎来，给人一种摇摇欲坠的感觉。
<br/>　　红色布片上，依稀能看到一个属于迦南学院的模糊标志。
<br/>　　慕清霄将地图收起，道：“仙儿，看来是这条路没错，在走一段路程，就到学院与森林的交界处了。”
<br/>　　“前面就是内院吗？真的有封印厄难毒体的方法吗？”小医仙美眸闪烁了一下，忍不住出声问道。
<br/>　　闻言，慕清霄伸手将她揽入怀中，轻轻抚摸着她的秀发，语气温柔：“我说有就有，走吧，前面就快到了。”
<br/>　　约莫在森林中走了十分钟，前面，森林的尽头，光线终于明亮了起来。
<br/>　　来到森林的尽头，放眼望去，映入眼中的是一处布满乱石的空旷地带，森林的蔓延，在这里停止。
<br/>　　天空上挥洒而下的阳光，直接倾洒在地面上，使得周围多了一分温暖，少了一分森林中的清冷昏暗。
<br/>　　乱石地带不远处，是一处隆起的山坡，山坡上各处都散步着一些形状各异的山岩。
<br/>　　在山岩的中央，竖立着一座高台，两名老者坐于其上，正是内院的苏长老与庆长老。
<br/>　　在两位长老的周围，还坐着几名气息沉稳的中年人，气息上来看，尽是清一色的斗王级别，估计都是属于内院的管事者。
<br/>　　见两人出现，高台上的身影也都睁开双眼，几道视线接连投射过来。
<br/>　　随后，高台最中央，犹如老僧入定一样的两位内院长老，也都睁开了双眼。
<br/>　　站起身来，苏长老目光看向慕清霄与小医仙，沉默片刻，清晰的声音，在整个森林中响起。
<br/>　　“我宣布，这一届火能猎捕赛结束，老生与新生尽快赶到指定地点......”
<br/>　　苏长老略有些苍老的声音，在森林上空回荡，久久不散，片刻后，森林中陆陆续续的走出一支支队伍。
<br/>　　其中，有新生队伍也有老生队伍，老生眼中都挂满了疲惫，特别是看向慕清霄与小医仙的目光，充满了无奈。
<br/>　　至于新生，大多鼻青脸肿，甚至有的衣衫不整，破破烂烂，整体上来说，伤势并不是很严重，可见老生下手也是有分寸的。
<br/>　　只不过，有的新生们，貌似并不领情，目光愤恨的盯着老生，恨不得一口将他们吞了。
<br/>　　老生们都不太在意新生的目光，他们自己都郁闷的要死，好不容易抢来的火能，到头来却全部便宜了别人。
<br/>　　而且，对方还是新生，可他们却毫无还手之力。
<br/>　　慕清霄与小医仙站在一旁，耳鬓厮磨的窃窃私语，亲密无间，突如其来的狗粮，使得老生与新生更是郁闷至极。
<br/>　　半个小时后，老生与新生的及时赶来，就在这时，森林中走出一支队伍，吸引了乱石地带的所有目光。
<br/>　　这支队伍是由五名新生组成，各各鼻青脸肿，衣衫凌乱，不堪入目，几乎与进入森林前，意气风发的模样截然相反。
<br/>　　走起路来都一瘸一拐，屁.股处都有些走光。
<br/>　　甚至隐隐能看到血迹，倒吸着凉气，走出森林后，目光第一时间在老生队伍中扫视，眼中充满了恨意。
<br/>　　“他...他好像是罗成？......卧槽，真的罗成，新生前五的罗成？”
<br/>　　“肯定是罗成没错，后面那个是他的弟弟，新生前五的罗超，怎么都被打成这样，下手的是哪只队伍，真是好狠啊！”
<br/>　　见着他们不堪入目模样，许多新生面面相觑，不知为何，心里居然平衡的多，而且看他们眼中的恨意，似乎都受到了非人的虐待。
<br/>　　老生们也都面面相觑，其中一支老生队伍脸色都有些不自然。
<br/>　　当然，想到猎捕赛规则后，脸上的不自然也消失不见，反而心中充满了快意，谁让这支新生队伍坑他们来着！
