　　“那到底是什么秘密？”
<br/>　　叶尘听到这里，已经有些不耐，这家伙絮絮叨叨说了半天，总算才说到正题上。
<br/>　　灭世仙君并没有着急开口，而是指着叶尘手中的灭世魔刀，缓缓道：
<br/>　　“你应该也能看出来，这是一把残缺的刀，更确切的说，现在你手里的这把刀，只是其中的一半，是代表毁灭的那一部分，所以我称之为灭刀，至于另外一半，则代表着生机，可以称之为生刀！”
<br/>　　“而当两者合为一体，才是这件混沌灵宝真正的本来面目，生灭刀！”
<br/>　　“生灭刀？”
<br/>　　叶尘顿时微微一愣，倘若真如灭世仙君所说，眼前的这把灭世魔刀，仅仅只是其中一半，就已经有如此威力，那这把刀的完全体，又该是何等逆天的存在？
<br/>　　灭世仙君点了点头，道：
<br/>　　“云苍生之所以处心积虑的接近我，就是因为，他从一开始，就知道这生灭刀的秘密！”
<br/>　　“无极宗，乃是八大仙门之中，历史最悠久的一个门派，据说从天地诞生之时，就已经存在这世间，而在无极门的庞大传承之中，包含着无数远古时期的传说，其中便有关于生灭刀的传说！”
<br/>　　“当年，云苍生利用我手中的灭刀，耗费了数百年的时间，走遍了整个修真界，终于找到了另外一半生刀的下落！”
<br/>　　“之后，云苍生对我突下杀手，自以为已经掌控一切，得意忘形之下，便将他们宗门传承中，关于生灭刀的传说讲了出来……”
<br/>　　“据云苍生所言，这把生灭刀，可以说古往今来第一神器！”
<br/>　　“因为，我们所在的这个世界，便是由一位来自混沌世界的上古大能，用生灭刀开辟而来！”
<br/>　　“换句话来说，没有生灭刀，就没有我们眼下的这个世界！”
<br/>　　“什么！”
<br/>　　听到这里，叶尘顿时全身巨震，怎么也没有想到，这把生灭刀的来头，竟然会如此之大！
<br/>　　开天辟地！
<br/>　　那岂不是就相当于地球上，盘古开天地的传说了？
<br/>　　等等！
<br/>　　记得之前吞天蟒老前辈曾经说过，地球所在的那一方宇宙，乃是那位昊天大帝，在混沌中开辟而来，莫非……
<br/>　　恍然之间，叶尘似乎对天地宇宙的生灭，有了更深刻的认识。
<br/>　　灭世仙君似乎早就料到，叶尘会被吓到，顿了顿，接着又道：
<br/>　　“据说，生灭刀开辟出这一方宇宙之后，因为耗损太过严重，所以从中断裂，一分为二，化成了如今的生刀和灭刀！”
<br/>　　“而一旦让两刀合体，便能破碎这一方宇宙的束缚，甚至不需要渡过天劫，也能进入到那无尽混沌之中……”
<br/>　　灭世仙君说到这里，眼眸中露出一抹神往之色，不过一想起当年，被自己最信任的朋友暗算，落得如此下场，不由得又露出一抹恨意，
<br/>　　“云苍生偷袭将我打成重伤，本以为一切尽在掌握，却根本不知道，其实我也早就对他起了疑心，暗暗留了一手底牌，侥幸从他的手上逃脱了，之后一路辗转来到此处，拼尽最后一道力量，将灭世魔刀封印在了这里……”
<br/>　　灭世仙君将来龙去脉说了一遍之后，一阵唏嘘不已，
<br/>　　“小子！你既然已经立下本命誓言，本座自然相信，你肯定想尽一切办法，灭掉云苍生和无极宗！”
<br/>　　“不过以你目前的修为，区区百年的时间，根本不可能做到！”
<br/>　　“除非，你能够拿到生刀，将两刀合一，才那么有一丝希望！”
<br/>　　“只可惜，如果不出意料之外的话，云苍生应该已经得到生刀，而且他一定会想尽一切办法，得到你手中的灭刀！”
<br/>　　“而最糟糕的是，今日灭刀一旦冲破封印，生刀必然会有所感应！”
<br/>　　“所以，云苍生必定会想尽一切办法追杀你，你不到万不得已，绝对不能轻易动用灭刀！！”
<br/>　　“这……”
<br/>　　听到这话，叶尘顿时一愣，差点没有爆出一声粗口出来。
<br/>　　他有十足的把握，在百年之内重新回到前世的巅峰状态，所以才敢立下那样的本命誓言，
<br/>　　可现在的他，面对云苍生这样的巨擘，无极宗这样的顶级大势力，他简直就和蝼蚁没有两样啊！
<br/>　　他原本只是想找一件兵器的，没想到如今居然陷入了灭世仙君和云苍生的恩怨纠结之中。
<br/>　　凭他如今的实力，想要和无极宗作对，无异于螳臂当车！
<br/>　　“怎么？后悔了？”
<br/>　　灭世仙君似乎也看出了叶尘的担忧，脸色不由得露出一抹玩味的笑容。
<br/>　　叶尘倒也没有掩饰，点了点头，道：
<br/>　　“这一次我却是太冲动了！不过如今已经到了这个地步，后悔还有用吗？”
<br/>　　灭世仙君两眼微微一眯，露出一抹赞赏之色，
<br/>　　“好！小子，看来你的心志，比本座预想的还要坚韧的多！还真是让人意外呢！或许将来你真的能帮本座报仇雪恨也不一定！”
<br/>　　“也罢，本座的这道残魂留在世上也已经无用，就再帮你最后一次吧！”
<br/>　　说完这话，灭世仙君的残魂忽然化作一道金光，钻入到灭世魔刀之中。
<br/>　　嗡嗡嗡~~
<br/>　　灭世魔刀顿时一阵剧烈颤抖，似乎随时都要从叶尘的手中挣脱出来。
<br/>　　“小子！本座以自己的残魂化作刀灵，寄居在着魔刀之中，帮你抑制魔刀的魔性，免得你将来被魔刀影响，成为杀人不眨眼的大魔头……”
<br/>　　灭世仙君的声音，越来越弱，等到他的残魂彻底融入魔刀之中，便没有了声音，而灭世魔刀也开始变得温顺起来，甚至连魔刀中的毁灭气息，也没有之前那么狂躁了。
<br/>　　叶尘顿时微微动容，当即肃然起敬，冲着魔刀，低声道：
<br/>　　“多谢前辈成全！我必不负你所托！”
<br/>　　说着，叶尘一手握住灭世魔刀，另外一只手，在刀刃上轻轻一划，
<br/>　　滋滋滋~~
<br/>　　随着叶尘的血液，融入魔刀之中，后者顿时发出一道欢快的鸣叫声。
<br/>　　片刻之后，
<br/>　　嗖！
<br/>　　已经成功认主的灭世魔刀，顿时化作一道光芒，钻入到了叶尘的体内。
<br/>　　叶尘心念一动，发现自己丹田之内，已经多了一把迷你型的灭世魔刀。
<br/>　　而随着叶尘将魔刀炼化，他整个人身上，也开始隐隐透着一股毁灭的气息！
<br/>　　“果然！这魔刀的气息太明显了！很容易引起注意，必须暂时封印起来！”
