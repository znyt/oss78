    第九二零章接二连三的消息，炸蒙了水域强者
<br/>
<br/>    其实它们这些强者哪里知道，水神之所以这么做，自然是有自己的想法和目的的，以后或许这些强者会明白，也或许它们只是以为自己明白了。
<br/>
<br/>    对于这一些，水神根本就不在乎，其他的强者怎么看，那是它们自己的事情，水神只需要知道自己这么做的目的是什么就可以了。
<br/>
<br/>    它水神之所以为那么做，那里是水域的强者此刻能够明白的，就算是以为自己明白了，但也只不过是假象罢了，就连苍梧殿的苍梧之主，都不会想到。
<br/>
<br/>    这个时候水神这么做，看似是对于苍梧殿收留螺妖一脉不满，觉得这是苍梧殿故意打自己的脸，收留了自己想要灭掉的螺妖一脉强者。
<br/>
<br/>    但是实际上，只不过是为了让苍梧之主，还有水域其他所有不之情的强者这么认为罢了，它水神的真正目的，又岂是其他强者可以明白的。
<br/>
<br/>    水神此举，看似是表达自己的态度，表示自己的不满，但实际上呢，却是为了让苍梧殿的苍梧之主，还有苍梧殿的强者们，更加的相信螺妖一脉的强者。
<br/>
<br/>    因为水神和螺妖一脉的那些最顶尖的强者，早已经暗中布置好了一个计划，现在这一切，不过是按照水神预计的轨迹进行罢了。
<br/>
<br/>    水域的众多强者都知道，水神在很长一段时间，对于螺妖一脉都很时亲近，但是也仅仅是知道这些，只不过是觉得，水神仅仅是看重了螺妖一脉可以对人族造成损失。
<br/>
<br/>    可是实际上呢，哪里会有这么简单，水神那么久以来对螺妖一脉的付出，岂是因为一件事情就算了的，何况还是在螺妖一脉强者没错的情况，就要对螺妖一脉赶尽杀绝。
<br/>
<br/>    之前的一切，只不过是为了让水域的强者，还有苍梧之主相信，自己真的跟螺妖一脉闹僵了而已，现在这么做，也是为了更好的让水域的强者知道，自己对螺妖一脉的厌恶。
<br/>
<br/>    因为只有这样，螺妖一脉的强者才可能更加渗透到苍梧殿之中，这本就是水神和螺妖一脉强者计划之中的一环，只是众人都不知道罢了。
<br/>
<br/>    从水神的一系列计划之后造成的效果来看，水域强者的反应正是在水神的预料之中，那么苍梧之主呢，现在必然也差不多是这样的想法了。
<br/>
<br/>    因为苍梧之主知道，现在苍梧殿和水神殿已经联手了，那么就算是苍梧殿在这个时候收留螺妖一脉的强者，水神也不会有什么过激的举动。
<br/>
<br/>    至少现在这个时候，水神不会因为这件事情，就对苍梧殿有做出些什么事情，也不会对螺妖一脉的强者出手，这一点是完全可以放心的。
<br/>
<br/>    此刻水神的这一番举动，让远在苍梧殿的苍梧之主看来，这只不过是水神表达自己不满的一种方式，也只能以这样的方式来表明自己的不满了。
<br/>
<br/>    因为现在这个时候，它水神绝对不会做出什么事情来的，但是又不能没有表示，所以这么做，在苍梧之主看来，也就是理所应当的事情了。
<br/>
<br/>    水神这么做，苍梧之主觉得自己也是明白为什么的，因为在水神殿之中，水神已经告诉了苍梧之主关于抛弃螺妖一脉强者的来龙去脉。
<br/>
<br/>    就算是水神做事情不地道，但是它是水神，是水神殿的执掌者，做出的决定那就是不能更改的，在那个时候，也算是给苍梧之主一个态度了。
<br/>
<br/>    可是苍梧之主知道了以后，依然按照自己的想法去做，根本就没有给它水神这个面子，回去之后就把螺妖一脉收到了苍梧殿之中。
<br/>
<br/>    正是因为如此，所以苍梧之主觉得，此刻水神的做法自己是可以明白的，因为换做是自己，或许也是跟水神同样的做法。
<br/>
<br/>    苍梧之主自以为了解水神的意图，殊不知，这一切都在水神的掌控之中，水神早已经料定了苍梧之主会怎么想，绝对不会起疑心的，而这些正是水神所希望的。
<br/>
<br/>    自己的目的达到了，自己想要埋入苍梧殿之中的钉子，此刻已经算是成功了，那么就等着以后启用这颗钉子的时候了。
<br/>
<br/>    螺妖一脉这颗钉子，越是在苍梧殿之中埋的深，那么以后对于瓦解苍梧殿就越有利，对苍梧殿可能造成的伤害就越是大。
<br/>
<br/>    水神做完了这一切之后，就静静的看着水域诸多强者的态度，此刻它们越是不平静越好，也就越是能够让自己的真实目的不被察觉。
<br/>
<br/>    果然，水域的众多强者，在听到了水神殿传出来的消息之后，这一次才真的是炸锅了，甚至有的强者都以为，可能很快就要出事了。
<br/>
<br/>    在水神殿的强者还没有跟人族强者开战之前，极有可能先跟苍梧殿的强者来上一场，因为这关乎到颜面的问题，开战也不是不可能啊。
<br/>
<br/>    但是，水域的众多强者失望了，它们在等待了许久之后，并没有发现自己想象中的情况出现，事实再一次出乎它们的预料。
<br/>
<br/>    因为这个时候，水神殿和苍梧殿，两大水域势力的执掌者，竟然同时宣布了一件事情，那就是两家即将联手，并且整合水域的力量，跟人族强者开战。
<br/>
<br/>    此刻水域的强者算是被接二连三的消息炸蒙圈了，这是什么情况，这跟本就跟它们所想的不一样，一点都不一样，简直是截然相反的。
<br/>
<br/>    本以为两家怎么说也得意思意思，没想到不但没有什么矛盾，反而这个时候宣布了联手的消息，这算是怎么回事，难道自己理解错了。
<br/>
<br/>    后来从苍梧殿之中传来消息，这一次的针对人族的一战，将会由水神为主导，苍梧殿辅助，水域的其他所有强者负责配合行动。
<br/>
<br/>    水域的强者对于很多真实的情况都不了解，只能暗自猜测，其实水神殿和苍梧殿之间，已经有了一场不为人知的较量，最后可能是苍梧殿输了。
<br/>
<br/>    因为要不是这样的话，苍梧殿怎么可能会让水神殿的强者主导这一次的事情，而它们苍梧殿的强者，只是辅助水神殿呢。
<br/>
<br/>    说的好听是辅助，其实不就是听从水神殿的调遣吗，至少水域的强者都是这么认为的，因为这件事情，只能有一个声音发声。
<br/>
<br/>    就算是到时候，苍梧殿的强者可以提出自己的想法和意见，但是最终拍板的，依然会是水神殿的强者，至少这一点是不会改变的。
<br/>
<br/>    因为说了是水神殿为主导，那么就必须要以水神殿为主导，若是苍梧殿都不能履行，那么怎么能够让水域的其他强者，去执行水神殿强者的计划呢。
<br/>
<br/>    对于此刻水域众多强者的看法，不管是水神殿的水神也好，还是苍梧殿的苍梧之主也好，对于这些其实并不怎么在意，因为根本就没有必要。
<br/>
<br/>    真是的情况是什么样子的，它们两个自然是比谁都清楚，有些事情做到自己心中清楚就好了，根本就没有必要去在意其他的什么言论和事情。
<br/>
<br/>    就在水域的强者，对于这些事情议论纷纷，分别发表各自的看法的时候，都想着为什么苍梧殿会突然之间选择出战的时候，水神殿和苍梧殿的强者，什么都没有表示。
<br/>
<br/>    其实水域的强者有这样的想法也不足为奇，因为都知道，苍梧殿一直以来，都是什么事情都不管的，就算是之前也是一样的态度。

<br/>
<br/>    只是现在苍梧殿的强者不说，水域的强者又哪里能够知道呢，它们又不能亲自去苍梧殿询问，就算是去了，苍梧殿的强者也不会搭理它们啊。
<br/>
<br/>    若是苍梧殿想说的话，那么肯定一早就公布出来了，若是苍梧殿的强者没有主动的公布出来，那么谁去问谁就是给自己找不痛快。
<br/>
<br/>    人家不想说的事情，你非得去问个明白，并且实力还没有苍梧殿强，那不就是去找死吗，说不定自己哪句话，就触碰到了苍梧殿强者的痛处。

<br/>    这种事情没有哪个强者去做，所以现在这个时候，它们也就只能凭着自己的想象去理解了，也正是因为这样，一水域强者的众说纷纭，持有什么观点的都有。
<br/>
<br/>    但是不管水域的强者怎么说，它们的观点都是一致的，那就是苍梧殿之中，必然是发生了什么事情，关于这一点，没有任何的强者有怀疑。

<br/>    现在只是不知道，苍梧殿之中到底发生了什么事情，越是不知道，就越发的好奇，可是又没有知情的强者，去满足它们的好奇心。

<br/>    水神没有开口，水神殿的那些强者，在苍梧之主来水神殿之前，就已经被水神下了封口令，自然更加不会有强者去说的。
<br/>
<br/>    因为在苍梧之主的心中，此刻水神殿知道这些事情的，最多也就是一些神剑圣强者，因为水神是绝对不可能，把这些消息，告诉水神殿更多的强者的。
