　　我不做英雄，英雄善始不善终。
<br/>　　我不做枭雄，机关算尽到头空。
<br/>　　我就是我，走在黑暗中的骑士，尽管孤独。
<br/>　　黑乎乎的墓道中，叶少阳跟梧桐结伴同行，路上他们讨论了之前看到的那些石棺，石棺里的鱼，还有那两只蝾螈。
<br/>　　一直认为，那两只蝾螈可能是早前就生活在山下的地下水系中，被修墓的人故意修了一口井，让他们多了一个出口，用来攻击擅自闯入古墓的人。之前的战斗中，叶少阳从两只蝾螈身上感受到了妖气的存在，因此，这两只蝾螈必然是被用什么邪术养成了妖精，有了自主攻击外来者的本能。
<br/>　　想起那只蝾螈庞大的身躯和凶残的力量，叶少阳其实是有些后怕的，不得不承认，这种活生生的用来守墓的邪物，比一些机关要好用的多。
<br/>　　如果当时两只蝾螈都一起出来的话，后果真的不堪设想。
<br/>　　至于那些鬼鱼和石棺，叶少阳想到石棺里的水，肯定是连着地下水系，用途一时间不得而知，也只能之后再去慢慢解释了。
<br/>　　“那留下那句话，让后人小心蝾螈的，真的是妙心的祖先？”
<br/>　　“很有可能，不管是谁，反正是进来过的，而且跟蝾螈交手过，侥幸逃生。”
<br/>　　“可是为什么之前的密道都是打开的，只有这里的石门是上锁的，那之前经过这里的人，是怎么样进来的，为什么出去后又把门给锁了？”梧桐的问题很多，不过问的却都在点子上。
<br/>　　叶少阳想了一下，说道：“这问题我想过，很有可能，之前那个人不是走的这条路，这条路……我怀疑是修墓的人走的，并不是正常的通道。”
<br/>　　“哦……”虽然叶少阳说得有点不清不楚，不过梧桐还是听懂了一些。对于古墓，她懂得不多，但也多少知道一点：
<br/>　　古代时候，在修墓的时候，都要留下一条工人走的路，通过这里，可以直接走到墓穴的各个位置，搬运石料器物，用来修建古墓。
<br/>　　之所以不走主道的原因，是因为很多地方都是“单行道”，修好了之后，就没办法出去了，这也是为了提防盗墓贼的一个办法，最重要的是，这条工人走的路一般都是在整个风水的死角，是另外开的一条路，在风水学之中，它不属于古墓的一部分，因此也不会破坏古墓的风水。
<br/>　　一般来说，等古墓修成了之后，这条路也就封死了，让人没法再走，更有甚者，为了防止泄密，墓主人会杀掉所有修墓的人，有些就直接坑杀在墓道里……
<br/>　　梧桐看着这条一无所有的墓道，想了一会说道：“你说这条是工人走的墓道，这个也是有可能的，只是……我们之前检查过了，除了这道打不开的石门，那个墓室里并没有别的出入口啊？”
<br/>　　“别忘了那口水井，也许，那里就是进入古墓的必经之路。”
<br/>　　“不会吧！”梧桐怔了怔说道，“如果下面就是地下河，进入不就被淹死了吗？”
<br/>　　“就是要让人淹死，这样才进不去古墓，就算不被淹死，如果是在水下遇到蝾螈，基本上也就是个死。”
<br/>　　从古到今，古墓为了防止被人盗墓，都修得戒备森严，各种机关之类的，不过除了后世因为地壳运动之类的，造成古墓填埋，形成真空，一般的古墓没有完全真空的。
<br/>　　因为所谓风水，一个“风”字，就代表一定要有空气——棺材可以密封，主墓室可以密封，但整个古墓一定要有气息畅通，“气”这个字在古代华夏人心中有着重要的地位，一座真正的大型古墓，如果完全没有流动的气息，那整个阴宅风水也就是死的，起不到保佑后人的作用。
<br/>　　为了空气流通，就一定会修墓道，最后寻找地下河之类的，将气息贯通，这样风水才是活的。这样一来，也就给了后人有机可趁，只要懂得风水秘术的，自然知道**开在什么地方，然后根据这些规则进入古墓……不过墓主人也会在墓道各处埋下各种机关，防止后人的闯入。
<br/>　　叶少阳虽然不是四宝那样的摸金校尉，不过对这些基本原理还是懂得的。
<br/>　　两人绕着墓道前进，最终转过了一道葫芦形状的拱门，来到了一块更大的所在：
<br/>　　四周墙壁上还镶嵌着那种发光的石头，将整个空间勉强照亮。
<br/>　　这是一间大概有一百多平米的墓室，十分空旷，周围每个墙角都摆着一只巨大的花瓶，除此之外，就只有位于墓室中间的那个东西了。
<br/>　　那是一个金光闪闪的东西，上面窄下面宽，就算是在墓室里只有冷光的情况下，依然闪烁着金色的光泽。最上面一层，占地有至少十个平方米，看上去像是金字塔，不过是一层一层的，上面有很多精美的雕刻。
<br/>　　“这是什么鬼东西……”叶少阳还是第一次看到这种东西，一时间有点呆滞。
<br/>　　“坛城……”梧桐喃喃自语。
<br/>　　“什么？”
<br/>　　“坦诚，西域佛国的一种佛教的设施，藏传佛教。”
<br/>　　“哦。”叶少阳一下子就听懂了，举着灯走过去，照了照，在灯光之下，这座坛城看上去更加显得金碧辉煌。
<br/>　　叶少阳忍不住伸手摸了摸坛城的一个角，惊道：“卧槽，金的！”
<br/>　　“你怎么知道？”梧桐连忙问。
<br/>　　“猜的。”叶少阳很是理直气壮。
<br/>　　梧桐也没怼他，兀自点头说道：“金的，也是有可能的，佛国一向都是用金银珠宝制作坛城。”
<br/>　　叶少阳用灯光照着，这才看到上面每一层的正中间，的确都镶嵌着很多宝石。大部分是绿色的。梧桐看到之后，表示这是绿松石，西域特产，于是更加印证了自己的猜测。
<br/>　　“我说，古墓里为什么会有这种东西？”叶少阳问道，一边打量着坛城上部的雕刻，果然发现都是佛一类的人物，不过造型有点抽象，形象跟汉人寺庙里供奉的不一样。
<br/>　　“不对，这些不是藏传的佛……”梧桐皱起了眉头说道。
