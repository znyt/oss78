        “阿嚏——”
<br/>
<br/>        童筱颖只不过是打了一个喷嚏，站在一旁给她挑今晚舞台服的唐倩兮就一脸紧张地问：“你又感冒了吗？今晚还能不能唱？”
<br/>
<br/>        “当然可以唱啊！”童筱颖随即抽了张纸巾，擦了擦鼻子，微微一笑。
<br/>
<br/>        唐倩兮瞥了她一眼，忽然间想到了什么，忍俊不禁地打趣：“说不定是有谁想你了！”
<br/>
<br/>        “诶？真的吗？”听唐倩兮这么一说，童筱颖的脑海里想到的第一个人，便是傅昼景。
<br/>
<br/>        唐倩兮再次看了童筱颖一眼，发现她精致的小脸上浮泛着甜美的笑容，也跟着笑道：“呦呦呦！你这小脑袋瓜子里是想到谁呢？”
<br/>
<br/>        童筱颖睐了一眼唐倩兮，嘴角微微上扬，勾出一抹浅浅的弧度，漂亮的眸子里，眸光变得格外清澈。
<br/>
<br/>        “只笑不语，这是心有所属？噢！对了！我听司峻提起过，你嫁人了吧！那个男人……是不是就是那次留你在他家里过夜，还自称是你‘男朋友’的那个？”
<br/>
<br/>        唐倩兮弯弯的柳叶眉微微扬了扬，从她红唇里挤出来的话十足的挑逗。
<br/>
<br/>        童筱颖内心一阵欣喜，几乎是条件反射般，脱口而出地反问：“昼景有谎称过他是我‘男朋友’吗？”
<br/>
<br/>        也对，她记得，他曾开玩笑地对她说过，要当她的男朋友。
<br/>
<br/>        “原来那个男人叫‘昼景’啊！”唐倩兮忍俊不禁地回答，其实，她确实只是从自己的男朋友，也就是她童筱颖的哥哥诺司峻那里听说过，有这么一个好男人在追求筱颖。
<br/>
<br/>        只是，她不知道姓甚名谁罢了！
<br/>
<br/>        想到这里，唐倩兮忽然意识到了什么，几乎不假思索地问：“你和你老公最近是出了什么资金问题吗？你居然愿意来我们这里驻唱！”
<br/>
<br/>        “我——”童筱颖欲言又止，抿了抿唇后，拉起了唐倩兮的双手，将自己和昼景的事情娓娓道来。
<br/>
<br/>        听完童筱颖的叙述后，唐倩兮很是同情地看着童筱颖，并信誓旦旦地说：“你放心，你在这里工作的事情，我会替你保密的！”
<br/>
<br/>        “嗯，谢谢嫂嫂，噢，不，是小倩姐。”童筱颖尴尬地咧嘴一笑。
<br/>
<br/>        唐倩兮摆了摆手，笑着说道：“我听你叫我嫂嫂叫习惯了，以后你就继续叫我嫂嫂吧！”
<br/>
<br/>        “嗯！”童筱颖开心得猛点头。
<br/>
<br/>        “今晚你穿旗袍吧！正好，今晚的歌曲都是民国风。”唐倩兮从衣架上挑了一件银白色的旗袍出来在童筱颖的身上比了比。
<br/>
<br/>        反正，唐倩兮给童筱颖挑什么衣服，童筱颖就穿什么衣服上台驻唱。
<br/>
<br/>        童筱颖换完衣服后，戴上了白羽假面眼镜，跟唐倩兮一起离开了休息室。
<br/>
<br/>        离开唱还有半个多小时，唐倩兮闲来无聊，便跟童筱颖聊起来白天找秋容顶替她去见那个“连少”的事情。
<br/>
<br/>        唐倩兮还忍俊不禁地嘲笑“连少”很好骗，晚上结算的时候，费尔南会让财会部的人把那五万块钱分红一部分给她童筱颖。
<br/>
<br/>        但童筱颖听着这话，心里却怎么也高兴不起来，感觉自己就像一个诈骗集团的始作俑者。
<br/>
<br/>        就在两人经过一条长廊，路过一张虚掩着的门口时，里面突然传来一阵瓷器摔在地上那支离破碎的声音。
<br/>
<br/>        唐倩兮立即给童筱颖比了一个噤声的手势，刚准备牵着童筱颖疾步离开的时候，只听到门内传来了秋容的声音。
<br/>
<br/>        “十爷，你要怎样，才能放过我？”
<br/>
<br/>        秋容几近哀腔，嗓音在微微颤抖。
<br/>
<br/>        唐倩兮的步伐一滞，又好奇地拉着童筱颖伫足在房门口。
<br/>
<br/>        童筱颖听晚霞姐提起过，“夜惑”里，暂时共有十个持股人，他们管那十个持股人叫“爷”。
<br/>
<br/>        “夜惑”里的九位爷，唐倩兮都见过，唯独这“十爷”最神秘。
<br/>
<br/>        唐倩兮抬起手来，轻轻地推了推房门。
<br/>
<br/>        透过门缝，只见秋容跪在地上，佝偻着身子。
<br/>
<br/>        房间内开了一盏光线微弱的小夜灯，看不清坐在沙发上那位“十爷”此刻是什么表情。
<br/>
<br/>        不过，透过他指缝间夹着的香烟闪着微弱红光，似乎是在意味着什么东西在一丁点一丁点地消失殆尽。
<br/>
<br/>        “你让我放过你？那谁来替婴宁偿命？”男人语气阴森，冷酷无情，“不要试图挑战我的耐性！”
<br/>
<br/>        “婴宁不是我害死的！你要我说多少遍，你才会相信？不是我！真的不是我！我没有害死婴宁！我没有！我没有！”秋容捂着脸，无助地抽泣了起来。
<br/>
<br/>        这是什么情况？
<br/>
<br/>        童筱颖和唐倩兮愣在门口，彼此看得面面相觑。
<br/>
<br/>        不过，那个“十爷”的声音……
<br/>
<br/>        怎么听起来像“连少”的？
<br/>
<br/>        或许唐倩兮没有发现这点，但是天生对音色敏感的童筱颖，却能听出来。
<br/>
<br/>        “这一辈子，你都别想离开‘夜惑’！你就永远地留在这里‘卖’！好好地赎罪！”坐在沙发上的男人，恍若浑身都散发着一股凛冽的寒气，冷冷地说道。
<br/>
<br/>        “我——”秋容欲言又止，所有的话，全都淹没在了哭声中。
<br/>
<br/>        唐倩兮听到这里，柳眉微蹙，下意识地握紧了童筱颖的手，拉着她疾步离开。
<br/>
<br/>        “嫂嫂，你怎么呢？”童筱颖一只手被唐倩兮牵着搭在了半空中，另一只手随着步伐的摆动，疾步跟在了唐倩兮的身后。
<br/>
<br/>        如果不是那十爷和秋容提起“婴宁”，唐倩兮还真就把五年前曾差点让“夜惑”从滨江城消失的那个事件给忘了。
<br/>
<br/>        童筱颖也有些好奇，有些按捺不住地问：“嫂嫂，‘夜惑’里……曾经是不是死过人？那个婴宁是不是死在了……”
<br/>
<br/>        “筱颍，晚霞应该教过你，在‘夜惑’里，不要多管闲事，不要打听自身以外的事情。”唐倩兮一脸严肃地打断了童筱颖欲要说的话，顿了顿后，又接着说道，“筱颍，你要记住了，从今以后，在这‘夜惑’里，不要让除了我和费老板以外的人，看到你的真实容颜。”
<br/>
<br/>        是啊！
<br/>
<br/>        这么重要的事情，她竟然给忘了，而且一忘就是五年。
<br/>
<br/>        唐倩兮是“夜惑”里的老员工了，而费尔南是去年才来接管“夜惑”的，所以，费尔南并不知道“婴宁”的事情。
