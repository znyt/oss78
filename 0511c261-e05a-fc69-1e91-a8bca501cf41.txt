　　第235章
<br/>　　留给程大雷放松的机会并不多，周围布满禁卫军，其中并不缺乏高手。此刻他们挥舞着兵器，高声咆哮着。
<br/>　　“杀呀！”
<br/>　　程大雷要走，因为他已经没有留在此地的意义。
<br/>　　可没人想让他走。
<br/>　　更多的人围过来，尤其是周声声为首的相府高手，此刻死缠着程大雷不放。
<br/>　　时天寒也想去捉程大雷，不用说，今天谁若能抓住程大雷，必然是大功一件。
<br/>　　可林冲不放他走，林冲像个锅盖，时天寒是锅中的水，不管水滚得再厉害，也冒不过锅盖。
<br/>　　时天寒冒不过林冲，在林冲的枪下，他不能说完全没有胜算，可却有种拼尽全力都无能为力的感觉。
<br/>　　“为什么，为什么十年前我胜不过你，十年后你却变得更强。”时天寒咆哮着，向林冲连刺数枪。
<br/>　　“因为我输了我就死了。”林冲轻描淡写躲开：“所以我不能输。”
<br/>　　因为不能输，就不会输么？世间可有如此简单的道理。时天寒歇斯底里发出一枪。
<br/>　　“我他妈的也不想输啊。”
<br/>　　依旧没有击中，却消耗了太多体力，现在胜利女神的吻离他越来越远了。
<br/>　　程大雷的情况愈发堪忧，以周声声为首的相府四大高手缠住了他，身周围着更多的禁卫军，如果不是顾及周声声四人，他们一拥而上的话，程大雷必死无疑。
<br/>　　程大雷且战且退，但依旧摆脱不了周声声的追击，再打下去，已经没有任何意义。
<br/>　　轰隆！
<br/>　　突然耳边传来剧烈的爆炸声，地面陷落，那种感觉就像是有道地龙在土里拱来拱去。
<br/>　　围着程大雷的禁卫军陷落在坑里，城墙底部被炸出一个大洞，一堆人冲了进来。
<br/>　　“大哥，大哥，我们来救你了。”
<br/>　　阿苦手持一杆铁棍，一马当先走在最前方。
<br/>　　长安是有下水道的，皇城的雨水要顺着下水道流出去。程大雷手下并不缺少盗墓贼之类的江湖人，而火药，在长安虽然很稀罕，但并不是买不到。假若你有足够的钱，在军中还有些关系，那就更容易了。
<br/>　　恰好，程大雷既有钱，也有关系。
<br/>　　程大雷可以为了一些事情冒险，但帝国没一样东西值得他去送死，杨龙停也不行。所以，他也不会傻到，单枪匹马来刺杀杨龙停，不然，他岂不是和林少羽成了一类人。
<br/>　　阿苦，秦蛮带着一群江湖客冲了进来，他们也不厮杀，而是从身上丢出一个个皮球状的物事，里面冒着白烟，弥漫开来，呛的人咳嗽不止。
<br/>　　这是用猪尿泡填了硫磺，白磷，胡椒粉等等之物。
<br/>　　这算是最初始的烟雾弹么。
<br/>　　“走啦，走啦。”
<br/>　　程大雷大呼小叫，声音是想让林冲知道，趁着这机会赶紧撤，毕竟，留给他们的时间并不多。
<br/>　　禁卫军是第一次遭遇这种事，刺鼻的气味倒是还能忍受，关键是眼睛看不见东西，他们不知道敌人在哪里。
<br/>　　“就是这样，就是这样！”
<br/>　　金问道腾地一下站起，紧紧握着拳头，似乎又回忆起什么。
<br/>　　程大雷想走，可是有人不像他走，这个人是周声声。
<br/>　　他紧随着程大雷，早已将囚龙棍换了一柄制事长枪，带着相府几大高手死缠着程大雷不放，不肯落下程大雷一步。
<br/>　　程大雷暗暗心焦，这个人是属狗的么，眼前的乱象持续不了太长时间，一旦被这些禁卫军清醒过来，肯定是大开杀戒。
<br/>　　好在此刻徐神机带着这些人炸塌城墙后，便立刻带着人撤退，真和禁卫军冲杀起来，只有死路一条。
<br/>　　程大雷是向城墙缺口冲去，奈何周声声紧追不放，而对方实力并不弱于自己，一旦程大雷有了退意，就更加敌不过对方。
<br/>　　锵！
<br/>　　忽地，一道剑音响起，不知不觉间，程大雷已退到场地中央的舞台上，在他和周声声激战时，有个人从地上站起，用手中剑挡了周声声一招。
<br/>　　是楚青衣。
<br/>　　程大雷愣了愣，这一幕绝对出乎他的预料。
<br/>　　“快走。”楚青衣且舞且道。
<br/>　　他既然会剑舞，自然会舞剑，剑随身转，与周围兵器碰撞，发出铛铛的声响，如同动人的乐声。
<br/>　　“你抽什么疯！”程大雷几乎是大喊。
<br/>　　程大雷已看过楚青衣的信息，他是一个绝世高手，程大雷遇到的第一个绝世高手。
<br/>　　但他这绝世高手职业是戏子，你特么要是个绝世砍柴的我都敬你是条汉子，但你只是一个戏子呐。你这个时候站出来拦住周声声，几乎等同于一个杂技演员挑战……吴京？而且这杂技演员还特么是女的。
<br/>　　这时候你不抱头在角落里钻着，偏要站出来做什么。你想以一当百，为我争取时间，可英雄是那么好当的么。
<br/>　　周声声看着楚青衣，微微皱眉，随之一枪捅进他的身体，紧跟着一脚将他踹飞。
<br/>　　他真正的舞蹈究竟进行了多久，一息，一瞬，一刹那……但那一瞬的艳绝，怕见过的永远不会忘记。
<br/>　　程大雷距离楚青衣大约十步，他见到楚青衣出现就回过头来，见到楚青衣舞剑就回身冲来……可依旧来不及。程大雷只来得及接住他在空中跌落的躯体。
<br/>　　他的脸变的更白，可又浮现一种诡异的血色，两只眼睛变得更加雪亮，宛若星辰。
<br/>　　“你为什么要这么做？”程大雷牙关扣紧。
<br/>　　他已看得出来，表演结束后，楚青衣一直倒在舞台上，之后大家便没再注意他，直到程大雷经过时，他才发出了一剑，微乎其微的一剑，但也的确为程大雷争取了些许时间。
<br/>　　“我的命是你救的，此时也还了你。”
<br/>　　“你……”程大雷一咬牙，将楚青衣背在身上：“我带你走。”
<br/>　　楚青衣头歪在程大雷肩上，血从他嘴里流出，浸湿程大雷肩膀的衣服。
<br/>　　“我的命已经保不住了。”
<br/>　　“你特么傻不傻。”程大雷一边用剑驱散周围的兵丁，一边向城墙缺口逃去。
<br/>　　“滴水之恩，当涌泉相报，青衣虽然在台上是个女子，可毕竟是个男儿，也有血性，也知恩义二字。”
<br/>　　“什么男子，你只是一个戏子。”程大雷忍不住骂道。
<br/>　　
<br/>(本章完)
